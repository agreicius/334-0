    Section 1.6 Span and linear independence

    There are many situations in mathematics where we want to describe an infinite set in a concise and useful manner. A guiding example of this for us is the parametric description of infinite sets of solutions to systems of linear equations that you learn about in a first course in linear algebra. We would like to develop an analogous technique for describing general vector spaces and their subspaces.
    As we will see in the next section the relevant linear algebraic tool for this purpose is the concept of a basis. Loosely speaking, a basis for a vector space \(V\) is a set of vectors that is large enough to generate the entire space, and small enough to contain no redundancies. What exactly we mean by “generate” is captured by the rigorous notion of span; and what we mean by “no redundancies” is captured by linear independence.

    Subsection Span

    In a nutshell we will define the span of a tuple \((v_i)_{i\in I}\) of vectors to be the set of all linear combinations of those vectors. Since we allow the index set \(I\) of a tuple to be arbitrary (and possibly infinite), we need to first make proper sense of what a linear combination of \((v_i)_{i\in I}\) is. We do so with the following somewhat technical definition.

    Definition 1.6.1. Linear combination of a tuple.

    Let \(V\) be an \(F\)-vector space, and let \((v_i)_{i\in I}\) be a tuple of elements of \(V\) where \(I\) is nonempty. A linear combination of the vectors \(v_i\) is an expression of the form
    \begin{equation} \sum_{i\in I}c_iv_i\text{,}\tag{1.12} \end{equation}
    where \(c_i\in I\) for all \(i\text{,}\) and \(c_i=0\) for all but finitely many \(i\in I\text{.}\)
    If \(c_i=0\) for all \(i\in I\text{,}\) we call the linear combination (1.12) the trivial linear combination and define
    \begin{equation*} \boldzero=\sum_{i\in I}0v_i\text{.} \end{equation*}
    Otherwise, letting \(c_{i_1},c_{i_2},\dots, c_{i_n}\) be the finitely many nonzero coefficients, we define
    \begin{equation*} \sum_{i\in I}c_iv_i=\sum_{k=1}^nc_{i_k}v_k=c_{i_1}v_{i_1}+c_{i_2}v_{i_2}+\cdots+c_{i_n}v_{i_n}\text{.} \end{equation*}

    Remark 1.6.2. Linear combination of tuples.

    For the most part we will be dealing with \(n\)-tuples of vectors, where the index set \(I=\{1,2,\dots, n\}\text{.}\) In this special case, the “for all but finitely many” condition is automatic, making Definition 1.6.1 very straightforward: a linear combination of a tuple \((v_1,v_2,\dots, v_n)\) is just a vector of the form
    \begin{equation*} c_1v_1+c_2v_2+\cdots +c_nv_n\text{,} \end{equation*}
    where \(c_i\in F\text{.}\)
    We now proceed to the definition of the span of a tuple \(S=(v_i)_{i\in I}\) of vectors. The definition is broken into two cases depending on whether \(S=()\) is the empty tuple (a list with no elements).

    Definition 1.6.3. Span.

    Let \(V\) be an \(F\)-vector space, and let \(S=(v_i)_{i\in I}\) be a tuple of vectors of \(V\text{,}\) where \(I\) is a nonempty set. The span of \((v_i)_{i\in I}\text{,}\) denoted \(\Span (v_i)_{i\in I}\text{,}\) is defined as follows.
    • If \(I=\emptyset\text{,}\) and consequently \(S=()\) is the empty tuple (or empty list), then we define
      \begin{equation} \Span S=\Span()=\{\boldzero\}\text{.}\tag{1.13} \end{equation}
    • If \(I\ne \emptyset\text{,}\) then we define \(\Span(v_i)_{i\in I}\) as the set of all linear combinations of \((v_i)_{i\in I}\text{:}\) i.e.,
      \begin{equation} \Span (v_i)_{i\in I}=\{v\in V\colon v=\sum_{k=1}^nc_{i_k}v_{i_k} \text{ for some } n\in \Z_+, c_{i_k}\in F\} \text{.}\tag{1.14} \end{equation}

    Remark 1.6.4.

    Let \((v_i)_{i\in I}\) be tuple of vectors in \(V\text{.}\)
    1. The zero vector is always an element of \(\Span(v_i)_{i\in I}\text{.}\) We treat the cases where \(I\) is empty or nonempty separately.
      If \(I=\emptyset\text{,}\) then \((v_i)_{i\in I}=()\) is the empty tuple and \(\Span S=\{\boldzero\}\) by definition.
      If \(I\ne \emptyset\text{,}\) then we can always express \(\boldzero\) as the trivial linear combination \(\boldzero=\sum_{i\in I}0v_i\text{.}\) Thus \(\boldzero\in \Span(v_i)_{i\in I}\text{.}\)
    2. The set \(\{v_i\colon i\in I\}\) is contained in \(\Span(v_i)_{i\in I}\text{.}\) Indeed, given any \(i\in I\text{,}\) we have \(v_i=1v_i\in \Span(v_i)_{i\in I}\text{.}\)
    3. For a tuple \((v)\) of length 1, we have \(\Span(v)=\{cv\colon c\in F\}\text{,}\) the set of all scalar multiples of \(v\text{.}\)

    Example 1.6.5. Examples in \(\R^2\).

    Let \(V=\R^2\text{.}\) For each tuple \(S=(v_i)_{i\in I}\text{,}\) identify \(\Span(v_i)_{i\in I}\) as a familiar geometric object.
    1. \(\displaystyle S=()\)
    2. \(\displaystyle S=(\boldzero)\)
    3. \(S=((a,b))\text{,}\) \((a,b)\ne (0,0)\)
    4. \(\displaystyle S=((1,0),(0,1))\)
    5. \(\displaystyle S=((1,1),(2,2))\)
    6. \(\displaystyle S=((1,1),(1,2))\)
    7. \(\displaystyle S=((a,b))_{(a,b)\in \R^2}\)
    Solution.
    1. \(\Span()=\{(0,0)\}\text{,}\) the set containing just the origin, by definition.
    2. \(\Span(\boldzero)\) is the set of all scalar multiples of \(\boldzero\text{.}\) Thus \(\Span(\boldzero)=\{\boldzero\}\text{,}\) the set containing just the origin.
    3. \(\Span((a,b))\) is the set of all scalar multiples of the nonzero vector \((a,b)\text{.}\) Geometrically, this is the line that passes through the the origin and the point \((a,b)\text{.}\)
    4. By definition
      \begin{align*} \Span((1,0),(0,1))\amp =\{a(1,0)+b(0,1)\colon a,b\in \R\} \\ \amp =\{(a,b)\colon a,b\in\R\}\amp\text{.} \end{align*}
    5. By definition
      \begin{align*} \Span((1,1),(2,2))\amp =\{a(1,1)+b(2,2)\colon a,b\in \R\}\\ \amp =\{(a+2b,a+2b)\colon a,b\in\R\} \amp \text{.} \end{align*}
      It is easy to see that this set is equal to \(\{(c,c)\colon t\in \R\}\text{,}\) the line with equation \(y=x\text{.}\) Note that in this case we have
      \begin{equation*} \Span((1,1), (2,2))=\Span ((1,1))\text{,} \end{equation*}
      and thus that the vector \((2,2)\) is in some sense redundant with respect to the span operation.
    6. By definition
      \begin{align*} \Span((1,1),(1,2))\amp =\{a(1,1)+b(1,2)\colon a,b\in \R\}\\ \amp =\{(a+b,a+2b)\colon a,b\in\R\} \amp \text{.} \end{align*}
      Claim: \(\Span((1,1),(1,2))=\R^2\text{.}\) Proving the claim amounts to showing that for all \((c,d)\in \R^2\) there exist \(a,b\in \R\) such that
      \begin{equation*} \begin{array}{ccccc} a \amp +\amp b \amp =\amp c\\ a \amp +\amp 2b \amp =\amp d \end{array}\text{.} \end{equation*}
      Solving this system using Gaussian elimination, we see that the system has the unique solution
      \begin{align*} a\amp =2c-d \amp b\amp =d-c\text{,} \end{align*}
      and thus that
      \begin{equation*} (2c-d)(1,1)+(d-c)(1,2)=(c,d)\text{.} \end{equation*}
      This proves \(\Span((1,1),(1,2))=\R^2\text{,}\) as claimed.
    7. Observe first that in this case the index set for our tuple is \(I=\R^2\text{,}\) and the tuple \(S\) is of the form \(S=(i)_{i\in I}\text{.}\) In other words, the tuple in this case is the “list” of all elements of \(\R^2\text{!}\) By Remark 1.6.4, we have \(\{v_i\colon i\in I\}\subseteq \Span(v_i)_{i\in I}\) for any tuple. Thus in this case
      \begin{equation*} \{(a,b)\colon (a,b)\in \R^2\}\subseteq \Span((a,b))_{(a,b)\in \R^2}\text{.} \end{equation*}
      Since clearly \(\{(a,b)\colon (a,b)\in \R^2\}=\R^2\text{,}\) we see that \(\R^2\subseteq \Span((a,b))_{(a,b)\in \R^2}\text{,}\) from which it follows \(\Span((a,b))_{(a,b)\in \R^2}=\R^2\text{.}\)
    You may have noticed that each span computation in the previous example produced a subspace of \(\R^2\text{.}\) This is no accident!

    Theorem 1.6.6. Spans are subspaces.

    Let \((v_i)_{i\in I}\) be a tuple of vectors of the vector space \(V\text{.}\)
    1. \(\Span(v_i)_{i\in I}\) is a subspace of \(V\text{.}\)
    2. \(\Span(v_i)_{i\in I}\) is the smallest subspace of \(V\) containing \(v_i\) for all \(i\in I\) in the following sense: if \(W\) is a subspace of \(V\) and \(v_i\in W\) for all \(i\in I\text{,}\) then \(\Span(v_i)_{i\in I}\subseteq W\text{.}\)

    Proof.

    1. We use Procedure 1.4.7. As observed in Remark 1.6.4, we always have \(\boldzero\in \Span(v_i)_{i\in I}\text{.}\) Next, given \(v,w\in \Span(v_i)_{i\in I}\) we have by definition
      \begin{align*} v \amp =\sum_{i\in I}c_iv_i\\ w \amp =\sum_{i\in I}d_iv_i \end{align*}
      where \(c_i,d_i\in F\) and \(c_i=d_i=0\) for all but finitely many \(i\in I\text{.}\) But then for any \(c,d\in F\text{,}\) we have
      \begin{equation*} cv+dw=\sum_{i\in I}(cc_i+dd_i)v_i\text{,} \end{equation*}
      and \(cc_i+dd_i=0\) for all but finitely many \(i\in I\text{.}\) Thus \(cv+dw\in \Span(v_i)_{i\in I}\text{,}\) as desired.
    2. If \(W\) is a subspace of \(V\) and \(v_i\in W\) for all \(i\in I\text{,}\) then since \(W\) is closed under addition and scalar multiplication, \(W\) contains all linear combinations of the \(v_i\text{.}\) Thus \(W\) contains \(\Span(v_i)_{i\in I}\text{:}\) i.e., \(\Span(v_i)_{i\in I}\subseteq W\text{.}\)

    Definition 1.6.7. Spanning set.

    Let \(W\) be a subspace of the vector space \(V\text{.}\) A spanning set of \(W\) is a tuple \((v_i)_{i\in I}\) of vectors of \(V\) such that \(W=\Span(v_i)_{i\in I}\text{.}\)

    Example 1.6.8. Standard spanning sets.

    Most of the vector spaces we’ve introduced there is a natural spanning set that comes to mind. We will refer to these loosely as standard spanning sets.
    • Zero space.
      Let \(V=\{\boldzero\}\text{.}\) By definition the empty list \(S=()\) is a spanning set of \(V\text{.}\)
    • Tuples.
      Let \(V=F^n\text{.}\) For \(1\leq i\leq n\text{,}\) define \(\bolde_i\) to be the \(n\)-tuple with a one in the \(i\)-th entry, and zeros elsewhere. Then \(S=(\bolde_1, \bolde_2,\dots, \bolde_n)\) is a spanning set for \(F^n\text{.}\)
    • Matrices.
      Let \(V=F^{m,n}\text{.}\) For each \((i,j)\) with \(1\leq i\leq m\) and \(1\leq j\leq n\text{,}\) define \(E(i,j)\) to be the \(m\times n\) matrix with a one in the \(ij\)-th entry, and zeros elsewhere. Then \(S=(E_{ij})_{\substack{1\leq i\leq m\\ 1\leq j\leq m}}\) is a spanning set for \(F^{m,n}\text{.}\)
    • Polynomials.
      Let \(F\in \{\R,\C\}\text{,}\) and let \(I\) be an infinite subset of \(F\text{.}\) The tuple \(S=(x^{i})_{i=0}^n=(1,x,x^2,\dots, x^n)\) is a spanning set \(P_n(I)\text{,}\) as by definition the elements of \(P_n(I)\) are functions of the form \(f(x)=\anpoly\text{.}\)
      Similarly, the tuple \((x^i)_{i=0}^\infty=(1,x,x^2,\dots)\) is a spanning set of \(P(I)\text{.}\)
    It is important to observe that spanning sets for vector spaces are not unique. Far from it! In general, for any nonzero vector space there are infinitely many choices of spanning sets.

    Example 1.6.9. Spanning sets are not unique.

    For each \(V\) and \(S\) below, verify that \(S\) is a spanning set for \(V\text{.}\)
    1. \(V=\R^2\text{,}\) \(S=\{(1,1), (1,2)\}\)
    2. \(V=M_{22}\text{,}\) \(S=\{A_1, A_2, A_3, A_4\}\text{,}\)
      \begin{equation*} A_1=\begin{amatrix}[rr]1\amp 1\\ 1\amp 1 \end{amatrix}, A_2=\begin{amatrix}[rr]1\amp -1\\ 0\amp 0 \end{amatrix}, A_3=\begin{amatrix}[rr]0\amp 0\\ 1\amp -1 \end{amatrix}, A_4=\begin{amatrix}[rr]1\amp 1\\ -1\amp -1 \end{amatrix}\text{.} \end{equation*}
    Solution.
    1. This was shown in Example 1.6.5
    2. We must show, given any \(A=\begin{amatrix}[rr]a\amp b\\ c\amp d \end{amatrix}\text{,}\) we can find \(c_1, c_2, c_3, c_4\in \R\) such that
      \begin{equation*} c_1A_1+c_2A_2+c_3A_3+c_4A_4=\begin{amatrix}[rr]a\amp b\\ c\amp d \end{amatrix}\text{,} \end{equation*}
      or
      \begin{equation*} \begin{amatrix}[rr]c_1+c_2+c_4 \amp c_1-c_2+c_4\\ c_1+c_3-c_4\amp c_1-c_3-c_4 \end{amatrix} = \begin{amatrix}[rr]a\amp b \\ c\amp d \end{amatrix}\text{.} \end{equation*}
      We can find such \(c_i\) if and only if the system with augmented matrix
      \begin{equation*} \begin{amatrix}[rrrr|r] 1\amp 1\amp 0\amp 1\amp a\\ 1\amp -1\amp 0\amp 1\amp b \\ 1\amp 0\amp 1\amp -1\amp c\\ 1\amp 0\amp -1\amp -1\amp d \end{amatrix} \end{equation*}
      is consistent. This matrix row reduces to
      \begin{equation*} \begin{amatrix}[rrrr|r] \boxed{1}\amp 1\amp 0\amp 1\amp a\\ 0\amp \boxed{1}\amp 0\amp 0\amp \frac{a-b}{2} \\ 0\amp 0\amp \boxed{1}\amp -2\amp c-\frac{a+b}{2}\\ 0\amp 0\amp 0\amp \boxed{1}\amp \frac{a+b-c-d}{4} \end{amatrix}\text{.} \end{equation*}
      Since the last column will never contain a leading one, we conclude that the system is consistent for any choice of \(a,b,c,d\text{,}\) and thus that \(\Span S=M_{22}\text{,}\) as claimed.

    Subsection Linear independence

    As mentioned above, the notion of linear independence is precisely what we need to guarantee that a given spanning set \(S=(v_i)_{i\in I}\) has no “redundancies”. This is not immediately evident from the definition, but will be made clear in ......

    Definition 1.6.10. Linear independence.

    A tuple \(S=(v_i)_{i\in i}\) of vectors of the vector space \(V\) is linearly independent if there is no nontrivial linear combination of the \(v_i\) equal to \(\boldzero\text{:}\) i.e., if \(\sum_{i\in I}c_iv_i\) is a linear combination satisfying \(\sum_{i\in I}c_iv_i=\boldzero\text{,}\) then \(c_i=0\) for all \(i\in I\text{.}\) Using logical shorthand:
    \begin{equation} \sum_{i\in I}c_iv_i=\boldzero\implies c_i=0 \text{ for all } i\in I\text{.}\tag{1.15} \end{equation}
    We say \((v_i)_{i\in I}\) is linearly dependent if it is not linearly independent: i.e., if there is a nontrivial linear combination \(\sum_{i\in I}c_iv_i=\boldzero\text{.}\)
    Lastly, observe that an empty list \(S=()\) of vectors is linearly independent as it satisfies the implication (1.15) vacuously.

    Procedure 1.6.11. Linear independence of a finite set.

    Let \(S=(v_i)_{i\in I}\) be a tuple of vectors of the vector space \(V\text{.}\) To decide whether \(S\) is linearly independent, proceed as follows.
    1. Write out the general expression for a linear combination of the form
      \begin{equation*} c_{i_1}\boldv_{i_1}+c_{i_2}\boldv_{i_2}+\cdots +c_{i_n}\boldv_{i_n}=\boldzero\text{.} \end{equation*}
    2. If possible, translate this vector equation into a homogeneous system of linear equations in the unknowns \(c_{i_1},c_{i_2},\dots, c_{i_n}\text{,}\) using the definition of equality for your vector space.
    3. Decide, using Gaussian elimination, whether this system has any nonzero (i.e., nontrivial) solutions. If it has no nontrivial solution, \(S\) is linearly independent; if it has a nontrivial solution, \(S\) is linearly dependent.

    Example 1.6.12. Linear independence.

    For each tuple \(S\) of the given vector space \(V\text{,}\) decide whether \(S\) is linearly independent.
    1. \(V=\R^3\text{,}\) \(S=((1,1,2),(1,0,1), (-2,1,-1))\)
    2. \(V=\R^{2,2}\text{,}\) \(S=(A_1, A_2, A_3)\text{,}\) where
      \begin{equation*} A_1=\begin{bmatrix}3\amp 1\\ 2\amp -3 \end{bmatrix} , A_2= \begin{bmatrix}0\amp 4\\ 2\amp 0 \end{bmatrix} , A_3=\begin{bmatrix}-2\amp -2\\ -2\amp 2 \end{bmatrix}\text{.} \end{equation*}
    3. \(V=P(\R)\text{,}\) \(S=(x^i)_{i=0}^\infty=(1,x,x^2,\dots)\)
    Solution.
    1. We have
      \begin{equation*} a(1,1,2)+b(1,0,1)+c(-2,1,-1)=(0,0,0) \end{equation*}
      if and only if
      \begin{equation*} \begin{linsys}{3} a \amp +\amp b\amp -\amp 2c\amp =0\\ a \amp \amp \amp +\amp c\amp =0\\ 2a \amp +\amp b\amp -\amp c\amp =0\\ \end{linsys}\text{.} \end{equation*}
      After a little Gaussian elimination we see that \((a,b,c)=(1,-3,-1)\) is a nonzero solution to this system, and thus that
      \begin{equation*} (1,1,2)-3(1,0,1)-(-2,1,-1)=(0,0,0) \end{equation*}
      is a nontrivial linear combination equal to \(\boldzero\text{.}\) We conclude that \(S\) is linearly dependent.
    2. We have \(aA_1+bA_2+cA_3=\boldzero\) if and only if
      \begin{align*} a\begin{bmatrix}3\amp 1\\ 2\amp -3 \end{bmatrix} +b\begin{bmatrix}0\amp 4\\ 2\amp 0 \end{bmatrix} +c\begin{bmatrix}-2\amp -2\\ -2\amp 2 \end{bmatrix}\amp = \begin{bmatrix}0\amp 0\\0\amp 0 \end{bmatrix} \amp \text{,} \end{align*}
      if and only if
      \begin{align*} \begin{bmatrix}3a-2c\amp a+4b-2c\\ 2a+2b-2c\amp -3a+2c \end{bmatrix}\amp =\begin{bmatrix}0\amp 0\\0\amp 0 \end{bmatrix} \amp \text{.} \end{align*}
      Gaussian elimination reveals that the corresponding system of equations
      \begin{equation*} \begin{linsys}{3} 3a\amp \amp \amp -\amp 2c\amp =\amp 0\\ a\amp +\amp 4b\amp -\amp 2c\amp =\amp 0\\ 2a\amp +\amp 2b \amp -\amp 2c \amp =\amp 0\\ -3a\amp \amp \amp +\amp 2c\amp =\amp 0 \end{linsys} \end{equation*}
      has infinitely many solutions, and thus nontrivial solutions (e.g., \((a,b,c)=(2,1,3)\)). We conclude that there is a nontrivial combination of the \(A_i\) equal to \(\boldzero\text{,}\) and hence that \(S\) is linearly dependent.

    Theorem 1.6.13. Span and independence.

    Let \(S=(v_i)_{i\in I}\) be a nonempty tuple of vectors of the vector space \(V\text{.}\) The following statements are equivalent.
    1. \(S\) is linearly dependent.
    2. There is an index \(i_{0}\in I\) such that \(v_{i_{0}}\) can be written as a linear combination of the other \(v_i\)’s: i.e., there exists \(i_k\in I\) such that \(v_{i_k}\in \Span(v_i)_{i\in I-\{i_k\}}\text{.}\) Furthermore, in this case we have
      \begin{equation} \Span(v_i)_{i\in I}=\Span(v_i)_{i\in I-\{i_k\}}\text{.}\tag{1.16} \end{equation}

    Proof.

    If the tuple \(S=(v_i)_{i\in I}\) is linearly dependent, then there is a nontrivial linear combination
    \begin{equation*} c_{i_1}v_1+c_{i_2}v_2+\cdots +c_{i_n}v_{i_n}=\boldzero\text{,} \end{equation*}
    where \(i_j\in I\text{.}\) Since the linear combination is nontrivial, we have \(c_{i_k}\ne 0\) for some \(1\leq k\leq n\text{.}\) After a little vector arithmetic, we can then solve for \(v_{i_k}\text{:}\)
    \begin{equation*} v_{i_k}=\sum_{j\ne k}\frac{c_{i_j}}{c_{i_k}}v_{i_j}\text{.} \end{equation*}
    This shows \(v_{i_k}\) is a linear combination of the vectors \(v_i\text{,}\) where \(i\ne i_k\text{,}\) and thus that \(v_{i_k}\in \Span(v_i)_{i\in I-\{i_k\}}\text{.}\)
    Conversely, if we have \(v_{i_k}\in \Span(v_i)_{i\in I-\{i_k\}}\text{,}\) then we have
    \begin{equation} v_{i_k}=\sum_{j=1}^nd_{i_j}v_{i_j}\tag{1.17} \end{equation}
    for some \(i_j\in I\) and \(d_{i_j}\in F\text{,}\) whence it follows that
    \begin{equation*} -d_{i_1}v_{i_1}-d_{i_2}v_{i_2}-\cdots --d_{i_n}v_{i_n}+v_{i_k}=\boldzero \end{equation*}
    is a nontrivial linear combination of the \(v_i\) equal to \(\boldzero\text{.}\) We conclude that \(S=(v_i)_{i\in i}\) is linearly dependent.
    We have thus proved that \(S\) is linearly dependent if and only if some \(v_{i_k}\) can be written as a linear combination of the other vectors \(v_i\text{.}\) We now prove that when this is the case, we have
    \begin{equation*} \Span(v_i)_{i\in I-\{i_k\}}=\Span(v_i)_{i\in I}\text{.} \end{equation*}
    Using statement (2) of Theorem 1.6.6, since \(\Span(v_i)_{i\in I}\) is a subspace and contains \(v_i\) for all \(i\in I-\{i_k\}\text{,}\) we have
    \begin{equation*} \Span(v_i)_{i\in I-\{i_k\}}\subseteq \Span(v_i)_{i\in I}\text{.} \end{equation*}
    For the other direction, take \(v\in \Span(v_i)_{i\in I}\text{.}\) By definition, we can write \(v\) as a finite linear combination of the vectors \(v_i\text{.}\) It follows that can write
    \begin{equation*} v=c_{i_k}v_{i_k}+\sum_{i\in J}c_iv_i\text{,} \end{equation*}
    where \(J\) is a finite subset of \(I\) not containing \(i_k\text{.}\) (Note that \(c_{i_k}\) is allowed to be 0 here.) Since \(v_{i_k}\) is assumed to be a linear combination of the other vectors \(v_i\text{,}\) we can express \(v_{i_k}\) as in (1.17), in which case we have
    \begin{equation*} v=\sum_{j=1}^nc_{i_k}d_{i_j}v_{i_j}+\sum_{i\in J}c_iv_i\text{.} \end{equation*}
    Since \(i_j\ne i_k\) for all \(1\leq j\leq n\text{,}\) and since \(i_k\notin J\text{,}\) we see that this is a linear combination of the tuple \((v_i)_{i\in I-\{i_k\}}\text{.}\) Thus \(v\in \Span(v_i)_{i\in I-\{i_k\}}\text{,}\) showing that
    \begin{equation*} \Span(v_i)_{i\in I}\subseteq \Span(v_i)_{i\in I-\{i_k\}}\text{,} \end{equation*}
    as desired.
    Theorem 1.6.13 is a precise articulation of the fact that a tuple \(S=(v_i)_{i\in I}\) is linearly dependent if and only if one of the vectors \(v_{i_0}\) is “redundant”. In more detail, it says that there is an index \(i_0\) such that
    \begin{equation*} v_{i_0}=c_{i_1}v_{i_1}+c_{i_2}v_{i_1}+\cdots +c_{i_n}v_{i_1} \end{equation*}
    where \(i_k\ne i_0\) for all \(1\leq k\leq n\text{,}\) and furthermore, that in terms of the span operation, this “redundant” vector \(v_{i_0}\) can be dropped. We will most often apply this result in the case where we have a tuple of finite length. We record the result in this special case as a corollary.

    Corollary 1.6.14. Span and independence.

    Let \(S=(v_1,v_2,\dots, v_n)\) be an \(n\)-tuple of vectors of the vector space \(V\text{.}\) The following statements are equivalent.
    1. \(S\) is linearly dependent.
    2. There exists \(k\in \{1,2,\dots, n\}\) such that
      \begin{equation} v_k\in \Span(v_1,v_2,\dots, v_{k-1})\text{.}\tag{1.18} \end{equation}
      Furthermore, when this is the case we have
      \begin{equation} \Span(v_1,v_2,\dots, v_n)=\Span(v_i)_{i\ne k}\text{.}\tag{1.19} \end{equation}
      In other words, the span of the vectors is unchanged when we drop the “redundant” vector \(v_k\text{.}\)

    Remark 1.6.15.

    For the case \(k=1\) in the statement of Corollary 1.6.14, we interpret the assertion
    \begin{equation*} v_k\in \Span(v_1,v_2,\dots, v_{k-1}) \end{equation*}
    as \(v_1\in \Span()\text{.}\) In this somewhat degenerate case, it turns out that we must have \(v_1=\boldzero\text{.}\) See the proof below.

    Proof of corollary.

    The corollary is for the most part simply a special case of Theorem 1.6.13. It remains only to prove that if \(S=(v_1,v_2,\dots, v_n)\) is linearly dependent, then there is a \(1\leq k\leq n\) such that
    \begin{equation*} v_k\in\Span(v_1,v_2,\dots, v_{k-1})\text{.} \end{equation*}
    This is proven as follows: if \(S\) is linearly dependent, then we have a nontrivial linear combination
    \begin{equation*} c_1v_1+c_2v_2+\cdots +c_nv_n=\boldzero\text{.} \end{equation*}
    Since \(c_i\ne 0\) for some \(i\text{,}\) we can set \(k\) to be the largest such \(i\text{:}\) i.e.,
    \begin{equation*} k=\max\{i\in \{1,2,\dots, n\}\colon c_i\ne 0\}\text{.} \end{equation*}
    We now treat the cases \(k=1\) and \(k> 1\) separately. If \(k=1\text{,}\) then by definition of \(k\text{,}\) we have \(c_1v_1=\boldzero\) and \(c_1\ne 0\text{.}\) Using Theorem 1.3.6, we conclude that \(v_1=\boldzero\) in this case, and hence that \(v_1\in \Span()=\{\boldzero\}\text{.}\) (When \(k=1\) we interpret \(\Span(v_1,\dots, v_{k-1})\) as \(\Span()\text{.}\))
    Now assume \(k\geq 2\text{.}\) By definition of \(k\text{,}\) we have
    \begin{equation*} c_1v_1+\cdots +c_kv_k=\boldzero \end{equation*}
    and \(c_k\ne 0\text{,}\) from whence it follows that
    \begin{equation*} v_k=-\frac{c_1}{c_k}v_1+\cdots -\frac{c_{k-1}}{c_k}v_{k-1}\in \Span(v_1,v_2,\dots, v_{k-1})\text{,} \end{equation*}
    as desired.

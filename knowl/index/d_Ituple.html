Definition 0.1.24. I-tuple.

Let \(I\) and \(X\) be sets. An \(I\)-tuple with entries (or coordinates) in \(X\) is a function \(f\colon I\rightarrow X \text{.}\) We call \(I\) the indexing set of the tuple \(f\text{,}\) and for all \(i\in I\text{,}\) we call \(f(i)\) the \(i\)-th entry (or coordinate) of \(f\text{.}\) Furthermore, we will write \(f=(x_i)_{i\in I}\) to denote the tuple \(f\colon I\rightarrow X\) whose \(i\)-th entry is \(f(i)=x_i\) for all \(i\in I\text{.}\)

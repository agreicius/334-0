Definition 1.9.12. Matrix transformations.

Let \(A\in F^{m,n}\text{.}\) The matrix transformation associated to \(A\) is the function \(T_A\colon F^n\rightarrow F^m\) defined as
\begin{equation*} T_A(\boldx)=A\boldx \end{equation*}
for all \(\boldx\in F^n\text{.}\) We call \(A\) the standard matrix of \(T_A\text{.}\)

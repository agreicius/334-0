Theorem 1.23.5. Polynomial evaluation of operator.

Let \(T\in \mathcal{V}\text{,}\) and let \(f(x)\) and \(g(x)\) be polynomials with coefficients in \(F\text{.}\)
  1. \(f(T)\) and \(g(T)\) commute: i.e.,
    \begin{equation} f(T)g(T)=g(T)f(T)\text{.}\tag{1.112} \end{equation}
    In particular, \(T\) commutes with \(f(T)\text{:}\) that is, we have \(T\, f(T)=f(T)\, T\text{.}\)
  2. If \(W\) is a \(T\)-invariant subspace, then \(W\) is a \(f(T)\)-invariant subspace.
  3. Assume \(F\in \{\R, \C\}\) and that \(V\) is an inner product space. We have
    \begin{align} (f(T))^*\amp =\overline{f}(T^*) \tag{1.113}\\ \amp = \overline{a_n}(T^*)^n+a_{n-1}(T^*)^{n-1}+\cdots +\overline{a_1}T^*+\overline{a_0}I_V \text{,}\tag{1.114} \end{align}
    where \(f(x)=\anpoly\text{.}\)
  4. Assume \(F\in \{\R, \C\}\) and that \(V\) is an inner product space. If \(T\) is normal, then \(f(T)\) is normal.

Proof.

  1. Let \(f(x)=\anpoly\) and \(g(x)=\bmpoly\text{.}\) First observe that for all integers \(k\geq 0\text{,}\) we have
    \begin{align*} T^k\, g(T) \amp =T^k(b_mT^m+\cdots +b_1T+b_0)\\ \amp = b_mT^{m+k}+\cdots b_1T^{k+1}+b_0T^k\\ \amp (b_mT^m+\cdots +b_1T+b_0)T^k \text{.} \end{align*}
    It follows that
    \begin{align*} f(T)g(T) \amp = a_nT^ng(T)+\cdots +a_1Tg(T)+a_0g(T)\\ \amp = a_ng(T)T^n+\cdots +a_1g(T)T+a_0g(T)I_V\\ \amp g(T)(a_nT^n+\cdots +a_1T+a_0I_V) \\ \amp =g(T)f(T)\text{.} \end{align*}
  2. First observe that if \(W\) is \(S\)- and \(T\)-invariant for operators \(S,T\in \mathcal{L}(V)\text{,}\) then it is \((cS+dT)\)-invariant for any \(c,d\in F\text{.}\) Indeed, if \(w\in W\text{,}\) then \(S(w)\in W\) and \(T(w)\in W\) implies that \((cS+dT)(w)=cS(w)+dT(w)\in W\text{.}\) It follows by induction that if \(W\) is \(T_i\)-invariant for operators \(T_i\in \mathcal{L}(V)\text{,}\) \(1\leq i\leq n\text{,}\) then it is \((c_1T_1+\cdots +c_nT_n)\)-invariant for any \(c_i\in F\text{.}\) Next, an easy induction argument shows that if \(W\) is \(T\)-invariant, then it is \(T^k\)-invariant for all \(k\geq 0\text{.}\) Thus, if \(W\) is \(T\)-invariant, then it is \(T^k\)-invariant for all \(0\leq k\leq n\text{,}\) and thus also \((a_nT^n+\cdot +a_1T+a_0I_V)\)-invariant for any \(a_i\in F\text{.}\) This shows \(W\) is \(f(T)\)-invariant.
  3. We have
    \begin{align*} (f(T))^* \amp = (a_nT^n+\cdot +a_1T+a_0I_V)^*\\ \amp = \overline{a_n}(T^n)^*+\cdot +\overline{a_1}T^*+\overline{a_0}I_V^* \amp \knowl{./knowl/xref/th_adjoint_props.html}{\text{1.21.5}}\\ \amp = \overline{a_n}(T^*)^n+\cdot +\overline{a_1}T^*+\overline{a_0}I_V \amp \knowl{./knowl/xref/th_adjoint_props.html}{\text{1.21.5}}\\ \amp =\overline{f}(T^*)\text{.} \end{align*}
  4. We have
    \begin{align*} (f(T))^*f(T) \amp = \overline{f}(T^*)f(T)\\ \amp = \sum_{0\leq i,j\leq n}\overline{a_i}a_j(T^*)^iT^j\\ \amp \sum_{0\leq i,j\leq n}\overline{a_i}a_jT^j(T^*)^i \\ \amp = f(T)\overline{f}(T^*)\\ \amp =f(T)(f(T))^*\text{.} \end{align*}

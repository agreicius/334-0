Theorem 1.7.5. Basis equivalence.

Let \(B=(v_i)_{i\in I}\) be a tuple of vectors of \(V\text{.}\) The following statements are equivalent.
  1. \(B\) is a basis of \(V\text{.}\)
  2. For all \(v\in V\) there is a unique linear combination \(\sum_{i\in I}c_iv_i\) of \(B\) such that
    \begin{equation} \sum_{i\in I}c_iv_i=v\text{.}\tag{1.20} \end{equation}

Proof.

We prove both implications separately.
Implication: (2)\(\implies\)(1).
If (2) holds, then in particular for all \(v\in V\) there is at least one linear combination \(\sum_{i\in I}c_iv_i\) of the vectors \(v_i\) equal to \(v\text{.}\) Thus \(\Span B=V\text{.}\) Furthermore, taking \(v=\boldzero\text{,}\) condition (2) tells us that there is exactly one linear combination \(\sum_{i\in I}c_iv_i=\boldzero\text{,}\) which then must be the trivial linear combination (i.e., \(c_i=0\) for all \(i\in I\)). This proves \(B\) is linearly independent. Since \(B\) is a spanning set and is linear independent, it is a basis.
Implication: (1)\(\implies\)(2).
Assume \(B\) is a basis. Since \(B\) spans \(V\text{,}\) for every vector \(v\in V\) there is at least one linear combination as in (1.20). Let us show that this linear combination is unique. Suppose we have
\begin{equation*} v=\sum_{i\in I}c_iv_i=\sum_{i\in I}d_iv_i\text{,} \end{equation*}
where \(c_i,d_i\in F\) and \(c_i=d_i=0\) for all but finitely many \(i\in I\text{.}\) After some vector arithmetic, we see that
\begin{equation*} \sum_{i\in I}(c_i-d_i)v_i=\boldzero\text{.} \end{equation*}
Since \(B\) is linearly independent, we conclude that \(c_i-d_i=0\text{,}\) and thus \(c_i=d_i\) for all \(i\in I\text{.}\) This shows there is a unique linear combination of the \(v_i\) equal to \(v\text{.}\)

Example 1.17.7. Matrix transformation.

Let \(T\colon \R^3\rightarrow \R^3\) be the matrix transformation \(T=T_A\text{,}\) where
\begin{equation*} A=\begin{bmatrix} 1\amp 1\amp 0 \\0\amp 1\amp 0\\ 0\amp 0\amp 2 \end{bmatrix}\text{.} \end{equation*}
Decide whether \(T_A\) is diagonalizable; if it is, find an invertible matrix \(P\) that diagonalizes \(A\) and compute \(D=P^{-1}AP\text{.}\)
Solution.
We compute the characteristic polynomial \(f\) as
\begin{align*} f(x) \amp = \det(xI-A)\\ \amp = \det \begin{amatrix}[ccc] x-1\amp -1 \amp 0\\ 0 \amp x-1\amp 0 \\ 0 \amp 0\amp x-2 \end{amatrix} \\ \amp = (x-1)^2(x-2)\text{.} \end{align*}
(Note how easy \(f\) was to compute, thanks to \(A\) being upper triangular.) The distinct eigenvalues of \(A\) are thus \(1\) and \(2\text{.}\) We compute
\begin{align*} E(1,T_A) \amp = \NS(I-A)\\ \amp =\NS \begin{amatrix}[ccc] 0\amp -1 \amp 0\\ 0 \amp 0\amp 0 \\ 0 \amp 0\amp -1 \end{amatrix}\\ \amp = \Span((1,0,0))\\ E(2,T_A) \amp = \NS(2I-A)\\ \amp =\NS \begin{amatrix}[ccc] 1\amp -1 \amp 0\\ 0 \amp 1\amp 0 \\ 0 \amp 0\amp 0 \end{amatrix}\\ \amp = \Span((0,0,1))\text{.} \end{align*}
Since
\begin{align*} \dim E(1,T_A)+\dim E(2,T_A) \amp = 1+1=2\ne 3\text{,} \end{align*}
we conclude that \(T_A\) is not diagaonalizable.

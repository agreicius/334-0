Example 1.10.9. Homogeneous linear differential equation.

Let \(I\) be an interval of \(\R\text{,}\) let \(q_{n-1},q_{n-2},\dots, q_1\) be continuous functions on \(I\) (i.e., \(q_i\in C(I)\)) for all \(1\leq i\leq n-1\text{,}\) and define \(W\) as the set of functions \(f\in C^n(I)\) satisfying the linear homogeneous differential equation
\begin{equation} f^{(n)}(x)+q_{n-1}(x)f^{(n-1)}(x)+\cdots +q_1(x)f(x)=0 \text{ for all } x\in I\text{.}\tag{1.27} \end{equation}
Prove that \(W\) is a subspace of \(C^n(I)\text{.}\)
Solution.
Define \(T\colon C^n(I) \rightarrow C(I)\) as
\begin{equation*} T(f)=f^{(n)}+q_{n-1}f^{(n-1)}+\cdots +q_1f\text{.} \end{equation*}
We have
\begin{align*} T(cf+dg) \amp = (f+g)^{(n)}+q_{n-1}(f+g)^{(n-1)}+\cdots +q_1(f+g)\\ \amp = c(f^{(n)}+q_{n-1}f^{(n-1)}+\cdots +q_1f)+d(g^{(n)}+q_{n-1}g^{(n-1)}+\cdots +q_1g)\\ \amp =cT(f)+dT(g)\text{,} \end{align*}
showing that \(T\) is a linear transformation. The set of solutions to (1.27) is precisely \(\NS T\text{,}\) making this set a subspace of \(C^n(I)\text{.}\)

Example 1.10.8. Homogeneous linear differential equation.

Let \(I\) be an interval of \(\R\text{,}\) let \(q_{n-1},q_{n-2},\dots, q_0\) be continuous functions on \(I\) (i.e., \(q_i\in C(I)\)) for all \(1\leq i\leq n-1\text{,}\) and define \(W\) as the set of functions \(f\in C^n(I)\) satisfying the linear homogeneous differential equation
\begin{equation} f^{(n)}(x)+q_{n-1}(x)f^{(n-1)}(x)+\cdots +q_1(x)f(x)+q_0(x)=0 \text{ for all } x\in I\text{.}\tag{1.27} \end{equation}
Prove that \(W\) is a subspace of \(C^n(I)\text{.}\)

Example 1.6.8. Standard spanning sets.

Most of the vector spaces weâ€™ve introduced there is a natural spanning set that comes to mind. We will refer to these loosely as standard spanning sets.
  • Zero space.
    Let \(V=\{\boldzero\}\text{.}\) By definition the empty list \(S=()\) is a spanning set of \(V\text{.}\)
  • Tuples.
    Let \(V=F^n\text{.}\) For \(1\leq i\leq n\text{,}\) define \(\bolde_i\) to be the \(n\)-tuple with a one in the \(i\)-th entry, and zeros elsewhere. Then \(S=(\bolde_1, \bolde_2,\dots, \bolde_n)\) is a spanning set for \(F^n\text{.}\)
  • Matrices.
    Let \(V=F^{m,n}\text{.}\) For each \((i,j)\) with \(1\leq i\leq m\) and \(1\leq j\leq n\text{,}\) define \(E(i,j)\) to be the \(m\times n\) matrix with a one in the \(ij\)-th entry, and zeros elsewhere. Then \(S=(E_{ij})_{\substack{1\leq i\leq m\\ 1\leq j\leq m}}\) is a spanning set for \(F^{m,n}\text{.}\)
  • Polynomials.
    Let \(F\in \{\R,\C\}\text{,}\) and let \(I\) be an infinite subset of \(F\text{.}\) The tuple \(S=(x^{i})_{i=0}^n=(1,x,x^2,\dots, x^n)\) is a spanning set \(P_n(I)\text{,}\) as by definition the elements of \(P_n(I)\) are functions of the form \(f(x)=\anpoly\text{.}\)
    Similarly, the tuple \((x^i)_{i=0}^\infty=(1,x,x^2,\dots)\) is a spanning set of \(P(I)\text{.}\)

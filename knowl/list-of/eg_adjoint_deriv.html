Example 1.21.11. Derivative operator.

Let \(W\) be the set of functions \(C^\infty(\R)\) of period 1. The operation
\begin{equation*} \angvec{f,g}=\int_0^1f(x)g(x)\, dx \end{equation*}
defines an inner product on \(W\text{.}\) It is easy to show that if \(f\in W\text{,}\) then \(f'\in W\text{:}\) i.e., the derivative of a \(C^\infty\) period-1 function is a \(C^\infty\) period-1 function. Thus we can define a linear transformation
\begin{align*} T\colon W \amp \rightarrow W\\ f \amp \mapsto f'\text{.} \end{align*}
Show that \(T^*=-T\text{.}\)
Solution.
Observe that \(-T(g)=-g'\text{,}\) by definition. We must show that \(-T\) satisfies the adjoint property with respect to \(T\text{.}\) Given any \(f,g\in W\text{,}\) we have
\begin{align*} \angvec{T(f),g} \amp = \int_0^1f'(x)g(x)\, dx\\ \amp = f(x)g(x)\Bigr\vert_0^1-\int_0^1f(x)g'(x)\, dx \amp (\text{int. by parts.})\\ \amp = 0+\int_0^1f(x)(-g'(x))\, dx \amp (f(0)=f(1), g(0)=g(1))\\ \amp = \angvec{f,-T(g)} \amp (-T(g)=-g')\text{.} \end{align*}

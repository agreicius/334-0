Theorem 1.9.8. Zero, identity, and scaling transformations.

Let \(V\) and \(W\) be vector spaces.
  1. The zero transformation \(0_{V,W}\colon V\rightarrow W\) is a linear transformation.
  2. For all scalars \(c\in F\) the dilation
    \begin{align*} c_V\colon V \amp \rightarrow W\\ v \amp \mapsto cv \end{align*}
    is a linear transformation. In particular, \(I_V=1_V\) is a linear transformation.

Proof.

  1. Let \(0_{V,W}\colon V\rightarrow W\) be the zero function. We verify each defining property separately.
    1. Given \(v, w\in V\text{,}\) we have
      \begin{align*} 0_{V,W}(v_1+v_2)\amp =\boldzero_W \amp (\text{by def.}) \\ \amp =\boldzero_W+\boldzero_W \\ \amp = 0_{V,W}(v_1)+0_{V,W}(v_2) \amp (\text{by def.})\text{.} \end{align*}
    2. Given \(c\in R\) and \(v\in V\text{,}\) we have
      \begin{align*} 0_{V,W}(cv) \amp = \boldzero_W \amp (\text{def. of } 0_{V,W})\\ \amp = c\boldzero_W \amp (\knowl{./knowl/xref/th_vs_props.html}{\text{1.3.6}} ) \\ \amp = c0_{V,W}(v) \amp (\text{def. of } 0_{V,W})\text{.} \end{align*}
    This proves that \(0_{V,W}\colon V\rightarrow W\) is a linear transformation.
  2. Fix a scalar \(c\in R\) and let \(T\colon V\rightarrow V\) be the scaling transformation defined as \(T(v)=cv\) for all \(v\in V\text{.}\)
    1. Given \(v, w\in V\text{,}\) we have
      \begin{align*} T(v_1+v_2)\amp =c(v_1+v_2) \amp (\text{def. of } T)\\ \amp = cv_1+cv_2 \amp (\text{vec. props.})\\ \amp = cT(v_1)+cT(v_2) \amp (\text{def. of } T)\text{.} \end{align*}
    2. Given \(d\in R\) and \(v\in V\text{,}\) we have
      \begin{align*} T(dv) \amp = c(dv) \amp (\text{def. of } T)\\ \amp = (cd)v \amp (\text{vec. props.}) \\ \amp = (dc)v \amp (\text{real mult. is comm.})\\ \amp = d(cv) \amp (\text{vec. props.})\\ \amp = dT(v) \amp (\text{def. of } T)\text{.} \end{align*}
    This proves that \(T\colon V\rightarrow V\) is a linear transformation.

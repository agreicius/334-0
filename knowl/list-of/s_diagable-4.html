Convention 1.17.2. Diagonalizable matrix.

We will say a matrix \(A\in F^{n,n}\) is diagonalizable if its corresponding matrix transformation \(T_A\colon F^{n}\rightarrow F^{n}\) is diagonalizable.
Let’s examine what this means in terms of matrix arithmetic. Let \(B=(\bolde_1,\bolde_2,\dots, \bolde_n)\) be the standard basis of \(F^n\text{,}\) and recall that \(A=[T_A]_B\text{:}\) i.e., the standard matrix of \(T=T_A\) is the matrix representation of \(T_A\) with respect to the standard basis. The matrix \(A\) is diagonalizable if and only if there is a basis \(B'\) of \(F^{n}\) such that \([T_A]_{B'}=D\) is a diagonal matrix. Using Theorem 1.14.18, this in turn is equivalent to there being an invertible matrix \(P\) such that
\begin{equation} D=P^{-1}AP\text{.}\tag{1.49} \end{equation}
Thus, we conclude that \(A\) is diagonalizable if and only if it is similar to a diagonal matrix. We say that the invertible matrix \(P\) in (1.49) diagonalizes the matrix \(A\text{.}\)

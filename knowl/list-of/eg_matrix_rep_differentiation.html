Example 1.13.12. Differentiation.

Let \(I\) be an infinite subset of \(\R\text{,}\) and define \(T\colon P_n(I)\rightarrow P_{n-1}(I)\) as \(T(f)=f'\text{.}\) Compute \([T]_B^{B'}\text{,}\) where \(B\) and \(B'\) are the standard bases of \(P_n(I)\) and \(P_{n-1}(I)\)
Solution.
Let \(B=(1,x,x^2,\dots, x^n)\) and \(B'=(1,x,x^2,\dots, x^n)\text{.}\) We have
\begin{equation*} T(x^k)=\frac{d}{dx}(x^k)=kx^{k-1} \end{equation*}
for all \(0\leq k\leq n\text{.}\) Since \(B'\) is standard, we easily compute
\begin{equation*} [T(x^k)]_{B'}=(0,0,\dots, \underset{(k-1)-\text{th coor.}}{k},0,\dots, 0)\text{.} \end{equation*}
Using recipe (1.34) we conclude that
\begin{equation*} [T]_B^{B'}= \begin{bmatrix} 0\amp 1\amp 0 \amp 0\amp \dots \amp 0 \\ 0\amp 0\amp 2 \amp 0 \amp \amp 0 \\ 0\amp 0\amp 0 \amp 3 \amp 0 \amp \vdots \\ \vdots \\ 0\amp 0 \amp 0 \amp 0 \amp \dots \amp n \end{bmatrix}\text{.} \end{equation*}
In more detail, \([T]_B^{B'}\) is the \((n-1)\times n\) matrix whose first column is the zero column vector, and whose \(k\)-th column for \(2\leq k\leq n\) is the column vector that has \(k\) in the \((k-1)\)-th entry and zeros elsewhere.

Theorem 1.9.20. Projection onto summand.

Let \(V=\bigoplus_{i=1}^rW_i\) be a direct sum decomposition of the vector space \(V\text{.}\) For all \(1\leq j\leq r\text{,}\) we define \(\pi_j\colon V\rightarrow V\) via the following recipe:
  • given \(v\in V\text{,}\) there is a unique expression
    \begin{equation*} v=w_1+w_2+\cdots + w_r\text{,} \end{equation*}
    where \(w_i\in W_i\) for all \(1\leq i\leq\text{;}\)
  • define \(\pi_j(v)=w_j\text{.}\)
The function \(\pi_j\) is a linear transformation. We call \(\pi_j\) the projection onto \(W_i\) operator.

Proof.

Let \(1\leq j\leq r\text{.}\) Observe first that \(\pi_j\) is well-defined precisely because the expression
\begin{equation*} v=v_1+v_2+\cdots +v_r, v_i\in W_i \end{equation*}
is unique by the definition (1.5.4).
We will show that \(\pi_j\) is linear using the 1-step technique. Given \(v,v'\in V\) we can write
\begin{align*} v \amp =w_1+w_2+\cdots + w_r \\ v' \amp = w'_1+w'_2+\cdots + w'_r\text{,} \end{align*}
where \(w_i,w_i'\in W_i\) for all \(i\in I\text{.}\) Note that by definition of \(\pi_j\) we have
\begin{align*} \pi_j(v) \amp = w_j \amp \pi_j(v')=w_j'\text{.} \end{align*}
For any \(c,d\in F\text{,}\) we then have
\begin{equation*} cv+dv'=(cw_1+dw_1)+(cw_2+dw_2)+\cdots + (cw_r+dw_r)\text{.} \end{equation*}
Since each \(W_i\) is a subspace, we have \(cw_i+dw_i'\in W_i\text{,}\) and thus this is the unique decomposition of the vector \(cv+dv'\) in terms of the \(W_i\text{.}\) We conclude that
\begin{align*} \pi_j(cv+dv') \amp = cw_j+dw_j \amp (\text{def. of } \pi_j) \\ \amp =c\pi_j(v)+d\pi_j(v')\text{,} \end{align*}
as desired.

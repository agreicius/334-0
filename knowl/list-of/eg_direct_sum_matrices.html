Example 1.5.11. Direct sum of matrix spaces.

Let \(V=F^{2,2}\text{,}\) the space of all \(2\times 2\) matrices with entries in \(F\text{,}\) and consider the following subsets of \(V\text{:}\)
\begin{align*} W_1 \amp =\{A\in F^{2,2}\colon A^T=A\}\\ W_2 \amp = \{A\in F^{2,2}\colon A^T=-A\}\text{.} \end{align*}
In other words, \(W_1\) is the set of all symmetric matrices of \(F^{2,2}\) and \(W_2\) is the set of all skew-symmetric matrices of \(F^{2,2}\text{.}\)
  1. Prove that \(W_1\) and \(W_2\) are subspaces of \(F^{2,2}\text{.}\)
  2. Prove that \(F^{2,2}=W_1\oplus W_2\text{.}\)
Solution.
  1. Using the defining conditions \(A^T=A\) and \(A^T=-A\text{,}\) it is easy to see that
    \begin{align*} W_1 \amp =\left\{\begin{bmatrix} a\amp b \\ b\amp c\end{bmatrix}\colon a,b,c\in F\right\}\\ W_2 \amp =\left\{\begin{bmatrix} 0\amp b \\ -b\amp 0\end{bmatrix}\colon b\in F\right\}\text{.} \end{align*}
    From these descriptions it follows easily that \(W_1\) and \(W_2\) contain the zero matrix, and are closed under addition and scalar multiplication. We leave the details to the reader.
  2. Let \(W=W_1+W_2\text{.}\) First we show that \(W=W_1\oplus W_2\text{.}\) To do so, it suffices to show \(W_1\cap W_2=\{\boldzero\}\) by CorollaryÂ 1.5.8. But this is clear from the parametric descriptions of \(W_1\) and \(W_2\) given above.
    Next we show that \(W=F^{2,2}\text{.}\) By definition we have \(W=W_1+W_2\subseteq F^{2,2}\text{,}\) so we need only show the reverse inclusion. Given an arbitrary matrix \(A\in F^{2,2}\text{,}\) we must show that we can write \(A=A_1+A_2\) where \(A_1\in W_1\) and \(A_2\in W_2\text{.}\) In other words, we must show that any matrix can be written as a sum of a symmetric matrix and a skew-symmetric matrix. This can be done essentially by inspection in this low-dimensional case (\(n=2\)): we have
    \begin{equation*} \begin{bmatrix} a\amp b\\ c\amp d \end{bmatrix} =\frac{1}{2}\begin{bmatrix} 2a\amp b+c \\ b+c \amp 2d \end{bmatrix}+ \frac{1}{2}\begin{bmatrix} 0\amp b-c \\ -b+c \amp 0 \end{bmatrix} \end{equation*}
    for any matrix \(A=\begin{bmatrix} a\amp b\\ c\amp d \end{bmatrix}\text{.}\) This fact is true more generally, however, and so it is worth adding an argument that works for all \(n\text{:}\) namely, observe that \(A=A_1+A_2\text{,}\) where
    \begin{align*} A_1 \amp =\frac{1}{2}(A+A^T)\\ A_2 \amp =\frac{1}{2}(A-A^T)\text{.} \end{align*}
    Using elementary properties of transpose, it is now easy to show that \(A_1^T=A_1\) and \(A_2^T=-A_2\text{:}\) i.e., \(A_1\in W_1\) and \(A_2\in W_2\text{.}\) We will remind you of these elementary properties of the transpose operator later on.

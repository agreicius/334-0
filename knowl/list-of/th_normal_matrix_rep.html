Theorem 1.22.6. Normality and matrix representations.

Let \((V,\angvec{\, ,})\) be a finite-dimensional inner product space, let \(T\in \mathcal{L}(V)\text{,}\) let \(B\) be an orthonormal basis of \(V\text{,}\) and let \(A=[T]_B\text{.}\)
  • \(T\) is normal if and only if \(A^*A=AA^*\text{.}\)
  • \(T\) is self-adjoint if and only if \(A^*=A\text{.}\)
  • \(T\) is skew-symmetric if and only if \(A^*=-A\text{.}\)
  • \(T\) is unitary if and only if \(A\) is invertible and \(A^*=A^{-1}\text{.}\)

Proof.

The equivalences are simple consequences of the following facts:
  • \([TS]_B=[T]_B[S]_B\) for all \(S,T\in \mathcal{L}(V)\) (Corollary 1.13.17);
  • \([T^*]_B=[T]_B^*\) for all \(T\in \mathcal{L}(V)\) (Corollary 1.21.11).

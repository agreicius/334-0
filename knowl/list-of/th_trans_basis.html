Theorem 1.9.21. Transformations and bases.

Let \(V\) and \(W\) be vector spaces, and let \(B=(v_i)_{i\in I}\) be a basis of \(V\text{.}\)
  1. Give linear transformations \(T_1,T_2\in \mathcal{L}(V,W)\text{,}\) we have \(T_1=T_2\) if and only if \(T_1(v_i)=T_2(v_i)\) for all \(i\in I\text{.}\)
  2. Given any tuple \((w_i)_{i\in I}\) of vectors of \(W\text{,}\) there is a linear transformation \(T\colon V\rightarrow W\) satisfying
    \begin{equation} T(v_i)=w_i\tag{1.24} \end{equation}
    for all \(i\in I\text{.}\) Furthermore, as a consequence of (1), this linear transformation \(T\) is unique.

Proof.

We assume \(B=(v_i)_{i\in I}\) is a basis of \(V\text{.}\)
  1. Assume \(T_1,T_2\in \mathcal{L}(V,W)\) and that \(T_1(v_i)=T_2(v_i)\) for all \(i\in I\text{.}\) Since \(B\) is a basis, given any \(v\in V\) we can write \(v\) as a linear combination
    \begin{equation*} v=\sum_{i\in I}c_iv_i \end{equation*}
    for some \(c_i\in F\text{,}\) in which case we have
    \begin{align*} T_1(v) \amp =\sum_{i\in I}c_iT_1(v_i) \amp (T_1 \text{ is linear})\\ \amp = \sum_{i\in I}c_iT_2(v_i) \amp (T_1(v_i)=T_2(v_i))\\ \amp = T_2(v) \amp (T_2 \text{ is linear})\text{.} \end{align*}
    We have shown that \(T_1(v)=T_2(v)\) for all \(v\in V\text{.}\) By definition, this means that \(T_1=T_2\text{.}\)
  2. Given a tuple \((w_i)_{i\in i}\) of vectors of \(W\text{,}\) the following recipe defines a function \(T\colon V\rightarrow W\text{:}\)
    • given \(v\in V\text{,}\) write \(v=\sum_{i\in I}c_iv_i\) as a linear combination of the vectors \(v_i\text{;}\)
    • define \(T(v)=\sum_{i\in I}c_iw_i\text{.}\)
    We need to address two details of this recipe in order to convince ourselves that this determines a well-defined function \(T\text{.}\) Firstly, since \(B\) is a basis, the coefficients \(c_i\) in the linear combination \(v=\sum_{i\in I}c_iv_i\) are uniquely determined by \(v\text{,}\) making the value \(T(v)=\sum_{i\in I}c_iw_i\) uniquely determined. Secondly, observe that since \(c_i=0\) for all but finitely many \(i\in I\text{,}\) the expression \(\sum_{i\in I}c_iw_i\) is a (finite) linear combination of vectors, and thus defines an element of \(W\text{.}\)
    From our definition of \(T\) it follows that \(T(v_i)=w_i\) for all \(i\in I\text{.}\) Indeed, to compute \(T(v_i)\) following our recipe, we write \(v_i=1v_i\) as a linear combination of the elements of \(B\text{,}\) whence it follows that \(T(v_i)=1w_i=w_i\text{.}\)
    Lastly, we must show that \(T\) is linear using the 1-step technique. Given \(v, w\in V\) and scalars \(c,d\in F\text{,}\) write
    \begin{align*} v \amp =\sum_{i\in I}c_iv_i\\ w \amp =\sum_{i\in I}d_iv_i \end{align*}
    as linear combinations of the \(v_i\text{,}\) and then compute
    \begin{align*} T(cv+dw) \amp =T\left(\sum_{i\in i}cc_iv_i+\sum_{i\in I}dd_iv_i\right)\\ \amp= T\left(\sum_{i\in i}(cc_i+dd_i)v_i\right)\\ \amp = \sum_{i\in I}(cc_i+dd_i)w_i \amp (\text{def. of } T)\\ \amp =c\sum_{i\in i}c_iw_i+d\sum_{i\in I}d_iw_i\\ \amp =cT(v)+dT(w) \amp (\text{def. of } T)\text{.} \end{align*}

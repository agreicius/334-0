Definition 1.12.3. Isomorphism.

A linear map \(T\in \mathcal{L}(V,W)\) is an isomorphism if there is a linear transformation \(S\in \mathcal{L}(W,V)\) satisfying \(ST=I_V\) and \(TS=I_W\text{.}\) When this is the case we call \(S\) the inverse of \(T\text{,}\) and write \(S=T^{-1}\text{.}\) Two vector spaces \(V\) and \(W\) are isomorphic if there is an isomorphism \(T\colon V\rightarrow W\text{.}\)

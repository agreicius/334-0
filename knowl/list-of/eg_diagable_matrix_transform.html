Example 1.17.8. Matrix transformation.

Let \(T\colon \R^3\rightarrow \R^3\) be the matrix transformation \(T=T_A\text{,}\) where
\begin{equation*} A=\begin{bmatrix} 1\amp 0\amp 1 \\0\amp 1\amp 0\\ 0\amp 0\amp 2 \end{bmatrix}\text{.} \end{equation*}
Decide whether \(T_A\) is diagonalizable; if it is, find an invertible matrix \(P\) that diagonalizes \(A\) and compute \(D=P^{-1}AP\text{.}\)
Solution.
We compute the characteristic polynomial \(f\) as
\begin{align*} f(x) \amp = \det(xI-A)\\ \amp = \det \begin{amatrix}[ccc] x-1\amp 0 \amp -\\ 0 \amp x-1\amp 0 \\ 0 \amp 0\amp x-2 \end{amatrix} \\ \amp = (x-1)^2(x-2)\text{.} \end{align*}
The distinct eigenvalues of \(A\) are thus \(1\) and \(2\text{.}\) We compute
\begin{align*} E(1,T_A) \amp = \NS(I-A)\\ \amp =\NS \begin{amatrix}[ccc] 0\amp 0\amp -1\\ 0 \amp 0\amp 0 \\ 0 \amp 0\amp -1 \end{amatrix}\\ \amp = \Span((1,0,0), (0,1,0))\\ E(2,T_A) \amp = \NS(2I-A)\\ \amp =\NS \begin{amatrix}[ccc] 1\amp 0 \amp -1\\ 0 \amp 1\amp 0 \\ 0 \amp 0\amp 0 \end{amatrix}\\ \amp = \Span((1,0,1))\text{.} \end{align*}
Since
\begin{align*} \dim E(1,T_A)+\dim E(2,T_A) \amp = 2+1=3\text{,} \end{align*}
we conclude that \(T_A\) is diagaonalizable. Furthermore we obtain an eigenbasis
\begin{equation*} B'=((1,0,0),(0,1,0),(1,0,1)) \end{equation*}
by concantenating the bases for our two eigenspaces. Let \(B\) be the standard basis of \(\R^3\text{.}\) By the change of basis formula, we have
\begin{equation*} [T_A]_{B'}=(\underset{B'\to B}{P})^{-1}[T_A]_B\underset{B'\to B}{P}=(\underset{B'\to B}{P})^{-1}A\underset{B'\to B}{P}\text{.} \end{equation*}
Since \(B'\) is an eigenbasis, we easily compute
\begin{equation*} D=[T_A]_{B'}=\begin{bmatrix} 1\amp 0\amp 0\\ 0\amp 1\amp 0\\ 0\amp 0\amp 2 \end{bmatrix}\text{.} \end{equation*}
Thus we have
\begin{equation*} D=P^{-1}AP, \hspace{10pt} P=\underset{B'\to B}{P}=\begin{bmatrix} 1\amp 0\amp 0\\ 0\amp 1\amp 0\\ 1\amp 0\amp 1 \end{bmatrix}\text{.} \end{equation*}

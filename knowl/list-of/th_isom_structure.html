Theorem 1.12.9. Isomorphisms preserve structure.

Let \(T\colon V\rightarrow W\) be an isomorphism.
  1. A tuple \(S=(v_i)_{i\in I}\) of vectors in \(V\) is linearly independent if and only if its image \(T(S)=(T(v_i))_{i\in I}\) is linearly independent.
  2. Let \(W=\Span S\text{,}\) where \(S=(v_i)_{i\in I}\) is a tuple in \(V\text{.}\) We have
    \begin{equation} T(W)=T(\Span S)=\Span T(S)=\Span (T(v_i))_{i\in I}\text{.}\tag{1.29} \end{equation}
  3. \(B=(v_i)_{i\in I}\) is a basis of \(V\) if and only if \(T(B)=(T(v_i))_{i\in I}\) is a basis of \(W\text{.}\)
  4. \(V\) is finite-dimensional if and only if \(W\) is finite-dimensional, and in this case we have \(\dim V=\dim W\text{.}\)
  5. \(U\) is a subspace of \(V\) if and only if \(T(U)\) is a subspace of \(W\text{.}\)
  6. We have \(V=V_1\oplus V_2\) for subspaces \(V_i\subseteq V\) if and only if \(W=T(V_1)\oplus T(V_2)\text{.}\)

Proof.

Most of the statements of this theorem follow almost immediately from the observation that since \(T\) is an isomorphism we have
\begin{equation*} \sum_{i\in I}c_iv_i=v \iff \sum_{i\in I}c_iT(v_i)=T(v) \end{equation*}
for any linear combination expression \(\sum_{i\in I}c_iv_i=v\) in \(V\text{,}\) along with the analogous statement for \(T^{-1}\text{.}\) Thus, for example, we have
\begin{align*} w\in T(\Span S) \amp \iff w=T(v) \text{ for some } v\in \Span S\\ \amp \iff w=T(\sum_{j=1}^nc_{i_j}v_{i_j})=\sum_{j=1}^nc_{i_j}T(v_{i_j}) \\ \amp \iff w\in \Span T(S)\text{.} \end{align*}
This proves (2). The proof of (1) is similar, and (3) and (4) follow immediately from (1) and (2). Statement (5) is a result of the fact that \(T(U)=\im T\vert_U\) and \(U=\im T^{-1}\vert_{T(U)}\text{.}\) We leave (6) as an exercise for the reader.

Theorem 1.12.10. Isomorphisms preserve structure.

Let \(T\colon V\rightarrow W\) be an isomorphism.
  1. A tuple \(S=(v_i)_{i\in I}\) of vectors in \(V\) is linearly independent if and only if its image \(T(S)=(T(v_i))_{i\in I}\) is linearly independent.
  2. Let \(W=\Span S\text{,}\) where \(S=(v_i)_{i\in I}\) is a tuple in \(V\text{.}\) We have
    \begin{equation} T(W)=T(\Span S)=\Span T(S)=\Span (T(v_i))_{i\in I}\text{.}\tag{1.29} \end{equation}
  3. \(B=(v_i)_{i\in I}\) is a basis of \(V\) if and only if \(T(B)=(T(v_i))_{i\in I}\) is a basis of \(W\text{.}\)
  4. \(V\) is finite-dimensional if and only if \(W\) is finite-dimensional, and in this case we have \(\dim V=\dim W\text{.}\)
  5. \(U\) is a subspace of \(V\) if and only if \(T(U)\) is a subspace of \(W\text{.}\)
  6. We have \(V=V_1\oplus V_2\) for subspaces \(V_i\subseteq V\) if and only if \(W=T(V_1)\oplus T(V_2)\text{.}\)

Proof.

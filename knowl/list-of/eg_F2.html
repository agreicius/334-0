Example 1.1.18. Field of two elements.

Let \(F=\{a,b\}\) be a set containing two distinct elements. Define addition and multiplication on \(F\) as follows:
\begin{align*} a+a \amp = a \amp a+b\amp = b \amp b+a \amp = b \amp b+b\amp = a\\ aa \amp = a \amp ab\amp= a \amp ba\amp = a \amp bb=b\text{.} \end{align*}
Verify that \(F\text{,}\) together with these two operations, constitutes a field. Make explicit which elements of \(F\) serve as the additive/multiplicative identities/inverses.
Solution.
Since \(F\) only has two elements, showing that Axioms (i)-(iii) hold is a simple matter of looking at all the possible cases: in other words, we can resort to a proof by exhaustion for these axioms. For example, verifying commutativity of multiplication requires looking at just 4 different cases. Furthermore, since it is trivially true that \(a a=aa\) and \(bb=bb\text{,}\) we need only look at the “mixed products” \(ab\) and \(ba\text{;}\) and in this case we see by the definition that
\begin{equation*} ab=a=ba\text{.} \end{equation*}
Similar arguments can be mounted for the rest of these arithmetic axioms.
We now turn to the existential axioms. First, we claim that \(a\) serves as the additive identity in this case: i.e., we have \(0=a\) for this field. To verify this, we need to show that \(a\) satisfies the defining axiom of the additive identity: namely, that \(a+\alpha=\alpha\) for any \(\alpha\in F\text{.}\) But this follows directly from the very definition of addition on \(F\text{,}\) as we declared that \(a+a=a\) and\(a+b=b\text{.}\) A similar argument shows that the element \(b\) acts as the multiplicative identity in this field: i.e., we have \(1=b\text{.}\)
Lastly we must show that each element of \(F\) has an additive inverse satisfying Axiom (v), and that each nonzero element of \(F\) has a multiplicative inverse satisfying Axiom (vii). Again, this is accomplished simply by inspection, building on our already established result that \(a=0\) and \(b=1\text{.}\) In detail, since \(a+a=a=0\) and \(b+b=a=0\text{,}\) we see that \(-a=a\) and \(-b=b\text{.}\) Turning at last to multiplicative inverses, note first that \(b\) is the only nonzero element of \(F\) (since \(a=0\)). Furthermore, since \(bb=b=1\text{,}\) we see that \(b^{-1}=b\text{.}\)
Since the operations on \(F\) satisfy the field axioms, we conclude that \(F\) is a field with respect to these operations. Having identified \(a=0\) and \(b=1\text{,}\) we can summarize the arithemetic rules of this particular field as follows:
\begin{align*} 0+0 \amp =0 \amp 0+1\amp=1+0=1 \amp 1+1\amp =0 \\ 0\cdot 0 \amp = 0 \amp 0\cdot 1\amp =1\cdot 0=0 \amp 1\cdot 1\amp =1\\ -0 \amp =0 \amp -1\amp =1 \amp 1^{-1}\amp =1\text{.} \end{align*}

Procedure 1.12.6. Isomorphisms.

Assume \(T\colon V\rightarrow W\) is a linear transformation. Below you find three separate approaches for deciding whether \(T\) is an isomorphism. Depending on the context, one approach may be more convenient than the others.
  1. Invertibility.
    Provide an inverse function \(T^{-1}\colon W\rightarrow V\text{,}\) or prove that no such inverse function exists. By TheoremÂ 1.12.5, an inverse function is automatically linear.
  2. Bijection.
    Determine whether \(T\) is bijective. This can be accomplished by computing \(\NS T\) and \(\im T\text{.}\)
  3. \(\dim V=\dim W< \infty\).
    If \(\dim V=\dim W< \infty\text{,}\) then the following statements are equivalent.
    1. \(T\) is an isomorphism.
    2. \(T\) is injective (equivalently, \(\NS T=\{\boldzero\}\)).
    3. \(T\) is surjective (equivalently, \(\Im T=W\)).

Example 1.3.3. Real-valued functions.

Let \(I=[a,b]\subseteq \R \) be an interval in the real line. Using the function interpretation of \(F^I\text{,}\) \(\R^{[a,b]}\) is the set of all functions of the form \(f\colon [a,b]\rightarrow \R\text{.}\) The vector operations defined in DefinitionÂ 1.3.1 in this setting correspond to the function-arithmetic operations you met in precalculus and calculus: i.e., given functions \(f,g\in \R^{[a,b]}\text{,}\) their vector sum \(f+g\) is the function defined as \((f+g)(x)=f(x)+g(x)\) for all \(x\in [a,b]\text{.}\) Similarly, given a scalar \(c\in \R\) and function \(f\in \R^{[a,b]}\text{,}\) the vector scalar multiple \(cf\) is the function defined as \((cf)(x)=cf(x)\) for all \(x\in \R\text{.}\)
In this context it is possible to visualize our vector operations in terms of the graphs of the functions on the interval \([a,b]\text{.}\)

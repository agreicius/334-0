Definition 1.9.24. Space of linear transformations.

Let \(V\) and \(W\) be vector spaces. Below we define a vector space structure on \(\mathcal{L}(V,W)\text{.}\)
  • Vector operations.
    Given linear transformations \(T,S\in \mathcal{L}(V,W)\) and scalar \(c\in F\text{,}\) we define linear transformations \(T+S, cT\in \mathcal{L}(V,W)\) as follows:
    \begin{align*} (T+S)(v) \amp= T(v)+S(v) \text{ for all } v\in V \\ (cT)(v) \amp =cT(v) \text{ for all } v\in V\text{.} \end{align*}
  • Zero vector.
    The zero vector of \(\mathcal{L}(V,W)\) is the zero transformation \(0_{V,W}\colon V\rightarrow W\) defined as \(0_{V,W}(v)=\boldzero\) for all \(v\in V\text{.}\)
  • Vector inverses.
    Given a linear transformation \(T\in \mathcal{L}(V,W)\text{,}\) its vector inverse is the linear transformation \(-T\in \mathcal{L}(V,W)\) defined as \((-T)(v)=-T(v)\) for all \(v\in V\text{.}\)
We call \(\mathcal{L}(V,W)\text{,}\) together with these vector operations, the space of linear transformations from \(V\) to \(W\text{.}\)

Theorem 1.21.20. Null space and image of adjoint.

Let \((V,\angvec{\, ,}_V)\) and \((W,\angvec{\, ,})\) be inner product spaces. Assume \(T\in \mathcal{L}(V,W)\) has adjoint \(T^*\in\mathcal{L}(W,V)\text{.}\)
  1. \(\NS T^*\).
    We have
    \begin{equation} \NS T^* =(\im T)^\perp\text{.}\tag{1.101} \end{equation}
  2. \(\im T^*\).
    We have
    \begin{equation} \im T^* \subseteq (\NS T)^\perp\text{.}\tag{1.102} \end{equation}
    If we assume further that \(\dim V< \infty\text{,}\) then
    \begin{equation} \im T^*=(\NS T)^\perp\text{.}\tag{1.103} \end{equation}

Proof.

Assume \(T^*\) exists. We first prove that \(\NS T^*=\im T^\perp\text{.}\) We have
\begin{align*} w\in \NS T^* \amp \iff T^*(w)=\boldzero\\ \amp \iff \angvec{T^*(w),v}=\boldzero \text{ for all } v\in V\\ \amp \iff \angvec{w,T(v)}=0 \text{ for all } v\in V \amp (\text{adj. prop.})\\ \amp \iff \angvec{w,u}=0 \text{ for all } u\in \im T \amp (\text{def. } \im T)\\ \amp \iff w\in \im T^\perp\text{.} \end{align*}
Thus \(\NS T^*=\im T^\perp\text{.}\)
To prove that \(\im T^*=\NS T^\perp\text{,}\) we use the fact that \(T=(T^*)^\perp\) and apply the result above to compute
\begin{align*} \NS T \amp =\NS (T^*)^*\\ \amp = (\im T^*)^\perp\text{.} \end{align*}
Thus \(\NS T=(\im T^*)^\perp\text{.}\) Since \((\im T^*)^\perp\subseteq V\) is finite dimensional, we have \(((\im T^*)^\perp)^\perp=\im T^*\text{,}\) and thus
\begin{equation*} \im T^*=((\im T^*)^\perp)^\perp=\NS T^\perp\text{,} \end{equation*}
as desired.

Procedure 1.26.7. Dot diagram.

Let \(T\in \mathcal{L}(V)\text{,}\) where \(V\) is finite dimensional. Fix an eigenvalue \(\lambda\text{,}\) and let \(B=B_{1}*B_{2}*\dots * B_{r}\) be a Jordan basis of \(G(\lambda, T)\text{,}\) where each \(B_{j}\) is a cycle of generalized \(\lambda\)-eigenvectors satisfying \(\len B_j\geq \len B_{j+1}\) for all \(1\leq j\leq r-1\text{.}\) The dot diagram of \(B\) is an staircase pattern of dots constructed in the manner described below. For all \(i\geq 1\text{,}\) let \(r_{i}\) be the number of dots in the \(i\)-th row of the dot diagram.
  • Dots are placed into a rectangular grid containing \(\len B_1\) rows and \(r\) columns. For all \(1\leq i\leq p\text{,}\) the \(i\)-th row contains \(r_{i}\) dots, and these are placed consecutively starting in the leftmost column.
  • We have
    \begin{equation*} r_i=\dim \NS(T-\lambda I)^{i}-\dim\NS(T-\lambda I)^{i-1}. \end{equation*}
    for all \(1\leq i\text{.}\)
The dot diagram is interpreted as follows.
  • For any \(k\geq 1\text{,}\) the total number of dots in the first \(i\) rows of the diagram is equal to \(\dim \NS (T-\lambda I)^{k}\text{:}\) i.e.,
    \begin{equation*} \dim \NS (T-\lambda I)^k=\sum_{i=1}^kr_i\text{.} \end{equation*}
    In particular, we have
    \begin{equation*} r=r_1=\dim \NS(T-\lambda I)\text{.} \end{equation*}
    Thus the number of \(\lambda\)-Jordan blocks of \(T\) is equal to the number of dots in the first row of the diagram, which is equal to \(\dim \NS(T-\lambda I)\text{.}\)
  • For each \(1\leq j\leq r\text{,}\) the number of dots \(c_j\) in the \(j\)-th column of the diagram is equal to the length of \(B_j\text{.}\) Equivalently, \(c_j\) is the size of the \(j\)-th \(\lambda\)-Jordan block.

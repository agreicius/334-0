Definition 1.4.6. Linear combination.

Let \(V\) be an \(F\)-vector space, and let \(v_1,v_2,\dots, v_n\) be a collection of vectors of \(V\text{.}\) A linear combination of the vectors \(v_i\) is a vector expression of the form
\begin{equation} c_1v_1+c_2v_2+\cdots +c_nv_n\text{,}\tag{1.5} \end{equation}
where \(c_i\in F\) for all \(1\leq i\leq n\text{.}\) The scalars \(c_i\) are called the coefficients of the linear combination. The linear combination (1.5) is trivial if \(c_i=0\) for all \(1\leq i\leq n\text{,}\) and nontrivial if \(c_i\ne 0\) for some \(1\leq i\leq n\text{.}\) A vector \(v\) is a linear combination of the \(v_i\) if we have
\begin{equation*} v=c_1v_1+c_2v_2+\cdots +c_nv_n \end{equation*}
for some choice of scalars \(c_i\in F\text{.}\)

Example 1.9.14. Matrix transformation.

Let \(A\in F^{m,n}\text{.}\) Prove that \(T_A\colon F^n\rightarrow F^m\) is a linear transformation.
Solution.
We use the 1-step technique. As you will see the proof follows directly from elementary properties of matrix multiplication. Given any \(\boldx,\boldy\in F^n\) and \(c,d\in F\text{,}\) we have
\begin{align*} T_A(c\boldx+d\boldy) \amp =A(c\boldx+d\boldy) \amp (\text{def. of } T_A)\\ \amp = cA\boldx+dA\boldy \amp (\text{matrix mult. props.})\\ \amp = cT_A(\boldx)+dT_A(\boldy) \amp (\text{def. of } T_A)\text{.} \end{align*}

Corollary 1.15.11.

Let \(V\) be finite dimensional, let \(T\in\mathcal{L}(V)\text{,}\) and let \(\lambda\in F\text{.}\) The following statements are equivalent.
  1. \(\lambda\) is an eigenvalue of \(T\text{.}\)
  2. \(T-\lambda I_V\) is not injective. (Equivalently, \(\NS (T-\lambda I_V)\ne \{\boldzero\}\text{.}\))
  3. \(T-\lambda I_V\) is not surjective. (Equivalently, \(\im (T-\lambda I_V)\ne V\text{.}\))
  4. \(T-\lambda I_V\) is not invertible.

Proof.

Let \(S=T-\lambda I_V\text{.}\) The additional equivalent statements we obtain when \(\dim V<\infty\) are the result of the fact (special to this setting) that \(S\) is invertible if and only if it is injective if and only if it is surjective.

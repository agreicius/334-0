Definition 1.1.11. Field.

A field is a set \(F\) together with two operations
\begin{align*} +\colon F\times F \amp\rightarrow F \amp \times\colon F\times F\amp \rightarrow F \\ (\alpha,\beta)\amp\mapsto \alpha+\beta \amp (\alpha,\beta)\amp \mapsto \alpha\,\beta \end{align*}
called addition and multiplication respectively, satisfying the following axioms.
  1. Commutativity.
    For all \(\alpha,\beta\in F\) we have
    \begin{align*} \alpha+\beta \amp = \beta+\alpha \amp \alpha\beta=\beta\alpha\text{.} \end{align*}
  2. Associativity.
    For all \(\alpha,\beta,\gamma \in F\) we have
    \begin{align*} \alpha+(\beta+\gamma)\amp =(\alpha+\beta)+\gamma \amp \alpha(\beta\gamma)=(\alpha\beta)\gamma\text{.} \end{align*}
  3. Distributivity.
    For all \(\alpha,\beta,\gamma \in F\) we have
    \begin{equation*} \alpha(\beta+\gamma)=\alpha\beta+\alpha\gamma\text{.} \end{equation*}
  4. Additive identity.
    There is an element \(0\in F\) satisfying
    \begin{equation*} 0+\alpha=\alpha \end{equation*}
    for all \(\alpha\in F\text{.}\) The element \(0\) is called the additive identity.
  5. Additive inverse.
    For all \(\alpha\in F\) there is an element \(-\alpha\in F\) satisfying
    \begin{equation*} \alpha+ -\alpha=0\text{.} \end{equation*}
    The element \(-\alpha\) is called the additive inverse of \(\alpha\text{.}\)
  6. Multiplicative identity.
    There is an element \(1\in F\) satisfying
    \begin{equation*} 1\times \alpha=\alpha \end{equation*}
    for all \(\alpha\in F\text{.}\) The element \(1\) is called the multiplicative identity.
  7. Multiplicative inverse.
    For all nonzero elements \(\alpha\in F\) (i.e., \(\alpha\ne 0\)) there is an element \(\alpha^{-1}\in F\) satisfying
    \begin{equation*} \alpha\,\alpha^{-1}=1\text{.} \end{equation*}
    The element \(\alpha^{-1}\text{,}\) also denoted \(1/\alpha\text{,}\) is called the multiplicative inverse of \(\alpha\text{.}\)

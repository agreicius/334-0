Procedure 1.7.7. One-step technique for bases.

Let \(B=(v_i)_{i\in I}\) be a tuple of vectors of \(V\text{.}\) To decide whether \(B\) is a basis, proceed as follows.
  1. Write out the vector equation
    \begin{equation} \sum_{i\in I}c_iv_i=v\text{,}\tag{1.21} \end{equation}
    where \(\boldv\) is an arbitrary element of the vector space \(V\text{.}\) (Typically you will give some parametric description of \(v\text{.}\))
  2. Translate the vector equation (1.21) into an equivalent linear system of equations in the unknowns \(c_i\text{.}\)
  3. If the system in (2) has a unique solution no matter what \(v\) is chosen, then \(B\) is a basis. If there is a choice of \(v\) for which either the system in (2) has no solution (i.e., is inconsistent) or has infinitely many solutions, then \(B\) is not a basis.

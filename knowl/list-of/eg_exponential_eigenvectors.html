Example 1.17.11. Exponential functions as eigenvectors.

Let \(I\) be an interval of \(\R\) containing more than one element. For all \(\lambda\in \R\text{,}\) define \(f_\lambda\in C^\infty(\R)\) as \(f_\lambda(x)=e^{\lambda x}\text{.}\) Prove that \(S=(f_\lambda)_{\lambda\in \R}\) is an uncountably infinite linear independent tuple in \(C^{\infty}(I)\text{.}\)
Solution.
We saw in Example 1.15.10 that the functions \(f_\lambda\) are all eigenvectors of the differential operator \(T\colon C^\infty(I)\rightarrow C^\infty(I)\text{,}\) \(T(f)=f'\text{.}\) Since \(T(f_\lambda)=\lambda f_\lambda\) for all \(\lambda\text{,}\) we see that the functions \(f_\lambda\) all have distinct eigenvalues. We conclude from Theorem 1.17.10 that \((f_\lambda)_{\lambda\in \R}\) is a linearly independent tuple. Since this tuple is indexed by \(\R\) itself, we see that it is an uncountably infinite linear independent tuple.

Theorem 1.24.7. \(T\)-cyclic subspace.

Let \(T\in \mathcal{L}(V)\text{,}\) let \(v\in V\) be nonzero, and let \(W=\Span(T^i(v))_{i=0}^\infty\text{.}\)
  1. \(W\) is a \(T\)-invariant subspace of \(V\text{.}\)
  2. If \(W'\) is a \(T\)-invariant subspace of \(V\) containing \(v\text{,}\) then \(W\subseteq W'\text{:}\) i.e., \(W\) is the smallest \(T\)-invariant subspace of \(V\) containing \(v\text{.}\)
  3. If \(\dim W< \infty\text{,}\) then there is a \(k\geq 0\) such that
    \begin{equation*} B=(T^{i}(v))_{i=0}^{k-1}=(v,T(v),\dots, T^{k-1}(v)) \end{equation*}
    is a basis of \(W\text{.}\)

Proof.

  1. This is left to the reader.
  2. Assume \(W'\) is a \(T\)-invariant subspace containing \(v\text{.}\) Since \(W'\) is \(T\)-invariant, it contains \(T^{i}(v)\) for all \(i\in \Z_{\geq 0}\text{.}\) By Theorem 1.6.6, we have
    \begin{equation*} W=\Span(T^i(v))_{i=0}^\infty\subseteq W'\text{.} \end{equation*}
  3. If \(W\) is finite dimensional, then the tuple \((T^i(v))_{i=0}^\infty\) must be linearly dependent. It follows that \(T^k(v)\) can be written as a linear combination of the previous \(T^i(v)\) for some \(k\in \Z_{\geq 1}\) (\(v\) is assumed to be nonzero). Let \(k\) be minimal in this respect. The tuple \(B=(v,T(v),\dots, T^{k-1}(v))\) is then linearly independent by Corollary 1.6.14. We claim \(B\) spans \(W\text{,}\) and hence is a basis. Since \(\Span B\subseteq W\) by definition, we need only show that \(W\subseteq \Span B\text{.}\) By the previous result it suffices to show that \(W\) is \(T\)-invariant (since \(v\in W\)). This is easy: given any \(v\in \Span B\text{,}\) we have \(v=\sum_{i=0}^{k-1}a_iT^i(v)\) for some \(a_i\text{,}\) in which case
    \begin{equation*} T(v)=\sum_{i=0}^{k-1}a_iT^{i+1}(v_i)=\sum_{i=1}^{k-1}a_{i-1}T^i(v)+T^k(v)\in \Span B\text{,} \end{equation*}
    since \(\sum_{i=1}^{k-1}a_{i-1}T^i(v)\in \Span B\) and \(T^k(v)\in \Span B\text{.}\)

Theorem 1.8.14. Dimension theory compendium.

Let \(V\) be a vector space of dimension \(n\text{,}\) and let \(S\) be a tuple of vectors of \(V\text{.}\)
  1. Street smarts.
    If \(\len S< n\text{,}\) then \(S\) does not span \(V\text{.}\)
  2. Street smarts.
    If \(\len S> n\text{,}\) then \(S\) is linearly dependent.
  3. Street smarts.
    If \(\len S =n\text{,}\) then \(S\) is a basis if and only if \(S\) spans if and only if \(S\) is linearly independent.
  4. Contract.
    If \(S\) spans \(V\text{,}\) then \(S\) can be contracted to a basis of \(V\text{.}\)
  5. Expand.
    If \(S\) is linearly independent, then \(S\) can be extended to a basis of \(V\text{.}\)
  6. Dimension of subspaces.
    If \(W\subseteq V\) is a subspace, then
    1. \(\dim W\leq \dim V\text{,}\) and
    2. \(\dim W=\dim V\) if and only if \(W=V\text{.}\)
  7. Existence of direct sums.
    Given any subspace \(W\subseteq V\text{,}\) there is a subspace \(W'\) satisfying \(V=W\oplus W'\text{.}\)
  8. Dimension of direct sums.
    Given \(W=\sum_{i=1}^r W_i\text{,}\) where \(W_i\) is a subspace of \(V\) for all \(1\leq i\leq k\text{,}\) we have
    \begin{equation*} W=\bigoplus_{i=1}^rW_i \iff \dim W=\sum_{i=1}^r\dim W_i\text{.} \end{equation*}

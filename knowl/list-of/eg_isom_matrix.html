Example 1.12.11. Matrix transformations.

Prove that a matrix transformation \(T_A\colon F^n\rightarrow F^m\) is an isomorphism if and only if \(m=n\) and \(A\in F^{n,n}\) is invertible.
Solution.
We know from Example 1.9.14 that matrix transformations are linear.
Assume \(m=n\) and \(A\in F^{n,n}\) is invertible. We claim that \(T_{A^{-1}}=T_A^{-1}\text{.}\) Indeed, for all \(\boldx\in F^n\text{,}\) we have
\begin{align*} T_AT_{A^{-1}}\boldx \amp =T_A(A^{-1}\boldx)\\ \amp =AA^{-1}\boldx=\boldx\\ T_{A^{-1}}T_{A}\boldx \amp =T_{A^{-1}}(A\boldx)\\ \amp =A^{-1}A\boldx=\boldx\text{.} \end{align*}
This proves that \(T_AT_{A^{-1}}=T_{A^{-1}}T_A=I_V\) and thus that \(T_{A^{-1}}=T_A^{-1}\text{,}\) as claimed. We conclude \(T_A\) is an isomorphism.
Lastly assume \(T_A\) is an isomorphism. By Corollary 1.12.10, we conclude that \(\dim F^n=\dim F^m\text{,}\) and hence that \(n=m\text{.}\) Since \(T_A\) is an isomorphism, it has a linear inverse \(S\colon F^n\rightarrow F^n\text{;}\) since all linear transformations of \(F^n\) are matrix transformations, we have \(S=T_B\) for some matrix \(B\in F^{n,n}\text{.}\) We have
\begin{align*} T_AT_B=I_{F^n} \amp \implies AB\boldx=\boldx \text{ for all } \boldx\in F^n\implies AB=I\\ T_BT_A=I_{F^n} \amp \implies BA\boldx=\boldx \text{ for all } \boldx\in F^n\implies BA=I\text{.} \end{align*}
Thus \(B=A^{-1}\text{,}\) showing that \(A\) is invertible.

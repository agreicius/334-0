Theorem 1.21.2. Riesz representation.

Let \((V,\angvec{\, ,})\) be an inner product space.
  1. For all \(v_0\in V\text{,}\) the function \(\phi_{v_0}\colon V\rightarrow F\) defined as \(\phi_{v_0}(v)=\angvec{v,v_0}\) is a linear functional.
  2. The function
    \begin{align*} \Phi\colon V \amp \rightarrow V^*=\mathcal{L}(V,F)\\ v \amp \mapsto \phi_v \text{,} \end{align*}
    where \(\phi_v(v')=\angvec{v',v}\text{,}\) is an injective linear transformation.
  3. Assume \(\dim V< \infty\text{.}\) In this case the map \(\Phi\) above is an isomorphism. In particular, given a linear functional \(\phi\colon V\rightarrow F\text{,}\) there is a unique vector \(v_0\in V\) such that \(\phi=\phi_{v_0}\text{:}\) i.e., such that \(\phi(v)=\angvec{v,v_0}\) for all \(v\in V\text{.}\)

Proof.

  1. This was already proved in Theorem 1.18.9. The current statement is just a slight reformulation of this fact, using the language of linear functionals.
  2. We use the 1-step technique. Given any \(v_1,v_2\in V\) and \(c,d\in F\text{,}\) the function \(\Phi(cv_1+dv_2)=\phi_{cv_1+dv_2}\) satisfies
    \begin{align*} \phi_{cv_1+dv_2}(v) \amp = \angvec{v,cv_1+dv+2} \\ \amp = c\angvec{v,v_1}+d\angvec{v,v_2}\\ \amp = c\phi_{v_1}(v)+d\phi_{v_2}(v)\\ \amp =(c\phi_{v_1}+d\phi_{v_2})(v) \end{align*}
    for all \(v\in V\text{.}\) Thus \(\phi_{cv_1+dv_2}=c\phi_{v_1}+d\phi_{v_2}\text{:}\) i.e., \(\Phi(cv_1+dv_2)=c\Phi(v_1)+d\Phi(v_2)\text{.}\)
    To show \(\Phi\) is injective, we will show that \(\NS \Phi=\{\boldzero\}\text{.}\) Given any \(v_0\in V\text{,}\) we have
    \begin{align*} \Phi(v_0)=\boldzero \amp \iff \phi_{v_0}=0_{V,F}\\ \amp \iff \phi_{v_0}(v)=0 \text{ for all } v\in V\\ \amp \iff \angvec{v,v_0}=0 \text{ for all } v\in V\\ \amp \iff v_0=\boldzero \amp (\knowl{./knowl/xref/th_inner_prod.html}{\text{1.18.9}}) \text{.} \end{align*}
    Thus \(\NS \Phi=\{\boldzero\}\) and \(\Phi\) is injective.
  3. Let \(\dim V=n< \infty\text{.}\) Since \(\dim F=1\text{,}\) by Theorem 1.13.19 we have
    \begin{equation*} \dim V^*=\dim \mathcal{L}(V,F)=n\cdot 1=n=\dim V\text{.} \end{equation*}
    Since \(\Phi\) is an injective linear transformation between two vector spaces of the same finite dimension, it follows from Corollary 1.11.4 that \(\Phi\) is bijective, and hence an isomorphism.

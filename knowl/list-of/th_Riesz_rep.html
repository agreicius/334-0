Theorem 1.21.18. Riesz representation.

Let \((V,\angvec{\, ,})\) be an inner product space.
  1. For all \(v_0\in V\text{,}\) the function \(\phi_{v_0}\colon V\rightarrow F\) defined as \(\phi_{v_0}(v)=\angvec{v,v_0}\) is a linear functional.
  2. The function
    \begin{align*} \Phi\colon V \amp \rightarrow V^*=\mathcal{L}(V,F)\\ v \amp \mapsto \phi_v \text{,} \end{align*}
    where \(\phi_v(v')=\angvec{v',v}\) is injective and satisfies
    \begin{equation*} \Phi(cv_1+dv_2)=\overline{c}v_1+\overline{d}v_2\text{.} \end{equation*}
    In other words, \(\Phi\) is conjugate linear.
  3. Assume \(\dim V< \infty\text{.}\) In this case the map \(\Phi\) above is bijective. In particular, given a linear functional \(\phi\colon V\rightarrow F\text{,}\) there is a unique vector \(v_0\in V\) such that \(\phi=\phi_{v_0}\text{:}\) i.e., such that \(\phi(v)=\angvec{v,v_0}\) for all \(v\in V\text{.}\)

Proof.

  1. This was already proved in Theorem 1.18.9. The current statement is just a slight reformulation of this fact, using the language of linear functionals.
  2. We use the 1-step technique. Given any \(v_1,v_2\in V\) and \(c,d\in F\text{,}\) the function \(\Phi(cv_1+dv_2)=\phi_{cv_1+dv_2}\) satisfies
    \begin{align*} \phi_{cv_1+dv_2}(v) \amp = \angvec{v,cv_1+dv+2} \\ \amp = \overline{c}\angvec{v,v_1}+\overline{d}\angvec{v,v_2}\\ \amp = \overline{c}\phi_{v_1}(v)+\overline{d}\phi_{v_2}(v)\\ \amp =(\overline{c}\phi_{v_1}+\overline{d}\phi_{v_2})(v) \end{align*}
    for all \(v\in V\text{.}\) Thus \(\phi_{cv_1+dv_2}=\overline{c}\phi_{v_1}+\overline{d}\phi_{v_2}\text{:}\) i.e., \(\Phi(cv_1+dv_2)=\overline{c}\Phi(v_1)+\overline{d}\Phi(v_2)\text{.}\)
    We now show \(\Phi\) is injective. Given \(v_1,v_2\in V\text{,}\) we have
    \begin{align*} \Phi(v_1)=\Phi(v_2) \amp \iff \phi_{v_1}=\phi_{v_2}\\ \amp \iff \phi_{v_1}(v)=\phi_{v_2}(v) \text{ for all } v\in V\\ \amp \iff \angvec{v,v_1}=\angvec{v,v_2} \text{ for all } v\in V\\ \amp \iff v_1=v_2 \amp (\knowl{./knowl/xref/th_inner_prod.html}{\text{1.18.9}})\text{.} \end{align*}
    Thus \(\Phi\) is injective.
  3. Let \(\dim V=n< \infty\text{.}\) Since \(\dim F=1\text{,}\) by Theorem 1.13.19 we have
    \begin{equation*} \dim V^*=\dim \mathcal{L}(V,F)=n\cdot 1=n=\dim V\text{.} \end{equation*}
    Since \(\Phi\) is conjugate linear, and not linear, we cannot invoke Corollary 1.11.4 directly to conclude that it is surjective. However, the same result does in fact apply to conjugate linear maps for the following two reasons (left to the reader to prove):
    • the image of a conjugate linear map is a subspace of the codomain (just as with linear maps);
    • the image of a linearly independent tuple under an injective conjugate linear map is a linearly independent tuple.
    Let \(B=(v_1,v_2,\dots, v_n)\) be a basis of \(V\text{.}\) From the two points above, we see that \(\im \Phi\) is a subspace of \(V^*\) containing the linearly independent tuple \(\Phi(B)=(\Phi(v_1),\dots, \Phi(v_n))\text{.}\) It follows that \(\dim \im \Phi=\dim V^*=n\text{,}\) and thus that \(\im \Phi=V^*\text{,}\) as desired.
    Alternatively, we can give a more constructive proof of the surjectivity of \(\Phi\) as follows:
    • pick an orthonormal basis \(B=(v_1,v_2,\dots, v_n)\) of \(V\text{;}\)
    • given \(\phi\in V^*=\mathcal{L}(V,F)\text{,}\) let \(c_i=\phi(v_i)\) for all \(i\in I\text{;}\)
    • let \(v=\sum_{i=1}^n\overline{c_i}v_i\text{;}\)
    • for all \(1\leq i\leq n\text{,}\) we have
      \begin{align*} \phi_v(v_i) \amp =\angvec{v_i, v}\\ \amp = \angvec{v_i, \overline{c}_iv_i} \amp (i\ne j\implies \angvec{v_i, c_jv_j}=0)\\ \amp = c_i\angvec{v_i,v_i} \amp (\text{conj. homog.})\\ \amp =c_i \amp (\norm{v_i}=1)\text{;} \end{align*}
    • since \(\phi\) and \(\phi_v\) agree on the basis \(B\text{,}\) we have \(\phi=\phi_v\text{.}\)

Example 1.26.12. Computing Jordan form.

Consider the matrix transformation \(T=T_A\colon \R^8\rightarrow \R^8\text{,}\) where
\begin{equation*} A=\begin{amatrix}[rrrrrrrr] 3 \amp 2 \amp 3 \amp 2 \amp 3 \amp -1 \amp -3 \amp -3 \\ -5 \amp -4 \amp -13 \amp -15 \amp -15 \amp 2 \amp 16 \amp 1 \\ -1 \amp -1 \amp -5 \amp -7 \amp -6 \amp 1 \amp 7 \amp -2 \\ -1 \amp -2 \amp 7 \amp 23 \amp 11 \amp -5 \amp -20 \amp 21 \\ 0 \amp 0 \amp 2 \amp -7 \amp -1 \amp 1 \amp 7 \amp -8 \\ 1 \amp 1 \amp 3 \amp 1 \amp 2 \amp 0 \amp -1 \amp -2 \\ -2 \amp -3 \amp 2 \amp 7 \amp 2 \amp -3 \amp -4 \amp 11 \\ -1 \amp -1 \amp -5 \amp -14 \amp -9 \amp 2 \amp 14 \amp -8 \end{amatrix}\text{.} \end{equation*}
Compute the characteristic polynomial \(f\) of \(A\text{.}\)
We see that \(f(x)=(x-2)^4(x+1)^4\text{.}\) This tells us that \(V=\R^8\) decomposes into two generalized eigenspaces
\begin{equation*} V=G(2,T)\oplus G(-1,T)\text{,} \end{equation*}
and moreover (looking at the multiplicities), that
\begin{equation*} \dim G(2,T)=\dim G(-1,T)=2\text{.} \end{equation*}
Now compute the dot diagram for \(\lambda=2\text{.}\) Since \(\dim(2,T)=4\text{,}\) we know that \(G(2, T)=\NS(T-2I)^4\text{,}\) and thus we only need to look at \(\dim \NS (T-2I)^i\) for \(1\leq i\leq 4\text{.}\)
Let \(r_i\) be the number of dots in the \(i\)-th row of our dot diagram for \(\lambda=2\text{.}\) From the computation above, we see that
\begin{align*} r_1 \amp =\dim \NS(A-2I)=3\\ r_2 \amp =\dim \NS(A-2I)^2-\dim \NS(A-2I)=1\\ r_i \amp =\dim \NS(A-2I)^i-\dim\NS(A-2I)^{i-1}=0, i\geq 3\text{.} \end{align*}
The dot diagram for \(\lambda=2\) is thus
\begin{equation*} {\Huge \begin{array}{rrr} \cdot \amp \cdot \amp \cdot \\ \cdot \amp \amp \end{array}} \end{equation*}
and the Jordan form of \(T\vert_{G(2, T)}\) breaks into 3 blocks of size \(2\times 2\text{,}\) \(1\times 1\text{,}\) and \(1\times 1\text{.}\)
\begin{equation*} \left[\begin{array}{rr|r|r} 2\amp 1 \amp 0 \amp 0\\ 0\amp 2\amp 0\amp 0 \\ \hline 0\amp 0\amp 2\amp 0 \\ \hline 0\amp 0\amp 0\amp 2 \end{array} \right]\text{.} \end{equation*}
Now compute the dot diagram for \(\lambda=-1\text{.}\)
The corresponding dot diagram now consists of a single column
\begin{equation*} {\Large \begin{array}{r} \cdot\\ \cdot\\ \cdot\\ \cdot \end{array} } \end{equation*}
and thus the Jordan form for \(T\vert_{G(-1,T)}\) consists of a single \(4\times 4\) Jordan block
\begin{equation*} \begin{bmatrix} -1\amp 1\amp 0\amp 0\\ 0\amp -1\amp 1\amp 0\\ 0\amp 0\amp -1\amp 1\\ 0\amp 0\amp 0\amp -1 \end{bmatrix}\text{.} \end{equation*}
In summary, we conclude that there is a Jordan basis \(B\) of the form
\begin{equation*} B=B_1*B_2*B_3*B_4\text{,} \end{equation*}
where
\begin{align*} B_1 \amp =((T-2I)v_1,v_1)\\ B_2 \amp =(v_2)\\ B_2 \amp =(v_3)\\ B_4 \amp =((T+I)^3(v_4),(T+I)^2(v_4),(T+I)(v_4),v_4) \end{align*}
for some nonzero vectors \(v_1,v_2,v_3,v_4\text{,}\) and we have
\begin{equation*} [T]_B=\left[\begin{array}{rr|r|r|rrrr} 2 \amp 1 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 2 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ \hline 0 \amp 0 \amp 2 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ \hline 0 \amp 0 \amp 0 \amp 2 \amp 0 \amp 0 \amp 0 \amp 0 \\ \hline 0 \amp 0 \amp 0 \amp 0 \amp -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp -1 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp -1 \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp -1 \end{array}\right]\text{.} \end{equation*}

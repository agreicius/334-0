Theorem 1.18.12. Cauchy-Schwarz inequality.

Let \((V,\angvec{\, ,})\) be an inner product space. We have
\begin{equation} \abs{\angvec{v,w}}\leq \norm{v}\norm{w}\tag{1.65} \end{equation}
for all \(v,w\in V\text{.}\) Furthermore, the inequality (1.65) is an equality if and only if one of \(v\) or \(w\) is a scalar multiple of the other.

Proof.

First observe that inequality (1.65) clearly holds if \(w=\boldzero\text{,}\) since in this case we have \(\abs{\angvec{v,w}}=\norm{v}\norm{w}=0\text{.}\)
We now show (1.65) holds when \(w\ne \boldzero\text{.}\) For any scalar \(c\in F\text{,}\) we have
\begin{equation} \angvec{v+cw,v+cw} \geq 0\text{.}\tag{1.66} \end{equation}
Let’s unpack this inequality using additional properties of the inner product:
\begin{align*} \angvec{v+cw,v+cw} \amp \geq 0\\ \angvec{v,v}+\angvec{cv,w}+\angvec{v,cw}+\angvec{cw,cw} \amp \geq 0\\ \angvec{v,v}+c\angvec{w,v}+\overline{c}\angvec{v,w}+c\overline{c}\angvec{w,w} \amp \geq 0 \\ \norm{v}^2+c\overline{\angvec{v,w}}+\overline{c}\angvec{v,w}+\abs{c}^2\norm{w}^2 \amp \geq 0 \text{.} \end{align*}
Assume \(w\ne \boldzero\text{.}\) Since the last inequality is true for any scalar \(c\in F\text{,}\) it is true for the particular choice choice \(c=-\frac{\angvec{v,w}}{\norm{w}^2}\text{.}\) (Note that \(c\) is well defined since \(w\ne \boldzero\text{,}\) and hence \(\norm{w}\ne 0\text{.}\)) We now substitute this expression in for \(c\) and do some further simplification:
\begin{align*} \norm{v}^2-\frac{\angvec{v,w}}{\norm{w}^2}\cdot\overline{\angvec{v,w}}-\frac{\overline{\angvec{v,w}}}{\norm{w}^2}\cdot\angvec{v,w}+\frac{\abs{\angvec{v,w}}^2}{\norm{w}^4}\cdot\norm{w}^2 \amp \geq 0 \\ \norm{v}^2-\frac{\abs{\angvec{v,w}}^2}{\norm{w}^2}- \cancel{\frac{\abs{\angvec{v,w}}^2}{\norm{w}^2}}+\cancel{\frac{\abs{\angvec{v,w}}^2}{\norm{w}^2}}\amp \geq 0 \\ \abs{\angvec{v,w}}^2 \leq \norm{v}^2\norm{w}^2\amp\text{.} \end{align*}
Taking square-roots of both sides (and using the fact that \(\sqrt{a^2}=a\) for all nonnegative real numbers \(a\)), we conclude that
\begin{equation*} \abs{\angvec{v,w}}\leq \norm{v}\norm{w}\text{,} \end{equation*}
as desired.
Having proved that (1.65) holds for all \(v,w\in V\text{,}\) we now show that we have equality if and only one of the vectors is a scalar multiple of the other. The reverse implication is easy to see: if \(v=cw\text{,}\) for example, then we have
\begin{align*} \abs{\angvec{v,w}} \amp = \abs{\angvec{cw,w}}=\abs{c}\norm{w}^2 \\ \norm{v}\norm{w} \amp =\norm{cw}\norm{w}=\abs{c}\norm{w}^2\text{,} \end{align*}
showing that equality holds in (1.65). The argument in the case \(w=cv\) is very similar. (We “cheated” somewhat above, using property (2) of Theorem 1.18.13. However, the proof of that property does not use Cauchy-Schwarz, so we are not indulging in circularity here.)
It remains only to show the forward implication. Assume \(\abs{\angvec{v,w}}=\norm{v}\norm{w}\text{;}\) we will show that one of the vectors is a scalar multiple of the other. If \(w=\boldzero\text{,}\) then the claim follows trivially, since \(w=0v\text{.}\) If \(w\ne \boldzero\text{,}\) then working our way backward through the various simplifications starting with (1.66), we see that
\begin{align*} \abs{\angvec{v,w}}=\norm{v}\norm{w}=0 \amp \iff \abs{\angvec{v,w}}^2=\norm{v}^2\norm{w}^2 \\ \amp\iff \angvec{v+cw,v+cw}=0, \hspace{10pt} c=-\frac{\angvec{v,w}}{\norm{w}^2} \\ \amp \implies v+cw=\boldzero, \hspace{10pt} c=-\frac{\angvec{v,w}}{\norm{w}^2}\\ \amp \iff v=-cw, \hspace{10pt} c=-\frac{\angvec{v,w}}{\norm{w}^2}\text{,} \end{align*}
where the last line follows from (3) of Theorem 1.18.9. Thus, one of the vectors (namely, \(v\)) is scalar multiple of the others in this case, as desired.

Definition 1.4.14. Function subspaces over \(\R\).

Let \(I\) be an interval of \(\R\) that contains at least two elements.
  1. We denote by \(C(I)\) the set of all continuous functions on \(I\text{:}\) i.e.,
    \begin{equation*} C(I)=\{f\in \R^I\colon f \text{ is continuous on }I\}\text{.} \end{equation*}
  2. Fix \(n\geq 1\text{.}\) A function \(f\colon I\rightarrow \R\) is \(C^{n}\) on \(I\) if \(f\) is \(n\)-times differentiable on \(I\) and its \(n\)-th derivative \(f^{(n)}(x)\) is continuous. The set of all \(C^{n}\) functions on \(I\) is denoted \(C^{n}(I)\text{.}\)
  3. A function \(f\colon I\rightarrow \R\) is \(C^{\infty}\) on I if \(f\) is infinitely differentiable on \(I\text{.}\) The set of all \(C^{\infty}\) functions on \(I\) is denoted \(C^{\infty}(I)\text{.}\)

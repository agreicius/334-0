Definition 1.2.11. Space of matrices.

Let \(F\) be a field. Given positive integers \(m,n\in\Z_+\text{,}\) we define \(F^{m,n}\) to be the set of all \(m\times n\) matrices with entries in \(F\text{:}\) i.e.,
\begin{equation*} F^{m,n}=\{\underset{m\times n}{A}=[a_{ij}]\colon a_{ij}\in F \text{ for all } (i,j)\in \{1,2\dots, m\}\times \{1,2,\dots, n\}\}\text{.} \end{equation*}
Below we define a vector space structure on \(F^{m,n}\text{.}\)
  • Vector operations.
    Given matrices \(A=[a_{ij}], B=[b_{ij}]\in F^{m,n}\) and \(c\in F\text{,}\) we define
    \begin{align*} A+B \amp =[a_{ij}+b_{ij}]\\ cA \amp =[ca_{ij}]\text{.} \end{align*}
  • Zero vector.
    The zero vector of \(F^{m,n}\) is the the zero matrix
    \begin{equation*} \boldzero=[0]_{\substack{1\leq i\leq m\\ 1\leq j\leq n}}\text{.} \end{equation*}
  • Vector inverses.
    Given a matrix \(A=[a_{ij}]\in F^{m,n}\) its vector inverse is
    \begin{equation*} -A=[-a_{ij}]\text{.} \end{equation*}
We call \(F^{m,n}\text{,}\) together with these vector operations, the space of \(m\times n\) matrices with entries in \(F\) (or the space of \(F\)-valued \(m\times n\) matrices).

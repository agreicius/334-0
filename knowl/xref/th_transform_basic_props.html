Theorem 1.9.9. Basic properties of linear transformations.

Let \(T\colon V\rightarrow W\) be a linear transformation.
  1. We have \(T(\boldzero)=\boldzero\text{.}\) In other words, \(T\) maps the zero vector of \(V\) to the zero vector of \(W\text{.}\)
  2. For all \(v\in V\text{,}\) we have \(T(-v)=-T(v)\text{.}\)
  3. Let \(S=(v_i)_{i\in i}\) be a tuple of vectors of \(V\text{.}\) Given any linear combination
    \begin{equation*} v=\sum_{i\in I}c_iv_i \end{equation*}
    of the \(v_i\text{,}\) we have
    \begin{equation*} T(v)=\sum_{i\in I}c_iT(v_i)\text{.} \end{equation*}

Proof.

  1. We employ some similar trickery to what was done in the proof of Theorem 1.3.6. Assuming \(T\) is linear:
    \begin{align*} T(\boldzero) \amp= T(\boldzero+\boldzero)\\ \amp =T(\boldzero)+T(\boldzero) \amp (\knowl{./knowl/xref/d_transformation.html}{\text{Definition 1.9.1}} ) \text{.} \end{align*}
    Thus, whatever \(T(\boldzero)\in W\) may be, it satisfies
    \begin{equation*} T(\boldzero)=T(\boldzero)+T(\boldzero)\text{.} \end{equation*}
    Canceling \(T(\boldzero)\) on both sides using \(-T(\boldzero)\text{,}\) we conclude
    \begin{equation*} \boldzero=T(\boldzero_V)\text{.} \end{equation*}
  2. The argument is similar:
    \begin{align*} \boldzero_W \amp= T(\boldzero_V) \amp (\text{by (1)})\\ \amp =T(-v+v)\\ \amp = T(-v)+T(v)\text{.} \end{align*}
    Since \(\boldzero_W=T(-v)+T(v)\text{,}\) adding \(-T(v)\) to both sides of the equation yields
    \begin{equation*} -T(v)=T(-v)\text{.} \end{equation*}
  3. Recall that by definition Definition 1.6.1 that there is a subset \(J=\{j_1,j_2,\dots, j_n\}\subseteq I\) such that \(c_i=0\) for all \(i\notin J\text{,}\) in which case we have we can write
    \begin{equation*} v=\sum_{i\in I}c_iv_i=c_{j_1}v_{j_1}+c_{j_2}v_{j_2}+\cdots +c_{j_n}v_{j_n}\text{.} \end{equation*}
    An easy induction argument on \(n\text{,}\) using the two defining properties of a linear transformation, shows that
    \begin{equation*} T(v)=c_{j_1}T(v_{j_1})+c_{j_2}T(v_{j_2})+\cdots +c_{j_n}T(v_{j_n})\text{.} \end{equation*}
    Since \(c_i=0\) for all \(i\notin J\text{,}\) we have
    \begin{align*} T(v) \amp = c_{j_1}T(v_{j_1})+c_{j_2}T(v_{j_2})+\cdots +c_{j_n}T(v_{j_n})\\ \amp =\sum_{i\in I}c_iT(v_i)\text{,} \end{align*}
    as desired.

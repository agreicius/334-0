Procedure 1.13.3. Computing coordinate vectors.

Let \(B=(v_i)_{i\in }\) be a basis of the vector space \(V\text{.}\) Given \(v\in V\text{,}\) compute the coordinate vector \([v]_B\in F^I\) by following these steps.
  1. Set up the vector equation
    \begin{equation} v=\sum_{i\in i}c_iv_i\tag{1.33} \end{equation}
    in the unknowns \(c_i\in F\text{.}\)
  2. Solve for the unknowns \(c_i\) in some manner. A surefire technique is to reduce the vector equation (1.33) to a linear system and use Gaussian elimination. However, there are some situations when you can simply produce the scalars \(c_i\) by inspection.
  3. Conclude that \([v]_B=(c_i)_{i\in I}\text{.}\)

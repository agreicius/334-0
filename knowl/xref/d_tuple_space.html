Definition 1.2.10. Space of \(F\)-valued functions/tuples.

Let \(F\) be a field. Given any set \(I\text{,}\) recall that \(F^I\) is defined as the set of all functions \(f\colon I\rightarrow F\text{,}\) or equivalently, the set of all \(F\)-valued tuples indexed by \(I\text{:}\) i.e.,
\begin{equation*} F^I=\{f\colon I\rightarrow F\colon f \text{ a function}\}=\{(a_i)_{i\in I}\colon a_i\in F\}\text{.} \end{equation*}
Below we define a vector space structure on \(F^I\text{.}\)
  • Vector operations.
    Given elements \(f=(f(i))_{i\in I}, g=(g(i))_{i\in I}\in F^I\) and scalar \(c\in F\text{,}\) we define
    \begin{align*} f+g \amp = (f(i)+g(i))_{i\in I}\\ cf \amp =(cf(i))_{i\in I}\text{.} \end{align*}
    In other words, \(f+g\) is the function defined as \((f+g)(i)=f(i)+g(i)\) for all \(i\in I\text{,}\) and \(cf\) is the function defined as \((cf)(i)=cf(i)\) for all \(i\in I\text{.}\)
  • Zero vector.
    The zero vector of \(F^I\) is the zero function \(0_I\colon I\rightarrow F\) defined as \(0_I(i)=0\) for all \(i\in I\text{.}\) In other words, using tuple notation, we have \(\boldzero=0_I=(0)_{i\in I}\text{.}\)
  • Vector inverses.
    Given an element \(f=(f(i))_{i\in I}\in F^I\) its vector inverse \(-f\) is defined as
    \begin{equation*} -f=(-f(i))_{i\in I}\text{.} \end{equation*}
    In other words \(-f\) is the function on \(I\) defined as \((-f)(i)=-f(i)\) for all \(i\in I\text{.}\)
We call \(F^I\text{,}\) together with these vector operations, the space of \(F\)-valued tuples indexed by \(I\).

Theorem 1.20.5. Orthogonal complement.

Let \(W\) be a subspace of the inner product space \((V,\angvec{\, ,})\text{.}\)
  1. \(W^\perp\) is a subspace of \(V\text{.}\)
  2. \(W\cap W^\perp=\{\boldzero\}\text{.}\)
  3. \(W\subseteq (W^\perp)^\perp\text{.}\)
  4. \(W\) finite dimensional.
    If \(W\) is finite dimensional, then
    \begin{align} V \amp= W\oplus W^\perp \text{.}\tag{1.73} \end{align}
    Furthermore, we have
    \begin{equation} (W^\perp)^\perp=W\tag{1.74} \end{equation}
    in this setting.

Proof.

  1. We use the 2-step technique.
    Since \(\angvec{\boldzero,v}=0\) for all \(v\in V\text{,}\) it is certainly true that \(\angvec{\boldzero,w}=0\) for all \(w\in W\text{.}\) Thus \(\boldzero\in W^\perp\text{.}\)
  2. Assume \(v_1,v_2\in W^\perp\text{.}\) By definition this means
    \begin{equation*} \angvec{v_1,w}=\angvec{v_2,w}=0 \end{equation*}
    for all \(w\in W\text{.}\) It follows that for any \(c,d\in F\text{,}\) and for all \(w\in W\) we have
    \begin{align*} \angvec{cv_1+dv_2,w} \amp = c\angvec{v_1,w}+d\angvec{v_2,w}\\ \amp =c\cdot 0+d\cdot 0\\ \amp =0\text{,} \end{align*}
    and thus that \(cv_1+dv_2\in W^\perp\text{.}\)
  3. Assume \(v\in W\cap W^\perp\text{.}\) Since \(v\in W\) and \(v\in W^\perp\text{,}\) we must have \(\angvec{v,v}=0\text{.}\) The positivity axiom for inner products then implies that \(v=\boldzero\text{.}\) Thus \(W\cap W^\perp=\{\boldzero\}\text{.}\)
  4. Let \(w\in W\text{.}\) Given any \(v\in W^\perp\text{,}\) we have
    \begin{align*} \angvec{w,v} \amp =\overline{\angvec{v,w}}=\overline{0}=0\text{.} \end{align*}
    Thus \(\angvec{w,v}=0\) for all \(v\in W^\perp\text{,}\) showing that \(w\in (W^\perp)^\perp\text{.}\) This proves \(W\subseteq (W^\perp)^\perp\text{.}\)
  5. Assume \(W\) is finite dimensional. By definition this means \(W\) has a finite basis. The Gram-Schmidt procedure then guarantees that we can produce an orthogonal basis \(B=(w_1,w_2,\dots, w_r)\) of \(W\text{.}\) Furthermore, the proof of the validity of the Gram-Schmidt procedure shows that for any \(v\in V\) the vector
    \begin{equation*} w^\perp=v-\sum_{i=1}^r\frac{\angvec{v,w_i}}{\angvec{w_i,w_i}}w_i \end{equation*}
    is orthogonal to \(w_i\) for all \(1\leq i\leq r\text{.}\) Since \(W=\Span ((w_1,w_2,\dots, w_n))\text{,}\) we conclude from TheoremÂ 1.20.2 that \(w^\perp\in W^perp\text{.}\) Noting that
    \begin{equation*} w=\sum_{i=1}^r\frac{\angvec{v,w_i}}{\angvec{w_i,w_i}}w_i\in W\text{,} \end{equation*}
    we see that for all \(v\in V\) we can write
    \begin{equation*} v=w+w^\perp\text{,} \end{equation*}
    where \(w\in W\) and \(w^\perp\in W^\perp\text{.}\) This proves that \(V=W+W^\perp\text{.}\) Since \(W\cap W^\perp=\{\boldzero\}\) by (2), we conclude that \(V=W\oplus W^\perp\text{.}\)
    Lastly, we show in this case that \(W=(W^\perp)^\perp\text{.}\) Using (3) above, it suffices to show \((W^\perp)^\perp\subseteq W\text{.}\) Take \(v\in (W^\perp)^\perp\text{,}\) and write \(v=w+w^\perp\text{,}\) where \(w\in W\) and \(w^\perp\in W\text{.}\) Since \(v\in (W^\perp)^\perp\text{,}\) we have
    \begin{align*} 0 \amp =\angvec{v,w^\perp}\\ \amp =\angvec{w+w^\perp,w^\perp}\\ \amp =\angvec{w,w^\perp}+\angvec{w^\perp,w^\perp}\\ \amp =0+\angvec{w^\perp,w^\perp}\\ \amp = \angvec{w^\perp,w^\perp}\text{.} \end{align*}
    Since \(\angvec{w^\perp,w^\perp}=0\text{,}\) we conclude that \(w^\perp=\boldzero\text{,}\) and hence that
    \begin{equation*} v=w+w^\perp=w+\boldzero=w\in W\text{,} \end{equation*}
    as desired.

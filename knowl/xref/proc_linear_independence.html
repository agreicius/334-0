Procedure 1.6.11. Linear independence of a finite set.

Let \(S=(v_i)_{i\in I}\) be a tuple of vectors of the vector space \(V\text{.}\) To decide whether \(S\) is linearly independent, proceed as follows.
  1. Write out the general expression for a linear combination of the form
    \begin{equation*} c_{i_1}\boldv_{i_1}+c_{i_2}\boldv_{i_2}+\cdots +c_{i_n}\boldv_{i_n}=\boldzero\text{.} \end{equation*}
  2. If possible, translate this vector equation into a homogeneous system of linear equations in the unknowns \(c_{i_1},c_{i_2},\dots, c_{i_n}\text{,}\) using the definition of equality for your vector space.
  3. Decide, using Gaussian elimination, whether this system has any nonzero (i.e., nontrivial) solutions. If it has no nontrivial solution, \(S\) is linearly independent; if it has a nontrivial solution, \(S\) is linearly dependent.

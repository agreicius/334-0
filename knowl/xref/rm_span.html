Remark 1.6.4.

Let \((v_i)_{i\in I}\) be tuple of vectors in \(V\text{.}\)
  1. The zero vector is always an element of \(\Span(v_i)_{i\in I}\text{.}\) We treat the cases where \(I\) is empty or nonempty separately.
    If \(I=\emptyset\text{,}\) then \((v_i)_{i\in I}=()\) is the empty tuple and \(\Span S=\{\boldzero\}\) by definition.
    If \(I\ne \emptyset\text{,}\) then we can always express \(\boldzero\) as the trivial linear combination \(\boldzero=\sum_{i\in I}0v_i\text{.}\) Thus \(\boldzero\in \Span(v_i)_{i\in I}\text{.}\)
  2. The set \(\{v_i\colon i\in I\}\) is contained in \(\Span(v_i)_{i\in I}\text{.}\) Indeed, given any \(i\in I\text{,}\) we have \(v_i=1v_i\in \Span(v_i)_{i\in I}\text{.}\)
  3. For a tuple \((v)\) of length 1, we have \(\Span(v)=\{cv\colon c\in F\}\text{,}\) the set of all scalar multiples of \(v\text{.}\)

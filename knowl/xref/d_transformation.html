Definition 1.9.1. Linear transformation.

Let \(V\) and \(W\) be \(F\)-vector spaces. A linear transformation from \(V\) to \(W\) is a function
\begin{equation*} T\colon V\rightarrow W \end{equation*}
satisfying the following properties.
  1. \(T(v+v')=T(v)+T(v')\) for all \(v,v'\in V\text{.}\)
  2. \(T(cv)=cT(v)\) for all \(c\in F\) and \(v\in V\text{.}\)
We let \(\mathcal{L}(V,W)\) denote the set of all linear transformations from \(V\) to \(W\text{:}\) i.e.,
\begin{equation*} \mathcal{L}(V,W)=\{T\colon V\rightarrow W\colon T \text{ is a linear transformation}\}\text{.} \end{equation*}

Theorem 1.5.2. Sum of subspaces.

Let \(W_1, W_2, \dots, W_n\) be subspaces of the \(F\)-vector space \(V\text{,}\) and let \(W=\sum_{i=1}^n W_i\) be their sum.
  1. \(W\) is a subspace of \(V\text{.}\)
  2. \(W\) is the smallest subspace of \(V\) satisfying \(W_i\subseteq W \) for all \(1\leq i\leq n\) in the following sense: if \(U\) is a subspace of \(V\) satisfying \(W_i\subseteq U\) for all \(1\leq i\leq n\text{,}\) then \(W\subseteq U\text{.}\)

Proof.

  1. We follow ProcedureÂ 1.4.7.
    Since \(\boldzero\in W_i\) for all \(1\leq i\leq n\text{,}\) and since
    \begin{equation*} \boldzero=\underset{W_1}{\boldzero}+\underset{W_2}{\boldzero}+\cdots +\underset{W_n}{\boldzero}\text{,} \end{equation*}
    we see that \(\boldzero\in W=\sum_{i=1}^n W_i\text{.}\)
    Assume \(w_1, w_2\in W\text{.}\) By definition, we have
    \begin{align*} w_1 \amp =v_1+v_2+\cdots +v_n, v_i\in W_i\\ w_2 \amp =v_1'+v_2'+\cdots +v_n', v_i'\in W_i\text{.} \end{align*}
    But then for any scalars \(c,d\in F\text{,}\) we have
    \begin{align*} cw_1+dw_2 \amp = c(v_1+v_2+\cdots +v_n)+d(v_1'+v_2'+\cdots +v_n')\\ \amp = cv_1+cv_2+\cdots +cv_n+dv_1'+dv_2'+\cdots +dv_n'\\ \amp =(cv_1+dv_1')+(cv_2+dv_2')+\cdots +(cv_n+dv_n')\\ \amp = v_1''+v_2''+\cdots v_n''\text{,} \end{align*}
    where \(v_i''=cv_i+dv_i'\) for all \(1\leq i\leq n\text{.}\) Since each \(W_i\) is a subspace, we have
    \begin{equation*} v_i''=cv_i+dv_i'\in W_i \end{equation*}
    for all \(1\leq i\leq n\text{.}\) We conclude that \(cw_1+dw_2\in W\text{,}\) as desired.
  2. For any \(v\in W_i\text{,}\) we have
    \begin{equation*} v=v_1+v_2+\cdots +v_n\text{,} \end{equation*}
    where
    \begin{equation*} v_j=\begin{cases} \boldzero \amp \text{if } j\ne i \\ v \amp \text{if } j=i \end{cases}\text{.} \end{equation*}
    Since \(v_j\in W_j\) for all \(1\leq j\leq n\text{,}\) we see that \(v\in W\text{.}\) Since \(v\) was an arbitrary element of \(W_i\text{,}\) we conclude that \(W_i\subseteq W\) for all \(1\leq i\leq n\text{.}\)
    Now assume \(U\) is a subspace of \(V\) that contains \(W_i\) for all \(1\leq i\leq n\text{.}\) We will show that \(W=\sum_{i=1}^n W_i\subseteq U\text{.}\) Indeed, given any \(v\in W\text{,}\) we have
    \begin{equation*} v=v_1+v_2+\cdots +v_n \end{equation*}
    for some vectors \(v_1,v_2,\dots, v_n\text{,}\) where \(v_i\in W_i\text{.}\) Since \(W_i\subseteq U\) for all \(i\text{,}\) we have \(v_i\in U\) for all \(i\text{;}\) since \(U\) is closed under addition, we have \(v=v_1+v_2+\cdots +v_n\in U\text{,}\) as desired.

Definition 1.22.2. Self-adjoint, skew-Hermitian, unitary.

Let \((V,\angvec{\, ,})\) be an inner product space. Below we define several special types of normal operators. Let \(T\in \mathcal{L}(V)\) and suppose \(T\) has an adjoint \(T^*\text{.}\)
  • Self-adjoint.
    \(T\) is self-adjoint (or symmetric when \(F=\R\)) if \(T^*=T\text{.}\)
  • Skew-Hermitian.
    \(T\in \mathcal{L}(V)\) is skew-Hermitian (or skew-symmetric when \(F=\R\)) if \(T^*=-T\text{.}\)
  • Unitary.
    \(T\in \mathcal{L}(V)\) is unitary (or orthogonal when \(F=\R\)) if \(T\) is invertible and \(T^{-1}=T^*\text{.}\)

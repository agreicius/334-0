    Section 1.4 Subspaces

    In this section we will study subspaces, which are special subsets of vector spaces that respect the defining structure of a vector spaces: namely, the two vector operations. Definition 1.4.1 makes precise what we mean here by ‘respect’.
    Subspaces arise naturally in any setting where vector spaces are at play, and are closely connected to solutions to linear systems. As we will see in Theorem 1.4.5, subspaces of vector spaces are vector spaces in their own right, furnishing us with yet more interesting examples of vector spaces.

    Subsection Definition and examples

    Definition 1.4.1. Subspace.

    Let \(V\) be a vector space. A subset \(W\subseteq V\) is a subspace of \(V\) if the following conditions hold.
    1. \(W\) contains the zero vector.
      \(\boldzero\in W\text{.}\)
    2. \(W\) is closed under addition.
      For all \(v_1, v_2\in V\text{,}\) if \(v_1, v_2\in W\text{,}\) then \(v_1+ v_2\in W\text{.}\) Using logical shorthand:
      \begin{equation*} v_1, v_2\in W\implies v_1+ v_2\in W\text{.} \end{equation*}
    3. \(W\) is closed under scalar multiplication.
      For all \(c\in F\) and \(v\in V\text{,}\) if \(v\in W\text{,}\) then \(c v\in W\text{.}\) Using logical shorthand:
      \begin{equation*} v\in W\Rightarrow c v\in W\text{.} \end{equation*}

    Example 1.4.2. Trivial subspaces.

    Let \(V\) be an \(F\)-vector space. The following are subspaces of \(V\text{:}\)
    1. \(\displaystyle W_1=\{\boldzero\}\)
    2. \(\displaystyle W_2=V\)
    In other words, the set containing just the zero vector is a subspace of \(V\text{,}\) as is \(V\) itself. These are sometimes called the trivial subspaces of \(V\text{.}\) Observe that every subspace \(W\) lies between these two: i.e., we have \(\{\boldzero\}\subseteq W\subseteq V\text{.}\)
    Solution.
    We prove that \(W_1=\{\boldzero\}\) is a subspace of \(V\text{,}\) leaving the proof for \(W_2=V\) to the reader.
    1. By definition, \(\boldzero\in W_1\text{.}\)
    2. Since \(\boldzero\) is the only element of \(W_1\) and since \(\boldzero+\boldzero=\boldzero\in W_1\text{,}\) we see that \(W_1\) is closed under addition.
    3. Since \(\boldzero\) is the only element of \(W_1\text{,}\) we need only show that \(c\boldzero\in W_1\) for all scalars \(c\in F\text{.}\) But we have \(c\boldzero=\boldzero\) by Theorem 1.3.6. Thus \(c\boldzero\in W_1\text{,}\) as desired.
    This proves \(W_1=\{\boldzero\}\) is a subspace of \(V\text{.}\)

    Example 1.4.3.

    Let \(V=\R^2\) and let
    \begin{equation*} W=\{(t,t)\in\R^2 \colon t\in\R\}\text{.} \end{equation*}
    Prove that \(W\) is a subspace.
    Solution.
    We must show properties (i)-(iii) hold for \(W\text{.}\)
    1. The zero element of \(V\) is \(\boldzero=(0,0)\text{,}\) which is certainly of the form \((t,t)\text{.}\) Thus \(\boldzero\in W\text{.}\)
    2. We must prove the implication \(v_1, v_2\in W\Rightarrow v_1+ v_2\in W\text{.}\)
      \begin{align*} v_1, v_2\in W\amp \Rightarrow\amp v_1=(t,t), v_2=(s,s) \text{ for some \(t,s\in\R\) }\\ \amp \Rightarrow\amp v_1+ v_2=(t+s,t+s)\\ \amp \Rightarrow\amp v_1+ v_2\in W\text{.} \end{align*}
    3. We must prove the implication \(v\in W\Rightarrow c v\in W\text{,}\) for any \(c\in \R\text{.}\) We have
      \begin{align*} v\in W\amp \Rightarrow\amp v=(t,t)\\ \amp \Rightarrow\amp c v=(ct,ct)\\ \amp \Rightarrow\amp c v\in W \end{align*}

    Example 1.4.4.

    Let \(V=\R^n\) and let
    \begin{equation*} W=\{(x,y)\in \R^2\colon x, y\geq 0\}\text{.} \end{equation*}
    Is \(W\) a vector space? Decide which of the of properties (i)-(iii) in Definition 1.4.1 (if any) are satisfied by \(W\text{.}\)
    Solution.
    1. Clearly \(\boldzero=(0,0)\in W\text{.}\)
    2. Suppose \(v_1=(x_1,y_1), v_2=(x_2,y_2)\in W\text{.}\) Then \(x_1, x_2, y_1, y_2\geq 0\text{,}\) in which case \(x_1+x_2, y_1+y_2\geq 0\text{,}\) and hence \(v_1+ v_2\in W\text{.}\) Thus \(W\) is closed under addition.
    3. The set \(W\) is not closed under scalar multiplication. Indeed, let \(v=(1,1)\in W\text{.}\) Then \((-2) v=(-2,-2)\notin W\text{.}\)
    As we now endeavor to show, if \(W\) is a subspace of a vector space \(V\text{,}\) then it inherits a vector space structure from \(V\) by simply restricting the vector operations defined on \(V\) to the subset \(W\text{.}\)

    Theorem 1.4.5. Subspaces are vector spaces.

    Let \(W\) be a subspace of the vector space \(V\text{.}\)
    1. The vector operations of \(V\) restrict to operations on \(W\) that satisfy the vector space axioms.
    2. The zero vector of \(W\text{,}\) considered as a vector space, is the zero vector of \(V\text{.}\)
    3. Given an element \(\boldw\in W\text{,}\) its vector inverse with respect to the vector space structure of \(W\) is equal to its vector inverse with respect to the vector space structure of \(V\text{.}\)

    Proof.

    It is important to understand how Axioms i–iii of Definition 1.4.1 come into play here. Without them we would not be able to say that restricting the vector operations of \(V\) to elements of \(W\) actually gives rise to well-defined operations on \(W\text{.}\) To be well-defined the operations must output elements that lie not just in \(V\text{,}\) but in \(W\) itself. This is precisely what being closed under addition and scalar multiplication guarantees. In more detail, since \(\boldw_1+\boldw_2\in W\) for all \(w_1,w_2\in W\) (Axiom (ii)), the vector addition on \(V\) gives rise by restriction to a well-defined operation on \(W\text{;}\) similarly, since \(c\boldw\in W\) for all \(c\in \R\) and \(\boldw\in W\text{,}\) the scalar multiplication operation on \(V\) gives rise by restriction to a well-defined scalar multiplication on \(W\text{.}\)
    Once we know restriction gives rise to well-defined operations on \(W\text{,}\) verifying the \(arithmetic axioms\) of Definition 1.2.2 amounts simply to observing that if a condition is true for all \(v\) in \(V\text{,}\) it is certainly true for all \(v\) in the subset \(W\text{.}\)
    The “existential axioms” (iii) and (iv) of Definition 1.2.2, however, require special consideration. By definition, a subspace \(W\) contains the zero vector of \(V\text{,}\) and clearly this still acts as the zero vector when we restrict the vector operations to \(W\text{.}\) What about vector inverses? We know that for any \(v\in W\) there is a vector inverse \(- v\) lying somewhere in \(V\text{.}\) We must show that in fact \(- v\) lies in \(W\text{:}\) i.e. we need to show that the operation of taking the vector inverse is well-defined on \(W\text{.}\) We prove this as follows:
    \begin{align*} v\in W \amp\implies (-1) v\in W \amp (\knowl{./knowl/xref/d_subspace.html}{\text{Definition 1.4.1}}, \text{(iii) } )\\ \amp\implies - v\in W \amp (\knowl{./knowl/xref/th_vs_props.html}{\text{Theorem 1.3.6}}, (iii)) \text{.} \end{align*}
    Before moving on to more examples of subspaces, we provide a procedure for determining whether something is a subspace. Note that the procedure merges the verification of Axioms (ii)-(iii) of Definition 1.4.1 into a single step, using the notion of a linear combination. Now is a good time to make this notion official.

    Definition 1.4.6. Linear combination.

    Let \(V\) be an \(F\)-vector space, and let \(v_1,v_2,\dots, v_n\) be a collection of vectors of \(V\text{.}\) A linear combination of the vectors \(v_i\) is a vector expression of the form
    \begin{equation} c_1v_1+c_2v_2+\cdots +c_nv_n\text{,}\tag{1.5} \end{equation}
    where \(c_i\in F\) for all \(1\leq i\leq n\text{.}\) The scalars \(c_i\) are called the coefficients of the linear combination. The linear combination (1.5) is trivial if \(c_i=0\) for all \(1\leq i\leq n\text{,}\) and nontrivial if \(c_i\ne 0\) for some \(1\leq i\leq n\text{.}\) A vector \(v\) is a linear combination of the \(v_i\) if we have
    \begin{equation*} v=c_1v_1+c_2v_2+\cdots +c_nv_n \end{equation*}
    for some choice of scalars \(c_i\in F\text{.}\)

    Procedure 1.4.7. Two-step proof for subspaces.

    We can merge conditions (ii)-(iii) of Definition 1.4.1 into a single statement about linear combinations, deriving the following two-step method for proving that a set \(W\) is a subspace of a vector space \(V\text{.}\)
    1. Show \(\boldzero\in W\)
    2. Show that
      \begin{equation*} v_1, v_2\in W\implies c v_1+d v_2\in W\text{,} \end{equation*}
      for all \(c,d\in F\text{.}\)
    Theorem 1.4.9 below serves both as an illustration of Procedure 1.4.7 and a rich source of examples of subspaces. It tells us that the solutions to a matrix equation \(A\boldx=\boldzero\) constitutes a subspace of \(\F^n\text{.}\) This set is called the null space of the matrix. (We will postpone an official definition of null space until we introduce linear transformations.)
    The statement and proof of Theorem 1.4.9 will make use of matrix multiplication and elementary properties that matrix multiplication satisfies. The usual definition of matrix multiplication (for real matrices) generalizes directly to an arbitrary field, and as such we will not include an official definition. Furthermore, all the usual matrix multiplication properties you know and love over the reals carry over to the more general setting of matrices over a general field \(F\text{.}\) We will take this for granted and not include an official statement of those properties.
    Lastly, we include another fiat that facilitates dealing with so-called column vectors \(\boldx\in F^{n,1}\text{.}\)

    Fiat 1.4.8. Tuples shall be column vectors, and vice versa.

    We hereby declare that all \(n\)-tuples \((a_1,a_2,\dots, a_n)\in \F^n\) shall be equated with the corresponding column vector in \(F^{n,1}\text{:}\) i.e.,
    \begin{equation*} (a_1,a_2,\dots, a_n) = \begin{bmatrix} a_1\\ a_2\\ \vdots \\ a_n \end{bmatrix} \end{equation*}
    in \(\F^{n,1}\text{.}\)

    Theorem 1.4.9. Matrix null space.

    Let \(A\in F^{m,n}\text{.}\) The set
    \begin{equation*} W=\{\boldx\in F^n \colon \underset{m\times n}{A}\, \underset{n\times 1}{\boldx}=\underset{m\times 1}{\boldzero}\} \end{equation*}
    of solutions to the matrix equation \(A\boldx=\boldzero\) is a subspace of \(\F^n\text{.}\)

    Proof.

    We follow Procedure 1.4.7.
    1. Note that according to Fiat 1.4.8, we have \(\boldzero=(0,0,\dots, 0)=\underset{n\times 1}{\boldzero}\text{,}\) and thus
      \begin{equation*} A\boldzero=A\underset{n\times 1}{\boldzero}=\underset{m\times 1}{\boldzero}\text{.} \end{equation*}
      This proves that \(\boldzero\in W\text{.}\)
    2. We want to show that if \(\boldx_1,\boldx_2\in W\text{,}\) then so is \(c\boldx_1+d\boldx_2\) for any \(c,d\in F\text{,}\)
      Assume \(\boldx_1,\boldx_2\in W\text{.}\) By definition this means \(A\boldx_1=A\boldx_2=\underset{m\times 1}{\boldzero}\text{.}\) But then we have
      \begin{align*} A(c\boldx_1+d\boldx_2) \amp = cA\boldx_1+dA\boldx_2 \amp (\text{matrix props.})\\ \amp = c\underset{m\times 1}{\boldzero}+d\underset{m\times 1}{\boldzero}\\ \amp = \underset{m\times 1}{\boldzero}\text{,} \end{align*}
      showing that \(c\boldx_1+d\boldx_2\in W\text{,}\) as desired.

    Example 1.4.10. Hyperplanes of \(\R^n\).

    A hyperplane of \(\R^n\) is a set \(H\subseteq \R^n\) defined as the set of solutions to a linear equation of the form
    \begin{equation} a_1x_1+a_2x_2+\cdots +a_nx_n=b\text{,}\tag{1.6} \end{equation}
    where \(a_i,b\in \R\) and \(a_i\ne 0\) for some \(i\text{.}\) (In other words \(H\) is the set of solutions to a nontrivial linear equation with coefficients in \(\R\text{.}\))
    1. Prove that if \(b\ne 0\text{,}\) then \(H\) is not a subspace of \(\R^n\text{.}\)
    2. Prove that if \(b=0\text{,}\) then \(H\) is a subspace of \(\R^n\text{.}\)
    Solution.
    1. If \(b\ne 0\text{,}\) then clearly \(\boldzero=(0,0,\dots, 0)\notin H\text{,}\) since
      \begin{equation*} a_10+a_20+\cdots +a_n0=0\ne b\text{.} \end{equation*}
      It follows that \(H\) is not a subspace in this case, since it fails Axiom (i).
    2. Assume \(b=0\text{.}\) Let
      \begin{equation*} A=\begin{bmatrix} a_1 \amp a_2 \amp \cdots \amp a_n \end{bmatrix}\text{,} \end{equation*}
      the \(1\times n\) matrix consisting of the coefficients of the linear equation. We have
      \begin{equation*} H=\{\boldx\in \R^n\colon A\boldx=[0]\}\text{.} \end{equation*}
      Thus \(H\) is a subspace by Theorem 1.4.9.

    Subsection Important function subspaces

    Definition 1.4.11. Polynomials.

    Assume \(F\in \{\R,\C\}\text{,}\) and let \(I\) be an infinite subset of \(F\text{.}\) A polynomial function on \(I\) is a function \(f\colon I\rightarrow F\) that can be written in the form
    \begin{equation} f(x)=a_nx^n+a_{n-1}x+\cdots +a_1x+a_0\text{,}\tag{1.7} \end{equation}
    for some nonnegative integer \(n\) and scalars \(a_k\in F\text{.}\) Given \(f\) as in (1.7), \(a_k\) is called the \(k\)-th coefficient of \(f\text{;}\) furthermore, if \(a_n\ne 0\text{,}\) we call \(a_nx^n\) the leading term of \(f\text{,}\) and we if \(a_n\ne 0\text{,}\) and we define the degree of \(f\text{,}\) denoted \(\deg f\) to be \(n\text{.}\)
    Given a nonnegative integer \(n\text{,}\) we define \(P_n(I)\) to be the set of all polynomial functions on \(I\) of degree at most \(n\text{:}\) i.e.,
    \begin{equation*} P_n(I)=\{f\in F^I\colon f(x)=a_nx^n+a_{n-1}x^{n-1}+\cdots +a_1x+a_0 \text{ for some } a_i\in F\}\text{.} \end{equation*}
    Lastly we define \(P(I)\) to be the set of all polynomial functions on \(I\text{.}\) Since any polynomial on \(I\) is of the form \(f(x)=\anpoly\) for some \(n\in \Z_{\geq 0}\text{,}\) we have
    \begin{equation*} P(I)=\bigcup_{n=0}^\infty P_n(I)\text{.} \end{equation*}

    Theorem 1.4.12. Polynomial spaces.

    Let \(F\in \{\R, \C\}\text{,}\) and let \(I\) be an infinite subset of \(F\text{.}\)
    1. \(P(I)\) is a subspace of \(F^I\text{.}\)
    2. \(P_n(I)\) is a subspace of \(F^I\) for all \(n\in \Z_{\geq 0}\text{.}\)

    Proof.

    We prove that \(P_n(I)\) is a subspace, using the 2-step technique. The proof for \(P(I)\) is similar.
    1. Recall that the zero vector of \(F^I\) is the the zero function \(O_I\colon I\rightarrow F\text{,}\) defined as \(0_I(x)=0\) for all \(x\in I\text{.}\) Writing
      \begin{equation*} 0_I(x)=0x^n+0x^{n-1}+\cdots +0x+0\text{,} \end{equation*}
      we see that \(\boldzero=0_I\in P_n(I)\text{.}\) (More generally, any constant function \(f(x)=c\) is an element of \(P_n(I)\) for all \(n\geq 0\text{.}\))
    2. Assume \(f,g\in P_n(I)\text{.}\) By definition this means that there are constants \(a_i, b_i\in F\text{,}\) \(1\leq i\leq n\text{,}\) such that
      \begin{align*} f(x) \amp = \anpoly\\ g(x) \amp = b_nx^n+b_{n-1}x^{n-1}+\cdots+b_1x+b_0 \end{align*}
      for all \(x\in I\text{.}\) It follows that
      \begin{align*} (cf+dg)(x) \amp = (a_n+b_n)x^n+(a_{n-1}+b_{n-1})x^{n-1}+\cdots+(a_1+b_1)x+(a_0+b_0) \end{align*}
      for all \(x\in I\text{,}\) showing that \(cf+dg\in P_n(I)\text{,}\) as desired.
    Although not necessary for the proof of Theorem 1.4.12, the result below will be useful to us for future discussions. In short, it says that the coefficients of a polynomial uniquely determine it (as long as the domain \(I\) is infinite).

    Theorem 1.4.13. Polynomial equality.

    Let \(F\in \{\R, \C\}\) and let \(I\) be an infinite subset of \(F\text{.}\)
    1. Given \(f(x)=\anpoly\in P_n(I)\text{,}\) we have
      \begin{equation*} f=\boldzero\iff a_k=0 \text{ for all } 1\leq k\leq n\text{.} \end{equation*}
    2. Given \(f(x)=\anpoly\in P_n(I)\) and \(g(x)=\bmpoly\in P_m(I)\) with \(a_n\ne 0\) and \(b_m\ne 0\text{,}\) we have
      \begin{equation*} f=g \iff n=m \text{ and } a_k=b_k \text{ for all } 1\leq k\leq n\text{.} \end{equation*}

    Proof.

    This is a standard result about polynomials, and is a consequence of the fact that a polynomial of degree \(n\) has at most \(n\) distinct roots. We do not provide a proof here.

    Definition 1.4.14. Function subspaces over \(\R\).

    Let \(I\) be an interval of \(\R\) that contains at least two elements.
    1. We denote by \(C(I)\) the set of all continuous functions on \(I\text{:}\) i.e.,
      \begin{equation*} C(I)=\{f\in \R^I\colon f \text{ is continuous on }I\}\text{.} \end{equation*}
    2. Fix \(n\geq 1\text{.}\) A function \(f\colon I\rightarrow \R\) is \(C^{n}\) on \(I\) if \(f\) is \(n\)-times differentiable on \(I\) and its \(n\)-th derivative \(f^{(n)}(x)\) is continuous. The set of all \(C^{n}\) functions on \(I\) is denoted \(C^{n}(I)\text{.}\)
    3. A function \(f\colon I\rightarrow \R\) is \(C^{\infty}\) on I if \(f\) is infinitely differentiable on \(I\text{.}\) The set of all \(C^{\infty}\) functions on \(I\) is denoted \(C^{\infty}(I)\text{.}\)

    Theorem 1.4.15. Function subspaces..

    Let \(I\subseteq \R\) be a an interval containing at least two elements. The sets \(C(I)\text{,}\) \(C^{n}(I)\text{,}\) \(C^{\infty}(I)\text{,}\) \(P(I)\text{,}\) \(P_{n}(I)\) are all subspaces of \(\R^I\text{,}\) and we have the following chain of subspaces:
    \begin{equation*} P_{n}(I)\subseteq P(I)\subseteq C^{\infty}(I)\subseteq C^{n}(I)\subseteq C(I)\subseteq \R^{I}\text{.} \end{equation*}

    Proof.

    This is an instance of “proof by calculus”. That each of these sets is closed under addition and scalar multiplication follows from elementary results in single-variable calculus: e.g., that sums and scalar multiples of continuous functions are continuous, that sums and scalar multiples of differentiable functions are differentiable, etc..

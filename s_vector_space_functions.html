    Section 1.3 Vector spaces (cont.)

    Subsection Function spaces

    As promised, we now prove that sets \(F^n\text{,}\) \(F^\infty\text{,}\) \(F^{m,n}\text{,}\) together with their indicated operations, do indeed satisfy the vector space axioms. We will do this in a somewhat indirect manner by first constructing a more general vector space \(F^I\text{,}\) and then showing that \(F^n\text{,}\) \(F^\infty\text{,}\) and \(F^{m,n}\) are just specific instances of this more general construction, corresponding to particular choices of the set \(I\text{.}\)
    The objects of the vector space \(F^I\) defined in Definition 1.3.1 can be thought of either as functions \(f\colon I\rightarrow F\) from the set \(I\) to the field \(F\text{,}\) or equivalently, as \(F\)-valued tuples \((a_i)_{i\in I}\) indexed by \(I\text{.}\) Hence the “functions/tuples” in its name. Depending on the context, one of these two equivalent conceptions (function or tuple) may be more convenient than the other. For example, to make the connection between \(F^I\text{,}\) \(F^n\text{,}\) \(F^\infty\text{,}\) and \(F^{m,n}\text{,}\) it is most convenient to think in terms of tuples. See Definition 0.1.24 for the general definition of tuples and Remark 0.1.25 for an explication of the equivalence between functions and tuples.

    Definition 1.3.1. Space of \(F\)-valued functions/tuples.

    Let \(F\) be a field. Given any set \(I\text{,}\) recall that \(F^I\) is defined as the set of all functions \(f\colon I\rightarrow F\text{:}\) i.e.
    \begin{equation*} F^I=\{f\colon I\rightarrow F\}\text{.} \end{equation*}
    Equivalently, thinking of functions as tuples, we can describe \(F^I\) as the set of all \(F\)-valued tuples indexed by \(I\text{:}\) i.e.,
    \begin{equation*} F^I=\{(a_i)_{i\in I}\colon a_i\in F\}\text{.} \end{equation*}
    Below we define a vector space structure on \(F^I\text{.}\)
    • Vector operations.
      Given elements \(f=(f(i))_{i\in I}, g=(g(i))_{i\in I}\in F^I\) and scalar \(c\in F\text{,}\) we define
      \begin{align*} f+g \amp = (f(i)+g(i))_{i\in I}\\ cf \amp =(cf(i))_{i\in I}\text{.} \end{align*}
      In other words, \(f+g\) is the function defined as \((f+g)(i)=f(i)+g(i)\) for all \(i\in I\text{,}\) and \(cf\) is the function defined as \((cf)(i)=cf(i)\) for all \(i\in I\text{.}\)
    • Zero vector.
      The zero vector of \(F^I\) is the zero function \(0_I\colon I\rightarrow F\) defined as \(0_I(i)=0\) for all \(i\in I\text{.}\) In other words, using tuple notation, we have \(\boldzero=0_I=(0)_{i\in I}\text{.}\)
    • Vector inverses.
      Given an element \(f=(f(i))_{i\in I}\in F^I\) its vector inverse \(-f\) is defined as
      \begin{equation*} -f=(-f(i))_{i\in I}\text{.} \end{equation*}
      In other words \(-f\) is the function on \(I\) defined as \((-f)(i)=-f(i)\) for all \(i\in I\text{.}\)
    We call \(F^I\text{,}\) together with these vector operations, the space of functions from \(I\) to \(F\), or, thinking in terms of tuples, the space of \(F\)-valued tuples indexed by \(I\).

    Theorem 1.3.2. Function/tuple spaces are vector spaces.

    Each of the sets \(F^n\text{,}\) \(F^\infty\text{,}\) \(F^{m,n}\text{,}\) \(F^I\text{,}\) together with the accompanying vector operations defined above, is a vector space.

    Proof.

    First we observe, thinking in terms of tuples, that each of the sets \(F^{n}\text{,}\) \(F^\infty\text{,}\) \(F^{m,n}\text{,}\) along with their accompanying vector space operations, is a specific example of \(F^I\) together with its vector operations. This is seen by choosing an appropriate index set \(I\) in each case. In more detail, we have
    \begin{align*} F^n\amp= F^I, \text{ where } I=\{1,2,\dots, n\} \\ F^\infty\amp= F^I, \text{ where } I=\Z_+=\{1,2,3,\dots\} \\ F^{m,n}\amp= F^I, \text{ where } I=\{(i,j)\colon 1\leq i\leq m, 1\leq j\leq n\} \text{.} \end{align*}
    Thus it suffices to show that \(F^I\) is a vector space with respect to the given vector operations. To do this, we follow Procedure 1.2.5. Since Definition 1.3.1 has already taken care of steps (1)-(3) for us, it remains to show that the vector space axioms are satisfied. As you will see, the tuple notation for functions makes these proofs quite straightforward consequences of the field axioms satisfied by \(F\text{.}\)
    Consider the arithmetic axioms first. We will verify that Axioms (i), (ii), and (v) are satisfied. Given \(f=(f(i))_{i\in I},g=(g(i))_{i\in I},h=(h(i))_{i\in I}\in F^I\) and \(c\in F\text{,}\) we have
    \begin{align*} f+g \amp = (f(i)+g(i))_{i\in I} \amp (\text{def. vec. add.}) \\ \amp = (g(i)+f(i))_{i\in I} \amp (\text{field prop.})\\ \amp = g+f \amp (\text{def. vec. add.})\\ \amp \\ f+(g+h) \amp = (f(i)+(g+h)(i))_{i\in I} \amp (\text{def. vec. add.})\\ \amp = (f(i)+(g(i)+h(i)))_{i\in I} \amp (\text{def. vec. add.})\\ \amp = ((f(i)+g(i))+h(i))_{i\in I} \amp (\text{field prop.})\\ \amp = ((f+g)(i)+h(i))_{i\in I} \amp (\text{def. vec. add.})\\ \amp = (f+g)+h \amp (\text{def. vec. add.})\\ \amp \\ c(f+g) \amp =(c(f+g)(i))_{i\in I} \amp (\text{def. scal. mult})\\ \amp = (c(f(i)+g(i)))_{i\in I} \amp (\text{def. vec. add.})\\ \amp = (cf(i)+cg(i))_{i\in I} \amp (\text{field prop.})\\ \amp =((cf)(i)+(cg)(i))_{i\in I} \amp (\text{def. scal. mult})\\ \amp = cf+cg \amp (\text{def. vec. add.})\text{.} \end{align*}
    Lastly we will show that \(0_I\) does indeed satisfy the additive identity axiom (Axiom (iii)), leaving the verification of Axiom (iv) to the reader. Given any element \(f=(f(i))_{i\in I}\in F^I\text{,}\) we have
    \begin{align*} 0_I+f \amp =(0_I(i)+f(i))_{i\in I} \amp (\text{def. vec. add.})\\ \amp = (0+f(i))_{i\in I}\amp (\text{def. of } 0_I)\\ \amp = (f(i))_{i\in I} \amp (\text{field prop.})\\ \amp = f\text{,} \end{align*}
    as desired.

    Example 1.3.3. Real-valued functions.

    Let \(I=[a,b]\subseteq \R \) be an interval in the real line. Using the function interpretation of \(F^I\text{,}\) \(\R^{[a,b]}\) is the set of all functions of the form \(f\colon [a,b]\rightarrow \R\text{.}\) The vector operations defined in Definition 1.3.1 in this setting correspond to the function-arithmetic operations you met in precalculus and calculus: i.e., given functions \(f,g\in \R^{[a,b]}\text{,}\) their vector sum \(f+g\) is the function defined as \((f+g)(x)=f(x)+g(x)\) for all \(x\in [a,b]\text{.}\) Similarly, given a scalar \(c\in \R\) and function \(f\in \R^{[a,b]}\text{,}\) the vector scalar multiple \(cf\) is the function defined as \((cf)(x)=cf(x)\) for all \(x\in \R\text{.}\)
    In this context it is possible to visualize our vector operations in terms of the graphs of the functions on the interval \([a,b]\text{.}\)

    Subsection Vector space properties

    We end this section with a few elementary vector space properties. Since these statements involve a general vector space (as opposed to a specific type of vector space), we will make no assumptions about \(V\) and the base field \(F\text{,}\) other than that \(V\) satisfies the vector space axioms and \(F\) satisfies the field axioms.

    Theorem 1.3.5. Uniqueness of zero vector and inverses.

    Let \(V\) be an \(F\)-vector space.
    1. The zero vector in \(V\) is unique: i.e., there is one and only one element of \(V\) that satisfies (1.3).
    2. Inverse vectors are unique: i.e., for all \(v\in V\) there is one and only one vector \(-v\) satisfying (1.4).

    Proof.

    1. If \(w\in V\) satisfies \(w+v=v\) for all \(v\in V\text{,}\) then in particular, taking \(v=\boldzero\text{,}\) we see that \(w+\boldzero=\boldzero\text{.}\) But since \(w+\boldzero=w\) (using the defining property of \(\boldzero)\text{,}\) we see that
      \begin{align*} w+\boldzero=\boldzero \amp \implies w=\boldzero\text{.} \end{align*}
      We conclude that \(w=\boldzero\text{,}\) as desired.
    2. Let \(v\in V\text{.}\) Given \(w\in V\) we have
      \begin{align*} w+v =\boldzero \amp \implies w+v=-v+v \amp (-v+v=\boldzero) \\ \amp \implies w+v+(-v)=-v+v+(-v)\\ \amp \implies w+\boldzero=-v+\boldzero \amp (v+(-v)=\boldzero)\\ \amp \implies w=-v \amp (\text{prop. of } \boldzero)\text{.} \end{align*}

    Theorem 1.3.6. Vector space properties.

    Let \(V\) be an \(F\)-vector space.
    1. For all \(v\in V\text{,}\) we have \(0\boldv=\boldzero\text{.}\)
    2. For all \(c\in F\text{,}\) we have \(c\boldzero=\boldzero\text{.}\)
    3. For all \(v\in V\text{,}\) we have \((-1)\boldv=-\boldv\text{.}\)
    4. For all \(v\in V\) and \(c\in F\text{,}\) if \(cv=\boldzero\text{,}\) then \(c=0\) or \(v=\boldzero\text{.}\)

    Proof.

    We provide proofs of (1) and (3), leaving (2) and (4) to the reader.
    Our proof of (1) is almost identical to a similar statement in Theorem 1.1.23. For any \(v\in V\text{,}\) we have
    \begin{align*} 0v \amp = (0+0)v \amp (\text{field prop. of } F)\\ \amp 0v+0v \amp (\text{dist. prop. of } V)\text{.} \end{align*}
    But then we have
    \begin{align*} 0v\amp = 0v+0v \\ -(0v)+0v\amp=-(0v)+0v+0v \amp (\text{add } -(0v)\text{ to both sides}) \\ \boldzero\amp=\boldzero+0v \amp (\text{vec. inv. prop.})\\ \boldzero\amp=0v \amp (\text{prop. of } \boldzero)\text{,} \end{align*}
    as desired.
    Let \(v\in V\text{.}\) To prove (2), we will show that \((-1)v\) satisfies the defining property (1.4) of \(-v\text{;}\) since each vector has a unique vector inverse (Theorem 1.3.5), it will then follow that \((-1)v=-v\text{.}\) To show \((-1)v\) satisfies (1.4), we observe that
    \begin{align*} (-1)v+v \amp =(-1)v+1v \amp (\text{Axiom (viii)})\\ \amp = (-1+1)v \amp (\text{Axiom (vii)})\\ \amp =0v\\ \amp = \boldzero (\text{by (1)})\text{.} \end{align*}
    Thus \((-1)v=-v\text{.}\)

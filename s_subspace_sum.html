    Section 1.5 Subspace sums

    Definition 1.5.1. Vector operations on sets.

    Let \(V\) be an \(F\)-vector space. Given subsets \(X_1,X_2,\dots, X_n\) of \(V\text{,}\) we define their sum, denoted \(X_1+X_2+\cdots +X_n\) or \(\sum_{i=1}^nX_i\text{,}\) as
    \begin{equation*} X_1+X_2+\cdots +X_n=\{v_1+v_2+\cdots +v_n\colon v_i\in X_i\}\text{.} \end{equation*}
    Similarly, given a set \(X\subseteq V\) and scalar \(c\in F\text{,}\) we define the scalar multiple \(cX\) as
    \begin{equation*} cX=\{cv\colon v\in X\}\text{.} \end{equation*}

    Theorem 1.5.2. Sum of subspaces.

    Let \(W_1, W_2, \dots, W_n\) be subspaces of the \(F\)-vector space \(V\text{,}\) and let \(W=\sum_{i=1}^n W_i\) be their sum.
    1. \(W\) is a subspace of \(V\text{.}\)
    2. \(W\) is the smallest subspace of \(V\) satisfying \(W_i\subseteq W \) for all \(1\leq i\leq n\) in the following sense: if \(U\) is a subspace of \(V\) satisfying \(W_i\subseteq U\) for all \(1\leq i\leq n\text{,}\) then \(W\subseteq U\text{.}\)

    Proof.

    1. We follow Procedure 1.4.7.
      Since \(\boldzero\in W_i\) for all \(1\leq i\leq n\text{,}\) and since
      \begin{equation*} \boldzero=\underset{W_1}{\boldzero}+\underset{W_2}{\boldzero}+\cdots +\underset{W_n}{\boldzero}\text{,} \end{equation*}
      we see that \(\boldzero\in W=\sum_{i=1}^n W_i\text{.}\)
      Assume \(w_1, w_2\in W\text{.}\) By definition, we have
      \begin{align*} w_1 \amp =v_1+v_2+\cdots +v_n, v_i\in W_i\\ w_2 \amp =v_1'+v_2'+\cdots +v_n', v_i'\in W_i\text{.} \end{align*}
      But then for any scalars \(c,d\in F\text{,}\) we have
      \begin{align*} cw_1+dw_2 \amp = c(v_1+v_2+\cdots +v_n)+d(v_1'+v_2'+\cdots +v_n')\\ \amp = cv_1+cv_2+\cdots +cv_n+dv_1'+dv_2'+\cdots +dv_n'\\ \amp =(cv_1+dv_1')+(cv_2+dv_2')+\cdots +(cv_n+dv_n')\\ \amp = v_1''+v_2''+\cdots v_n''\text{,} \end{align*}
      where \(v_i''=cv_i+dv_i'\) for all \(1\leq i\leq n\text{.}\) Since each \(W_i\) is a subspace, we have
      \begin{equation*} v_i''=cv_i+dv_i'\in W_i \end{equation*}
      for all \(1\leq i\leq n\text{.}\) We conclude that \(cw_1+dw_2\in W\text{,}\) as desired.
    2. For any \(v\in W_i\text{,}\) we have
      \begin{equation*} v=v_1+v_2+\cdots +v_n\text{,} \end{equation*}
      where
      \begin{equation*} v_j=\begin{cases} \boldzero \amp \text{if } j\ne i \\ v \amp \text{if } j=i \end{cases}\text{.} \end{equation*}
      Since \(v_j\in W_j\) for all \(1\leq j\leq n\text{,}\) we see that \(v\in W\text{.}\) Since \(v\) was an arbitrary element of \(W_i\text{,}\) we conclude that \(W_i\subseteq W\) for all \(1\leq i\leq n\text{.}\)
      Now assume \(U\) is a subspace of \(V\) that contains \(W_i\) for all \(1\leq i\leq n\text{.}\) We will show that \(W=\sum_{i=1}^n W_i\subseteq U\text{.}\) Indeed, given any \(v\in W\text{,}\) we have
      \begin{equation*} v=v_1+v_2+\cdots +v_n \end{equation*}
      for some vectors \(v_1,v_2,\dots, v_n\text{,}\) where \(v_i\in W_i\text{.}\) Since \(W_i\subseteq U\) for all \(i\text{,}\) we have \(v_i\in U\) for all \(i\text{;}\) since \(U\) is closed under addition, we have \(v=v_1+v_2+\cdots +v_n\in U\text{,}\) as desired.

    Example 1.5.3. Sums of subspaces.

    Let \(V=\R^3\) and consider the following subsets of \(V\text{:}\)
    \begin{align*} W_1 \amp = \{(r,0,0)\colon r\in \R\}\\ W_2 \amp = \{(0,s,0)\colon s\in \R\}\\ W_3 \amp = \{(t,t,0)\colon t\in \R\}\text{.} \end{align*}
    1. Prove that each \(W_i\) is a subspace of \(\R^3\text{.}\)
    2. Identify \(W_1+W_2\) as a familiar subspace of \(\R^3\text{.}\) Justify your answer.
    3. Identify \(W_1+W_3\) as a familiar subspace of \(\R^3\text{.}\) Justify your answer.
    4. Identify \(W_1+W_2+W_3\) as a familiar subspace of \(\R^3\text{.}\) Justify your answer.
    Solution.
    1. It is fairly clear that each \(W_i\) above contains the zero vector and is closed under addition and scalar multiplication. We leave a careful proof of these facts to the reader.
    2. Let \(W=\{(a,b,0)\colon a,b\in \R\}\text{,}\) the \(xy\)-plane. Claim: \(W_1+W_2=W\text{.}\) We prove both inclusions of this set equality separately.
      Given \(v\in W_1+W_2\text{,}\) we have \(v=v_1+v_2\) for some \(v_1=(a,0,0)\in W_1\) and \(v_2=(0,b,0)\in W_2\text{.}\) But then
      \begin{equation*} v=(a,0,0)+(0,b,0)=(a,b,0)\in W\text{.} \end{equation*}
      Thus \(W_1+W_2\subseteq W\text{.}\)
      For the other direction, given any \(v=(a,b,0)\in W\text{,}\) we have
      \begin{equation*} v=(a,0,0)+(0,b,0)\in W_1+W_2\text{.} \end{equation*}
      Thus \(W\subseteq W_1+W_2\text{,}\) and we conclude \(W=W_1+W_2\text{.}\)
    3. Claim: \(W_1+W_3=W\text{,}\) with \(W\) as above. The inclusion \(W_1+W_3\subseteq W\) is obvious and can be proved just as above. The reverse inclusion requires a little more work.
      Given \(v=(a,b,0)\in W\) we must show that there are elements \(v_1=(r,0,0)\in W_1\) and \(v_2=(s,s,0)\in W_3\) satisfying
      \begin{equation*} v=(a,b,0)=(r,0,0)+(s,s,0)=(r+s,s,0)\text{.} \end{equation*}
      Equating coordinates we see that we want to find a solution \((r,s)\) to the linear system
      \begin{align*} r+s \amp = a\\ s \amp =b\text{,} \end{align*}
      where \(a\) and \(b\) are given. We easily solve to conclude that there is in fact exactly one solution: \(r=a-b, s=b\text{.}\) In particular we have
      \begin{equation*} v=(a,b,0)=(a-b,0,0)+(b,b,0)\in W_1+W_3\text{.} \end{equation*}
      This proves \(W\subseteq W_1+W_3\) and hence that \(W=W_1+W_3\text{.}\)
    4. Claim: \(W_1+W_2+W_3=W\text{,}\) with \(W\) as above. First observe that since \(W\) is a subspace and \(W_i\subseteq W\) for all \(1\leq i\leq 3\text{,}\) we then have
      \begin{equation*} W_1+W_2+W_3\subseteq W \end{equation*}
      by (2) of Theorem 1.5.2. For the other inclusion, observe that \(W_1+W_2\) by the first part, and thus
      \begin{equation*} W=W_1+W_2\subseteq W_1+W_2+W_3\text{.} \end{equation*}
      We conclude that \(W=W_1+W_2+W_3\text{.}\)

    Definition 1.5.4. Direct sum of subspaces.

    Let \(W_1,W_2,\dots, W_n\) be subspaces of the \(F\)-vector space \(V\text{.}\) We say \(W\) is the direct sum of the subspaces \(W_i\text{,}\) denoted
    \begin{equation} W=W_1\oplus W_2+\cdots \oplus W_n\text{,}\tag{1.8} \end{equation}
    or \(W=\bigoplus_{i=1}^n W_i\text{,}\) if the following conditions hold.
    1. \(W=V_1+V_2+\cdots +V_n\text{.}\)
    2. For each \(v\in W\) there is a unique choice of vectors \(v_i\in W_i\) such that
      \begin{equation} v=v_1+v_2+\cdots +v_n\text{.}\tag{1.9} \end{equation}
      In other words if we have
      \begin{equation*} v_1+v_2+\cdots +v_n=v_1'+v_2'+\cdots +v_n'\text{,} \end{equation*}
      where \(v_i,v_i'\in W_i\) for all \(1\leq i\leq n\text{,}\) then
      \begin{equation*} v_i=v_i' \end{equation*}
      for all \(1\leq i\leq n\text{.}\)
    We call (1.8) a direct sum decomposition of \(W\text{.}\)

    Example 1.5.5. Direct sum.

    Let \(V=\R^3\) and consider the following subspaces of \(V\text{:}\)
    \begin{align*} W_1 \amp = \{(r,0,0)\colon r\in \R\}\\ W_2 \amp = \{(0,s,0)\colon s\in \R\}\\ W_3 \amp = \{(t,t,0)\colon t\in \R\}\\ W \amp = \{(a,b,0)\colon a,b\in \R\}\text{.} \end{align*}
    1. Show that \(W=W_1\oplus W_2\) and \(W=W_1\oplus W_3\text{.}\)
    2. Show that \(W\ne W_1\oplus W_2\oplus W_3\text{.}\)
    Solution.
    1. We know already that \(W=W_1+W_2=W_1+W_3\) from Example 1.5.3. Furthermore, it is clear by inspection that the decomposition
      \begin{equation*} (a,b,0)=v_1+v_2, v_1\in W_1, v_2=W_2 \end{equation*}
      if and only if \(v_1=(a,0,0)\) and \(v_2=(0,b,0)\text{,}\) showing that \(W=W_1\oplus W_2\text{.}\) Furthermore, in Example 1.5.3 we showed that
      \begin{equation*} (a,b,0)=v_1+v_3, v_1\in W_1, v_3\in W_3 \end{equation*}
      if and only if \(v_1=(a-b,0,0), v_2=(b,b,0)\text{.}\) Thus \(W=W_1\oplus W_3\text{.}\)
    2. We know that \(W=W_1+W_2+W_3\) from Example 1.5.3. To see that \(W\) is not the direct sum of the \(W_i\) it suffices to show one example of a vector \((a,b,0)\in W\) that can be decomposed into a sum of elements of \(W_1,W_2,W_3\) in two distinct ways. Here is such an example:
      \begin{align*} (1,1,0) \amp =(0,0,0)+(0,0,0)+(1,1,0)\\ \amp =(1,0,0)+(0,1,0)+(0,0,0)\text{.} \end{align*}

    Example 1.5.6. Direct sum decomposition of \(F^n\).

    Consider the space \(F^n\) of \(n\)-tuples with coefficients in \(F\text{.}\) For each \(1\leq i\leq n\text{,}\) let \(W_i\) be the set of all \(n\)-tuples of the form
    \begin{equation*} v=(0,\dots,0,\underset{i\text{-th coor.}}{a}, 0,\dots, 0), a\in F\text{.} \end{equation*}
    In other words \(W_i\) is the set of all \(n\)-tuples whose \(j\)-th coordinates are equal to zero for all \(j\ne i\text{.}\)
    Prove that
    \begin{equation*} F^n=W_1\oplus W_2\cdots \oplus W_n\text{.} \end{equation*}
    You may take for granted that \(W_i\) is a subspace of \(F^n\) for all \(i\text{.}\)
    Solution.
    Given \(v=(a_1,a_2,\dots, a_n)\in F^n\) we have
    \begin{equation} v=v_1+v_2+\cdots +v_n\text{,}\tag{1.10} \end{equation}
    where
    \begin{equation*} v_i=(0,\dots, a_i,0,\dots, 0)\in W_i\text{.} \end{equation*}
    It follows that \(F^n=\sum_{i=1}^n W_i\text{.}\) Furthermore, the decomposition (1.10) is unique: if
    \begin{equation*} (a_1,a_2,\dots, a_n)=v_1'+v_2'+\cdots +v_n' \end{equation*}
    with \(v_i'\in W_i\) for all \(i\text{,}\) then since \(v_i'\) is the only vector with a nonzero \(i\)-th coordinate, we must have \(v_i'=(0,\dots, 0,a_i,0,\dots,0)\text{.}\) Since the decompotision (1.10) is unique, we conclude that \(F^n=\bigoplus_{i=1}^nW_i\text{.}\)

    Theorem 1.5.7. Direct sum of subspaces.

    Let \(W_1,W_2,\dots, W_n\) be subspaces of the \(F\)-vector space \(V\text{,}\) and let \(W=\sum_{i=1}^n W_i\text{.}\) The following statements are equivalent.
    1. \(\displaystyle W=\bigoplus_{i=1}^n W_i\)
    2. If \(\boldzero=v_1+v_2+\cdots +v_n\) with \(v_i\in W_i\) for all \(1\leq i\leq n\text{,}\) then \(v_i=\boldzero\) for all \(1\leq i\leq n\text{.}\) Using logical shorthand:
      \begin{equation} \boldzero=v_1+v_2+\cdots +v_n, v_i\in W_i\implies v_1=v_2\cdots =v_n=\boldzero \text{.}\tag{1.11} \end{equation}

    Proof.

    Since the premises include the assumption that \(W=\sum_{i=1}^n W_i\) we need only show that condition (ii) of Definition 1.5.4 is equivalent to condition (1.11). We prove both implications separately.
    Suppose condition (ii) of Definition 1.5.4 is satisfied. Since \(\boldzero=\boldzero+\boldzero+\cdots +\boldzero\) is one decomposition of \(\boldzero\) into elements of the \(W_i\text{,}\) and since (ii) asserts such decompositions are unique, we conclude that (1.11) holds.
    Assume (1.11) holds, and suppose we have
    \begin{equation*} v=v_1+v_2+\cdots +v_n=v_1'+v_2'+\cdots +v_n' \end{equation*}
    with \(v_i, v_i'\in W_i\text{.}\) It follows, after a little algebra, that
    \begin{equation*} (v_1-v_1')+(v_2-v_2')+\cdots +(v_n-v_n')=\boldzero\text{.} \end{equation*}
    Since we assume condition (1.11), we conclude that \(v_i-v_i'=\boldzero\) for all \(1\leq i\leq n\text{,}\) or equivalently, \(v_i=v_i'\) for all \(1\leq i\leq n\text{,}\) as desired.

    Corollary 1.5.8.

    Let \(W_1\) and \(W_2\) be subspaces of the \(F\)-vector space \(V\text{,}\) and let \(W=W_1+W_2\text{.}\) The following statements are equivalent.
    1. \(W=W_1\oplus W_2\text{.}\)
    2. \(W_1\cap W_2=\{\boldzero\}\text{.}\)

    Proof.

    We prove both implications separately.

    Implication: (1)\(\implies\)(2).

    Assume \(W=W_1\oplus W_2\text{.}\) Since \(0\in W_i\) for \(1\leq i\leq 2\text{,}\) we have \(\{\boldzero\}\subseteq W_1\cap W_2\text{.}\) It remains to prove the reverse inclusion \(W_1\cap W_2\subseteq \{\boldzero\}\text{.}\) Given \(v\in W_1\cap W_2\text{,}\) we have \(v\in W_1\) and \(-v\in W_2\text{,}\) and thus
    \begin{equation*} \boldzero=v+ (-v) \end{equation*}
    is a decomposition of \(\boldzero\) in terms of \(W_1\) and \(W_2\text{.}\) Since \(W=W_1\oplus W_2\text{,}\) we conclude from Theorem 1.5.7 that \(v=-v=\boldzero\text{.}\) Thus \(W_1\cap W_2=\{\boldzero\}\text{.}\)

    Implication: (2)\(\implies\)(1).

    Now assume \(W_1\cap W_2=\{\boldzero\}\text{.}\) Using Theorem 1.5.7, it suffices to show that if \(\boldzero=v_1+v_2\) with \(v_1\in W_1\) and \(v_2\in W_2\text{,}\) then \(v_1=v_2=\boldzero\text{.}\) If \(\boldzero=v_1+v_2\) with \(v_i\in W_i\text{,}\) then we have
    \begin{align*} v_1\amp =-v_2\in W_2 \\ v_2 \amp =-v_1\in W_1\text{,} \end{align*}
    and hence that \(v_1\in W_1\cap W_2\) and \(v_2\in W_1\cap W_2\text{.}\) Since \(W_1\cap W_2=\{\boldzero\}\text{,}\) we conclude that \(v_1=v_2=\boldzero\text{,}\) as desired.

    Definition 1.5.9. Matrix notation.

    Let \(F\) be a field. We adopt the following notation for matrices of \(F^{m,n}\text{.}\)
    • Matrix-building notation.
      The notation \([a_{ij}]_{m\times n}\) denotes the matrix of \(F^{m,n}\) whose \(ij\)-th entry is \(a_{ij}\) for all \(1\leq i\leq m\) and \(1\leq j\leq n\text{.}\) When there is no danger of confusion, this notation will be shortened to \([a_{ij}]\text{.}\)
    • Matrix entry notation.
      Given a matrix \(A\in F^{m,n}\text{,}\) for all \(1\leq i\leq m\) and \(1\leq j\leq n\text{,}\) \([A]_{ij}\) denotes the \(ij\)-th entry of \(A\text{.}\)

    Definition 1.5.10. Transpose matrix.

    Given a matrix \(A\in F^{m,n}\text{,}\) its transpose is the matrix \(A^T\in F^{n,m}\) whose \(ij\)-th entry is the \(ji\)-th entry of \(A\) for all \(1\leq i\leq n, 1\leq j\leq m\text{.}\) In other words, \(A^T\) is the unique matrix of \(F^{n,m}\) satisfying
    \begin{equation*} [A^T]_{ij}=[A]_{ji} \end{equation*}
    for all \(1\leq i\leq n\) and \(1\leq j\leq m\text{.}\)

    Example 1.5.11. Direct sum of matrix spaces.

    Let \(V=F^{2,2}\text{,}\) the space of all \(2\times 2\) matrices with entries in \(F\text{,}\) and consider the following subsets of \(V\text{:}\)
    \begin{align*} W_1 \amp =\{A\in F^{2,2}\colon A^T=A\}\\ W_2 \amp = \{A\in F^{2,2}\colon A^T=-A\}\text{.} \end{align*}
    In other words, \(W_1\) is the set of all symmetric matrices of \(F^{2,2}\) and \(W_2\) is the set of all skew-symmetric matrices of \(F^{2,2}\text{.}\)
    1. Prove that \(W_1\) and \(W_2\) are subspaces of \(F^{2,2}\text{.}\)
    2. Prove that \(F^{2,2}=W_1\oplus W_2\text{.}\)
    Solution.
    1. Using the defining conditions \(A^T=A\) and \(A^T=-A\text{,}\) it is easy to see that
      \begin{align*} W_1 \amp =\left\{\begin{bmatrix} a\amp b \\ b\amp c\end{bmatrix}\colon a,b,c\in F\right\}\\ W_2 \amp =\left\{\begin{bmatrix} 0\amp b \\ -b\amp 0\end{bmatrix}\colon b\in F\right\}\text{.} \end{align*}
      From these descriptions it follows easily that \(W_1\) and \(W_2\) contain the zero matrix, and are closed under addition and scalar multiplication. We leave the details to the reader.
    2. Let \(W=W_1+W_2\text{.}\) First we show that \(W=W_1\oplus W_2\text{.}\) To do so, it suffices to show \(W_1\cap W_2=\{\boldzero\}\) by Corollary 1.5.8. But this is clear from the parametric descriptions of \(W_1\) and \(W_2\) given above.
      Next we show that \(W=F^{2,2}\text{.}\) By definition we have \(W=W_1+W_2\subseteq F^{2,2}\text{,}\) so we need only show the reverse inclusion. Given an arbitrary matrix \(A\in F^{2,2}\text{,}\) we must show that we can write \(A=A_1+A_2\) where \(A_1\in W_1\) and \(A_2\in W_2\text{.}\) In other words, we must show that any matrix can be written as a sum of a symmetric matrix and a skew-symmetric matrix. This can be done essentially by inspection in this low-dimensional case (\(n=2\)): we have
      \begin{equation*} \begin{bmatrix} a\amp b\\ c\amp d \end{bmatrix} =\frac{1}{2}\begin{bmatrix} 2a\amp b+c \\ b+c \amp 2d \end{bmatrix}+ \frac{1}{2}\begin{bmatrix} 0\amp b-c \\ -b+c \amp 0 \end{bmatrix} \end{equation*}
      for any matrix \(A=\begin{bmatrix} a\amp b\\ c\amp d \end{bmatrix}\text{.}\) This fact is true more generally, however, and so it is worth adding an argument that works for all \(n\text{:}\) namely, observe that \(A=A_1+A_2\text{,}\) where
      \begin{align*} A_1 \amp =\frac{1}{2}(A+A^T)\\ A_2 \amp =\frac{1}{2}(A-A^T)\text{.} \end{align*}
      Using elementary properties of transpose, it is now easy to show that \(A_1^T=A_1\) and \(A_2^T=-A_2\text{:}\) i.e., \(A_1\in W_1\) and \(A_2\in W_2\text{.}\) We will remind you of these elementary properties of the transpose operator later on.

    Section 1.2 Vector spaces

    In the last section we isolated useful number system properties of \(\R\) and generalized them in the form of our axiomatic definition of a field (Definition 1.1.11). Similarly, in this section we give a similar treatment to the type of vector spaces usually studied in a first course in linear algebra: that is, the space of real \(n\)-tuples \(\R^n=\{(a_1,a_2,\dots, a_n)\colon a_i\in \R\}\text{.}\) The two operations on \(\R^n\) we choose to generalize are its version of vector addition
    \begin{align*} \R^n\times \R^n \amp \rightarrow \R^n\\ ((a_1,a_2,\dots, a_n),(b_1,b_2,\dots, b_n))\amp\mapsto (a_1+b_1, a_2+b_2,\dots, a_n+b_n) \text{,} \end{align*}
    and its version of scalar multiplication
    \begin{align*} \R\times \R^n \amp \rightarrow \R^n\\ (c,(a_1,a_2,\dots, a_n)) \amp\rightarrow (ca_1,ca_2,\dots, ca_n) \text{.} \end{align*}
    You should think of the axioms of Definition 1.2.2 as an enumeration of precisely which properties of these two operations on \(\R^n\) we want a structure to enjoy in order to ensure that we can perform a form of vector arithmetic and algebra just as we do in \(\R^n\text{.}\) Note that there is an important qualitative difference between the two vector operations on \(\R^n\) that is treated carefully in Definition 1.2.2; namely, whereas vector addition takes as its input two vectors \(v,w\in \R^n\) and returns another, scalar multiplication takes a scalar \(c\in \R\) and a vector \(v\in \R^n\) and returns another vector. In this sense, scalar multiplication should be thought of as a sort of hybrid operation.

    Fiat 1.2.1. \(F\) stands for field.

    Henceforth, unless stated otherwise, \(F\) will always denote a field. For any explicit example in this course the field \(F\) will be either \(\R\) or \(\C\text{.}\) However, all statements of theory formulated in terms of \(F\) are understood to be valid for any field.

    Definition 1.2.2. Vector space.

    Let \(F\) be a field. A vector space over \(F\) (or \(F\)-vector space) is a set \(V\) together with two operations
    \begin{align*} V\times V \amp \rightarrow V \amp F\times V\amp \rightarrow V\\ (v,w) \amp\mapsto v+w \amp (c,v)\amp \mapsto cv \text{,} \end{align*}
    called respectively vector addition and scalar multiplication, that satisfy the following vector space axioms.
    1. Vector addition is commutative.
      \(v+w=w+v\) for all \(v,w\in V\text{.}\)
    2. Vector addition is associative.
      \((u+v)+w=u+(v+w)\) for all \(u, v, w\in V\text{.}\)
    3. Zero vector.
      There is an element \(\boldzero\in V\) such that for all \(v\in V\text{,}\) we have
      \begin{equation} \boldzero+v=v\text{.}\tag{1.3} \end{equation}
      We call \(\boldzero\) the zero vector of \(V\text{.}\)
    4. Vector inverses.
      For all \(v\in V\text{,}\) there is another element \(-v\) satisfying
      \begin{equation} -v+v=\boldzero\text{.}\tag{1.4} \end{equation}
      We call \(-v\) the vector inverse of \(v\text{.}\)
    5. Distribution over vector addition.
      \(c(v+w)=cv+cw\) for all \(c\in F\) and \(v, w\in V\text{.}\)
    6. Distribution over scalar addition.
      \((c+d)v=cv+dv\) for all \(c,d\in F\) and \(v\in V\text{.}\)
    7. Scalar multiplication is associative.
      \(c(dv)=(cd)v\) for all \(c,d\in F\) and all \(v\in V\text{.}\)
    8. Scalar multiplication identity.
      \(1\,v=v\) for all \(v\in V\text{.}\)
    We call elements of the vector space \(V\) vectors and the elements of \(F\) scalars.

    Remark 1.2.3. Arithmetic and existential axioms.

    As with the axioms in Definition 1.1.11 we divide the vector space axioms into the arithmetic axioms (Axioms (i)-(ii),(v)-(vi),(vii)) and the existential axioms (Axioms (iii)-(iv)).
    We now proceed to a litany of examples. Each one will be stated as a definition (for reference purposes), but nonetheless requires a proof that the given structure does indeed constitute a vector space. In a classic mathematical move, we begin with the simplest of all vector spaces, the zero space. Elementary as this example is, it serves well to illustrate the axiomatic nature of Definition 1.2.2.

    Definition 1.2.4. Zero space.

    Let \(F\) be a field, and let \(V=\{\bot\}\text{,}\) a set containing exactly one element. There is a unique \(F\)-vector space structure that can be given to \(V\text{,}\) defined as follows.
    • Vector operations.
      Vector addition on \(V\) is defined as \(\bot+\bot=\bot\text{;}\) scalar multiplication on \(V\) is defined as \(c\cdot \bot=\bot\) for all \(c\in F\text{.}\)
    • Zero vector.
      The zero vector of \(V\) is \(\bot\text{:}\) i.e., \(\boldzero=\bot\text{.}\)
    • Vector inverses.
      The vector inverse of \(\bot\) is \(\bot\text{:}\) i.e., \(-\bot=\bot\text{.}\)
    Since \(\bot=\boldzero\) with respect to this vector space structure, we have \(V=\{\boldzero\}\text{.}\) Accordingly, we call \(V\) a zero space.
    Definition 1.2.4 makes two claims: that the given operations make \(V=\{\bot\}\) into a vector space, and that this is the only way to make \(V=\{\bot\}\) into a vector space. As with all claims in mathematics, these need to be proved, but as you will see, the proof is a very light affair.

    Proof for Definition 1.2.4.

    Since \(V=\{\bot\}\) only has one item, there is no choice for what vector addition \(V\times V\rightarrow V\) and scalar multiplication \(F\times V\rightarrow V\) can be. They must be defined in the manner given in Definition 1.2.4. Similarly, we must have \(\boldzero=\bot\) and \(-\bot=\bot\text{,}\) as once again, \(\bot\) is the only element of \(V\text{!}\) This shows that there can be at most one way of giving \(V\) a vector space structure.
    It is now easy to see that these choices do indeed satisfy the vector space axioms. That \(\bot\) satisfies the identity of Axiom iii defining the zero vector \(\boldzero\) follows from the fact that for all \(v\in V\) we have \(v=\bot\) (since \(V=\{\bot\}\)), and thus
    \begin{align*} \bot+v \amp =\bot+\bot\\ \amp = \bot\\ \amp =v\text{.} \end{align*}
    Thus \(\bot=\boldzero\) is the zero vector of the space.
    Similarly, to show all elements of \(V\) have vector inverses amounts to showing that \(\bot\) has a vector inverse, since this is the only element of \(V\text{.}\) It is claimed that \(-\bot=\bot\) (i.e., \(\bot\) is its own vector inverse), which follows from the fact that
    \begin{align*} -\bot+\bot \amp = \bot+\bot\\ \amp = \bot\\ \amp = \boldzero\text{.} \end{align*}
    Lastly, the identities of Axioms (i)-(ii) and (v)-(viii) in this setting all reduce to trivial statements of the form \(\bot+\bot=\bot+\bot\text{.}\) Consider Axiom (vii), for example. For all \(v,w\in V\text{,}\) we have \(v=w=\bot\text{,}\) in which case
    \begin{align*} c(v+w) \amp =c(\bot+\bot) \\ \amp =c\cdot \bot\\ \amp = \bot \amp \text{(def. of sc. mult.)} \end{align*}
    and
    \begin{align*} cv+dw \amp =c\cdot \bot+d\cdot \bot\\ \amp = \bot+\bot\\ \amp =\bot\text{.} \end{align*}
    Thus
    \begin{equation*} c(v+w)=\bot=cv+dw \end{equation*}
    for all \(v, w\in V\) and \(c\in F\text{.}\)
    We leave verification of the rest of the axioms to the reader.
    It is worth formalizing the proof technique used above into an official procedure for showing whether something is a vector space.

    Procedure 1.2.5. Vector space verification.

    To decide whether a given set and operations is a vector space, proceed as follows.
    1. Make explicit the underlying set \(V\) of the proposed vector space.
    2. Make explicit what the scalar multiplication and vector addition operations are.
    3. Identify an element of \(V\) that serves as the zero vector (i.e., satisfies (1.3)) and for each \(\boldv\in V\) show that there is a vector \(-\boldv\) satisfying (1.4).
    4. Show that the two vector operations and our choice of zero vector and vector inverses satisfy the axioms of Definition 1.2.2.

    Remark 1.2.6. Vector space verification.

    Think of steps (1)-(3) of Procedure 1.2.5 as the issuing of official declarations about the makeup of our proposed vector space: “The underlying set shall be as stated”; “We declare the vector operations thusly”; “The zero vector shall be this element here, and vector inverses shall be assigned in this manner”. Step (4) is where we get down to the nitty gritty of showing that the proposed vector space structure articulated in (1)-(3) does indeed satisfy all the necessary properties.
    In each of the remaining examples below we carefully lay out the details of items (1)-(3) while often leaving much of the work of item (4) to you. You will meet these vector spaces frequently throughout the rest of your life. Each time you do, it will be helpful for orientation purposes to mentally run through items (1)-(3). Ask yourself: What is the underlying set? What are vector operations? What acts as the zero vector, and how do I assign vector inverses?
    We now work our way up in complexity, considering first three examples that might already be familiar to you in some form or other. Note that (a) the vector space operations for each of Definition 1.2.7, Definition 1.2.9, Definition 1.2.11 are all very similar in nature (roughly speaking the operations are always defined “component-wise”), and (b) that we postpone the verification of the vector space axioms for these examples until Theorem 1.3.2 in the next section! This is because each of the vector spaces in Definition 1.2.7–1.2.11 is in fact one particular example of the vastly more general vector space defined in Definition 1.3.1! We will make this clear in the proof of Theorem 1.3.2. For now, consider the treatment of the vector spaces below simply as a “getting to know” session.

    Definition 1.2.7. \(n\)-tuples.

    Let \(F\) be a field. Given a positive integer \(n\text{,}\) recall that \(F^n\) is the set of all \(F\)-valued \(n\)-tuples: i.e.,
    \begin{equation*} F^n=\{(a_1,a_2,\dots, a_n)\colon a_i\in F\}\text{.} \end{equation*}
    Below we define a vector space structure on \(F^n\text{.}\)
    • Vector operations.
      Given \(v=(a_1,a_2,\dots, a_n), w=(b_1,b_2,\dots, b_n)\in F^n\) and \(c\in \F\text{,}\) we define
      \begin{align*} v+w \amp =(a_1+b_1,a_2+b_2,\cdots, a_n+b_n)\\ cv \amp =(ca_1,ca_2,\dots, ca_n)\text{.} \end{align*}
    • Zero vector.
      The zero vector of \(F^n\) is \(\boldzero=(0,0,\dots, 0)\text{.}\)
    • Vector inverses.
      Given a sequence \(v=(a_1,a_2,\dots, a_n)\in F^n\text{,}\) its vector inverse \(-v\) is
      \begin{equation*} -v=(-a_1,-a_2,\dots, -a_n)\text{.} \end{equation*}
    We call \(F^n\text{,}\) together with these vector operations, the space of \(F\)-valued \(n\)-tuples (or the space of \(n\)-tuples with coordinates in \(F\)).

    Proof.

    The proof that \(F^n\) is a vector space with respect to these operations is postponed until Theorem 1.3.2.

    Example 1.2.8. Complex pairs.

    Consider the complex vector space \(\C^2=\{(z,w)\colon z,w\in \C\}\text{.}\) Given vectors \(v=(1+i, 2i), w=(i, 5)\in \C^2\) and scalar \(1-i\in \C\text{,}\) we have
    \begin{align*} v+w \amp = (1+2i, 5+2i)\\ (1-i)v \amp =((1-i)(1+i),(1-i)2i)\\ \amp = (2,2+2i)\text{.} \end{align*}

    Definition 1.2.9. Space of infinite sequences.

    Let \(F\) be a field. Recall that \(F^\infty\) is the set of all \(F\)-valued infinite sequences: i.e.,
    \begin{equation*} F^\infty=\{(a_i)_{i=1}^\infty\colon a_i\in F \text{ for all } i\in \Z_+\}\text{.} \end{equation*}
    Below we define a vector space structure on \(F^\infty\text{.}\)
    • Vector operations.
      Given \(s=(a_i)_{i=1}^\infty, t=(b_i)_{i=1}^\infty\in F^\infty\) and \(c\in F\text{,}\) we define
      \begin{align*} s+t\amp=(a_i+b_i)_{i=1}^\infty=(a_1+b_1, a_2+b_2,\dots) \\ cs \amp =(ca_i)_{i=1}^\infty=(ca_1,ca_2,\dots) \text{.} \end{align*}
    • Zero vector.
      The zero vector of \(F^\infty\) with respect to these operations is
      \begin{equation*} \boldzero=(0)_{i=1}^\infty=(0,0,\dots)\text{.} \end{equation*}
    • Vector inverses.
      Given a sequence \(s=(a_i)_{i=1}^\infty\in F^\infty\text{,}\) its vector inverse is
      \begin{align*} -s \amp = (-a_i)_{i=1}^\infty=(-a_1,-a_2,\dots )\text{.} \end{align*}
    We call \(F^\infty\text{,}\) together with these vector operations, the space of infinite sequences with entries in \(F\) (or the space of \(F\)-valued infinite sequences).

    Proof.

    The proof that \(F^n\) is a vector space with respect to these operations is postponed until Theorem 1.3.2.

    Example 1.2.10. Complex infinite sequences.

    Consider the complex vector space \(\C^\infty\) of all infinite sequences with complex entries. Given sequences \(s=(0,1,0,1,0,\dots)\) and \(t=(i,i,i,\dots) \text{,}\) and scalar \(i\in \C\text{,}\) we have
    \begin{align*} s+t \amp =(i,i+1,i,i+1,i,\dots)\\ it \amp =(i^2,i^2,\dots)\\ \amp =(-1,-1,\dots )\text{.} \end{align*}

    Definition 1.2.11. Space of matrices.

    Let \(F\) be a field. Given positive integers \(m,n\in\Z_+\text{,}\) we define \(F^{m,n}\) to be the set of all \(m\times n\) matrices with entries in \(F\text{:}\) i.e.,
    \begin{equation*} F^{m,n}=\{\underset{m\times n}{A}=[a_{ij}]\colon a_{ij}\in F \text{ for all } (i,j)\in \{1,2\dots, m\}\times \{1,2,\dots, n\}\}\text{.} \end{equation*}
    Below we define a vector space structure on \(F^{m,n}\text{.}\)
    • Vector operations.
      Given matrices \(A=[a_{ij}], B=[b_{ij}]\in F^{m,n}\) and \(c\in F\text{,}\) we define
      \begin{align*} A+B \amp =[a_{ij}+b_{ij}]\\ cA \amp =[ca_{ij}]\text{.} \end{align*}
    • Zero vector.
      The zero vector of \(F^{m,n}\) is the the zero matrix
      \begin{equation*} \boldzero=[0]_{\substack{1\leq i\leq m\\ 1\leq j\leq n}}\text{.} \end{equation*}
    • Vector inverses.
      Given a matrix \(A=[a_{ij}]\in F^{m,n}\) its vector inverse is
      \begin{equation*} -A=[-a_{ij}]\text{.} \end{equation*}
    We call \(F^{m,n}\text{,}\) together with these vector operations, the space of \(m\times n\) matrices with entries in \(F\) (or the space of \(F\)-valued \(m\times n\) matrices).

    Proof.

    The proof that \(F^n\) is a vector space with respect to these operations is postponed until Theorem 1.3.2.

    Example 1.2.12. Complex matrices.

    Consider the vector space \(\C^{2,2}\) of all \(2\times 2\) matrices with complex entries. Given matrices
    \begin{align*} A \amp = \begin{bmatrix}2i\amp 1+i \\ 0 \amp 3\end{bmatrix} \\ B \amp = \begin{bmatrix}-3-i\amp 2 \\ i \amp 4+4i\end{bmatrix} \end{align*}
    and scalar \(1+i\in \C\text{,}\) we have
    \begin{align*} A+B\amp = \begin{bmatrix}-3-i\amp 3+i \\ i \amp 7+4i\end{bmatrix} \\ (1+i)A \amp = \begin{bmatrix}-2+2i\amp 2i \\ 0 \amp 3+3i\end{bmatrix} \text{.} \end{align*}

    Section 1.20 Orthogonal projection

    A trick we learn early on in physics-- specifically, in dynamics problems in \(\R^2\)-- is to pick a convenient axis and then decompose any relevant vectors (force, acceleration, velocity, position, etc.) into a sum of two components: one that points along the chosen axis, and one that points perpendicularly to it. As we will see in this section, this technique can be vastly generalized. Namely, instead of \(\R^2\) we can take any inner product space \((V, \langle\, , \rangle)\text{;}\) and instead of a chosen axis in \(\R^2\text{,}\) we can choose any finite-dimensional subspace \(W\subseteq V\text{;}\) then any \(v\in V\) can be decomposed in the form
    \begin{equation*} v=w+w^\perp, \end{equation*}
    where \(w\in W\) and \(w^\perp\) is a vector orthogonal to \(W\text{,}\) in a sense we will make precise below. Just as in our toy physics example, this manner of decomposing vectors helps simplify computations in problems where the subspace \(W\) chosen is of central importance.

    Subsection Orthogonal complement

    Definition 1.20.1. Orthogonal complement.

    Let \((V,\angvec{\, ,})\) be an inner product space. Given a subspace \(W\subseteq V\text{,}\) its orthogonal complement \(W^\perp\) is the set of all vectors of \(v\) that are orthogonal to every vector in \(W\text{:}\) i.e.,
    \begin{equation} W^\perp=\{v\in V\colon \angvec{v,w}=0 \text{ for all } w\in W\}\text{.}\tag{1.72} \end{equation}
    The next theorem tells us that to check \(w\in W^\perp\text{,}\) it suffices to check that \(w\) is orthogonal to the elements of a spanning set of \(W\text{.}\)

    Theorem 1.20.2. Orthogonality and spanning sets.

    Let \((V,\angvec{\, , })\) be an inner product space, and let \(W=\Span (w_i)_{i\in I}\text{.}\) We have \(v\in W^\perp\) if and only if \(\angvec{v,w_i}=0\) for all \(i\in I\text{:}\) i.e.,
    \begin{equation*} W^\perp=\{v\in V\colon \angvec{v,w_i}=0 \text{ for all } i\in I\}\text{.} \end{equation*}

    Proof.

    The forward implication follows purely from logic. Conversely, assume that \(\angvec{v,w_i}=0\) for all \(i\in I\text{.}\) Given \(w\in W\text{,}\) we can write \(w=\sum_{i\in J}c_{i}v_{i}\) for some finite subset \(J\subseteq I\text{,}\) in which case
    \begin{align*} \angvec{v,w} \amp =\sum_{i\in J}\overline{c_i}\angvec{v,\sum_{i\in J}c_iw_i}\\ \amp =\sum_{i\in J}\overline{c_i}\angvec{v,w_i}\\ \amp =\sum_{i\in J}\overline{c_i}\cdot 0\\ \amp = 0\text{.} \end{align*}
    Thus \(v\in W^\perp\text{.}\)
    Theorem 1.20.2 is especially useful for computing \(W^\perp\) when \(W\) admits a finite basis or spanning set.

    Example 1.20.3.

    Consider the inner product space \(\R^2\) together with the dot product. Let \(W=\Span\left((1,1)\right)=\{(t,t)\colon t\in \R\}\text{:}\) the line \(\ell\subseteq \R^2\) with equation \(y=x\text{.}\) Compute \(W^\perp\) and identify it as a familiar geometric object in \(\R^2\text{.}\)
    Solution.
    According to Theorem 1.20.2, since \(W=\Span((1,1))\text{,}\) we have
    \begin{equation*} \boldx\in W^\perp \iff \boldx\cdot (1,1)=0\text{.} \end{equation*}
    Letting \(\boldx=(x,y)\text{,}\) we see that \(\boldx\cdot (1,1)=0\) if and only if \(x+y=0\text{,}\) if and only if \(y=-x\text{.}\) Thus \(W^\perp=\{(x,y)\colon y=-x\}\) is the line \(\ell'\subseteq \R^2\) with equation \(y=-x\text{.}\) Observe that the lines \(\ell\) and \(\ell'\) are indeed perpendicular to one another. (Graph them!)

    Example 1.20.4.

    Consider the inner product space \(\R^3\) together with the dot product. Let \(W\subseteq \R^3\) be the plane with equation \(x-2y-z=0\text{.}\) Compute \(W^\perp\) and identify this as a familiar geometric object in \(\R^3\text{.}\)
    Solution.
    First, solving \(x-2y-z=0\) for \((x,y,z)\text{,}\) we see that
    \begin{equation*} W=\{(2s+t,s,t)\colon s,t\in \R\}=\Span\left((2,1,0),(1,0,1)\right)\text{.} \end{equation*}
    Next, according to Theorem 1.20.2 we have
    \begin{equation*} \boldx\in W^\perp\iff \boldx\cdot (2,1,0)=0 \text{ and } \boldx\cdot (1,0,1)=0\text{.} \end{equation*}
    It follows that \(W^\perp\) is the set of vectors \(\boldx=(x,y,z)\) satisfying the linear system
    \begin{equation*} \begin{linsys}{3} 2x\amp +\amp y \amp \amp \amp = \amp 0\\ x\amp \amp \amp +\amp z \amp = \amp 0 \end{linsys}. \end{equation*}
    Solving this system using Gaussian elimination we conclude that
    \begin{equation*} W^\perp=\{(t,-2t,-t)\colon t\in \R\}=\Span\left((1,-2,-1)\right)\text{,} \end{equation*}
    which we recognize as the line \(\ell\subseteq \R^3\) passing through the origin with direction vector \((1,-2,-1)\text{.}\) This is none other than the normal line to the plane \(W\) passing through the origin.

    Theorem 1.20.5. Orthogonal complement.

    Let \(W\) be a subspace of the inner product space \((V,\angvec{\, ,})\text{.}\)
    1. \(W^\perp\) is a subspace of \(V\text{.}\)
    2. \(W\cap W^\perp=\{\boldzero\}\text{.}\)
    3. \(W\subseteq (W^\perp)^\perp\text{.}\)
    4. \(W\) finite dimensional.
      If \(W\) is finite dimensional, then
      \begin{align} V \amp= W\oplus W^\perp \text{.}\tag{1.73} \end{align}
      Furthermore, we have
      \begin{equation} (W^\perp)^\perp=W\tag{1.74} \end{equation}
      in this setting.

    Proof.

    1. We use the 2-step technique.
      Since \(\angvec{\boldzero,v}=0\) for all \(v\in V\text{,}\) it is certainly true that \(\angvec{\boldzero,w}=0\) for all \(w\in W\text{.}\) Thus \(\boldzero\in W^\perp\text{.}\)
    2. Assume \(v_1,v_2\in W^\perp\text{.}\) By definition this means
      \begin{equation*} \angvec{v_1,w}=\angvec{v_2,w}=0 \end{equation*}
      for all \(w\in W\text{.}\) It follows that for any \(c,d\in F\text{,}\) and for all \(w\in W\) we have
      \begin{align*} \angvec{cv_1+dv_2,w} \amp = c\angvec{v_1,w}+d\angvec{v_2,w}\\ \amp =c\cdot 0+d\cdot 0\\ \amp =0\text{,} \end{align*}
      and thus that \(cv_1+dv_2\in W^\perp\text{.}\)
    3. Assume \(v\in W\cap W^\perp\text{.}\) Since \(v\in W\) and \(v\in W^\perp\text{,}\) we must have \(\angvec{v,v}=0\text{.}\) The positivity axiom for inner products then implies that \(v=\boldzero\text{.}\) Thus \(W\cap W^\perp=\{\boldzero\}\text{.}\)
    4. Let \(w\in W\text{.}\) Given any \(v\in W^\perp\text{,}\) we have
      \begin{align*} \angvec{w,v} \amp =\overline{\angvec{v,w}}=\overline{0}=0\text{.} \end{align*}
      Thus \(\angvec{w,v}=0\) for all \(v\in W^\perp\text{,}\) showing that \(w\in (W^\perp)^\perp\text{.}\) This proves \(W\subseteq (W^\perp)^\perp\text{.}\)
    5. Assume \(W\) is finite dimensional. By definition this means \(W\) has a finite basis. The Gram-Schmidt procedure then guarantees that we can produce an orthogonal basis \(B=(w_1,w_2,\dots, w_r)\) of \(W\text{.}\) Furthermore, the proof of the validity of the Gram-Schmidt procedure shows that for any \(v\in V\) the vector
      \begin{equation*} w^\perp=v-\sum_{i=1}^r\frac{\angvec{v,w_i}}{\angvec{w_i,w_i}}w_i \end{equation*}
      is orthogonal to \(w_i\) for all \(1\leq i\leq r\text{.}\) Since \(W=\Span ((w_1,w_2,\dots, w_n))\text{,}\) we conclude from Theorem 1.20.2 that \(w^\perp\in W^perp\text{.}\) Noting that
      \begin{equation*} w=\sum_{i=1}^r\frac{\angvec{v,w_i}}{\angvec{w_i,w_i}}w_i\in W\text{,} \end{equation*}
      we see that for all \(v\in V\) we can write
      \begin{equation*} v=w+w^\perp\text{,} \end{equation*}
      where \(w\in W\) and \(w^\perp\in W^\perp\text{.}\) This proves that \(V=W+W^\perp\text{.}\) Since \(W\cap W^\perp=\{\boldzero\}\) by (2), we conclude that \(V=W\oplus W^\perp\text{.}\)
      Lastly, we show in this case that \(W=(W^\perp)^\perp\text{.}\) Using (3) above, it suffices to show \((W^\perp)^\perp\subseteq W\text{.}\) Take \(v\in (W^\perp)^\perp\text{,}\) and write \(v=w+w^\perp\text{,}\) where \(w\in W\) and \(w^\perp\in W\text{.}\) Since \(v\in (W^\perp)^\perp\text{,}\) we have
      \begin{align*} 0 \amp =\angvec{v,w^\perp}\\ \amp =\angvec{w+w^\perp,w^\perp}\\ \amp =\angvec{w,w^\perp}+\angvec{w^\perp,w^\perp}\\ \amp =0+\angvec{w^\perp,w^\perp}\\ \amp = \angvec{w^\perp,w^\perp}\text{.} \end{align*}
      Since \(\angvec{w^\perp,w^\perp}=0\text{,}\) we conclude that \(w^\perp=\boldzero\text{,}\) and hence that
      \begin{equation*} v=w+w^\perp=w+\boldzero=w\in W\text{,} \end{equation*}
      as desired.

    Subsection Orthogonal projection

    Definition 1.20.6. Orthogonal projection.

    Let \((V, \langle\, , \rangle)\) be an inner product space, and let \(W\) be a finite-dimensional subspace of \(V\text{.}\) Since \(V=W\oplus W^{\perp}\text{,}\) given any \(v\in V\) there is a unique choice of \(w\in W\) and \(w^{\perp}\in W^{\perp}\) satisfying
    \begin{equation} v=w+w^{\perp}.\tag{1.75} \end{equation}
    The vectors \(w\) and \(w^{\perp}\) are the orthogonal projections of \(v\) onto \(W\) and \(W^{\perp}\text{,}\) respectively, and are denoted \(w=\proj_W(v)\) and \(w^{\perp}=\proj_{W^\perp}(v)\text{.}\) In this manner the direct sum decomposition \(V=W\oplus W^{\perp}\) gives rise to functions
    \begin{align*} \proj_W\colon V \amp \rightarrow V \amp \proj_{W^\perp}\colon V\amp \rightarrow V \end{align*}
    called orthogonal projection onto \(W\) and \(W^\perp\text{,}\) respectively.
    From the proof of Theorem 1.20.5 we extract the following procedure for computing \(\proj_W(v)\) for a finite-dimensional subspace of an inner product space.

    Procedure 1.20.7. Orthogonal projection.

    Let \(W\) be a finite-dimensional subspace of the inner product space \((V,\angvec{\, ,})\text{.}\) To compute \(\proj_W\) and \(\proj_{W^\perp}\text{,}\) proceed as follows.
    1. Compute an orthogonal basis \(B=(v_1,v_2,\dots, v_r)\) of \(W\) using the Gram-Schmidt procedure.
    2. Given \(v\in V\text{,}\) we have
      \begin{align} \proj_W(v) \amp =\sum_{i=1}^r\frac{\angvec{v,w_i}}{\angvec{w_i,w_i}}w_i\tag{1.76}\\ \proj_{W^\perp} \amp = v-\proj_W(v)\text{.}\tag{1.77} \end{align}

    Example 1.20.8. Orthogonal projection onto plane.

    Consider the inner product space \(\R^3\) with the dot product. Let \(W\subseteq\R^3\) be the plane with equation \(x+y+z=0\text{.}\) Compute \(\proj_W(v)\) for each \(\boldv\) below.
    1. \(\displaystyle \boldv=(3,-2,2)\)
    2. \(\displaystyle \boldv=(2,1,-3)\)
    3. \(\displaystyle \boldv=(-7,-7,-7)\)
    Solution.
    According to Procedure 1.20.7 our first step is to produce an orthogonal basis of \(W\text{.}\) We do so by inspection. Since \(\dim W=2\text{,}\) we simply need to find two solutions to \(x+y+z=0\) that are orthogonal to one another: e.g., \(\boldw_1=(1,-1,0)\) and \(\boldw_2=(1,1,-2)\text{.}\) Thus we choose \(B=((1,-1,0), (1,1,-2))\) as our orthogonal basis, and our computations become a matter of applying (1.76), which in this case becomes
    \begin{equation*} \proj_W(v)=\frac{\boldv\cdot\boldw_1}{\boldw_1\cdot \boldw_1}\boldw_1+\frac{\boldv\cdot\boldw_2}{\boldw_2\cdot \boldw_2}\boldw_2= \frac{\boldv\cdot\boldw_1}{2}\boldw_1+\frac{\boldv\cdot\boldw_2}{6}\boldw_2\text{.} \end{equation*}
    Now compute:
    \begin{align*} \proj_W(3,-2,2) \amp =\frac{5}{2}(1,-1,0)+\frac{-3}{6}(1,1,-2)=(2,-3,1)\\ \proj_W(2,1,-3) \amp =\frac{1}{2}(1,-1,0)+\frac{9}{6}(1,1,-2)=(2,1,-3)\\ \proj_W(-7,-7,-7) \amp =\frac{0}{2}(1,-1,0)+\frac{0}{6}(1,1,-2)=(0,0,0)\text{.} \end{align*}
    The last two computations might give you pause. Why do we have \(\proj_W(2,1,-3)=(2,1,-3)\) and \(\proj_W(-7,7-7,-7)=(0,0,0)\text{?}\) The answer is that \((2,1,-3)\) is already an element of \(W\text{,}\) so it stands to reason that its projection is itself; and \((-7,-7,-7)\) is already orthogonal to \(W\) (it is a scalar multiple of \((1,1,1)\)), so it stands to reason that its projection is equal to \(\boldzero\text{.}\)

    Theorem 1.20.9. Orthogonal projection.

    Let \((V, \langle\, , \rangle)\) be an inner product space, and let \(W\) be a finite-dimensional subspace of \(V\text{.}\)
    1. Orthogonal projection is linear.
      \(\operatorname{proj}_{W}, \operatorname{proj}_{W^\perp}\in \mathcal{L}(V)\text{.}\)
    2. Orthogonal projection as closest element.
      \(\proj_W(v)\) is the unique element \(w\in W\) satisfying the following property:
      \begin{equation} d(v,w)\leq d(v,w') \text{ for all } w\in W\text{.}\tag{1.78} \end{equation}
      In other words, \(\proj_W(v)\) is the unique element in \(W\) that minimizes the distance to \(v\text{:}\) i.e., it is the unique element of \(W\) closest to \(v\text{.}\)
      Similarly, \(\proj_{W^\perp}(v)\) is the unique element in \(W^\perp\) that minimizes the distance to \(v\text{.}\)
    3. We have \(\proj_W(v)+\proj_{W^\perp}(v)=v\) for all \(v\in V\text{:}\) i.e.,
      \begin{equation} \proj_W+\proj_{W^\perp}=I_V\text{.}\tag{1.79} \end{equation}

    Proof.

    1. Both \(\proj_W\) and \(\proj_{W^\perp}\) are specific examples of the more general projection onto a direct summand operator. We saw in Theorem 1.9.20 the such operators are linear.
    2. Given \(v\in V\text{,}\) write \(v=w+w^\perp\) with \(w\in W\) and \(w^\perp\in W^\perp\text{.}\) By definition this means \(\proj_W(v)=w\text{.}\) Given any \(w'\in W\) we have
      \begin{align*} d(v,w') \amp = \norm{v-w'}\\ \amp = \norm{(w-w')+w^\perp}\\ \amp = \sqrt{\norm{w-w'}+\norm{w^\perp}}^2 \amp (w-w' \text{ ortho. to } w^\perp ) \\ \amp \geq \sqrt{\norm{w^\perp}^2}\\ \amp = \norm{w^\perp}\\ \amp = \norm{v-w}\\ \amp = d(v,\proj_W(v))\text{.} \end{align*}
      This proves \(d(v,\proj_W(v))\leq d(v,w')\) for all \(w'\in W\text{:}\) i.e., \(\proj_W(v)\) is an element of \(W\) that minimizes the distance to \(v\text{.}\) It remains to show that \(\proj_W(v)\) is the unique element of \(W\) satisfying this property. To this end, if \(w'\in W\) is another element of \(W\) that minimizes the distance to \(v\text{,}\) then in particular we have \(d(v,w')\leq d(v,\proj_W(v))\text{.}\) Since we also have \(d(v,\proj_W(v))\leq d(v,w')\text{,}\) we conclude that \(d(v,w')=d(v,\proj_W(v))\text{.}\) Looking at the chain of inequalities above, this would imply that \(sqrt{\norm{w-w'}+\norm{w^\perp}}^2=\sqrt{{w^\perp}^2}\text{.}\) The latter is true if and only if \(\norm{w-w'}=0\text{,}\) if and only if \(w'=w=\proj_W(v)\text{,}\) as desired.
      Note that the proof above only used the fact that \(V=W\oplus W^\perp\) is a direct sum decomposition into two orthogonal spaces. As such, essentially the same argument can be used to show that \(\proj_{W^\perp}(v)\) is the unique element of \(W^\perp\) minimizing the distance to \(v\text{.}\)
    3. This is essentially by definition of \(\proj_W\) and \(\proj_{W^\perp}\text{.}\) Any vector \(v\in V\) can be written as \(v=w+w^\perp\text{,}\) where \(w\in W\) and \(w^\perp\in W^\perp\text{.}\) By definition we have \(w=\proj_W(v)\) and \(w^\perp=\proj_{W^\perp}(v)\text{,}\) whence
      \begin{equation*} v=w+w^\perp=\proj_W(v)+\proj_{W^\perp}(v)\text{.} \end{equation*}

    Definition 1.20.10. Distance to subspace.

    Let \(W\) be a subspace of the inner product space \((V,\angvec{\, ,})\text{.}\) Given a vector \(v\in V\text{,}\) its distance to \(W\text{,}\) denoted \(d(v,W)\text{,}\) is defined as
    \begin{equation*} d(v,W)=\inf\{d(v,w)\colon w\in W\}\text{.} \end{equation*}

    Corollary 1.20.11. Distance to \(W\).

    Let \(W\) be a finite-dimensional subspace of the inner product space \((V,\angvec{\, ,})\text{.}\) For all \(v\in V\text{,}\) we have
    \begin{equation} d(v,W)=d(v,\proj_W(v))=\norm{v-\proj_W(v)}\text{.}\tag{1.80} \end{equation}

    Example 1.20.12. Projection onto a line.

    Any line in \(\R^3\) passing through the origin can be described as \(\ell=\Span\{\boldv\}\text{,}\) for some \(\boldv=(a,b,c)\ne 0\text{.}\) The tuple \(((a,b,c))\) is trivially an orthogonal basis of \(\ell\text{.}\) Using (1.76), we have
    \begin{equation*} \proj_\ell(\boldx)=\frac{\boldx\cdot \boldv}{\boldv\cdot\boldv}\boldv=\frac{ax+by+cz}{a^2+b^2+c^2}(a,b,c)\text{.} \end{equation*}
    It follows that \(\operatorname{proj}_\ell=T_A\text{,}\) where
    \begin{equation*} A=\begin{bmatrix} \vert \amp \vert \amp \vert \\ \proj_\ell(1,0,0)\amp \proj_\ell(0,1,0)\amp \proj_\ell(0,0,1) \\ \vert \amp \vert \amp \vert \end{bmatrix}=\frac{1}{a^2+b^2+c}\begin{bmatrix} a^2\amp ab\amp ac\\ ab\amp b^2\amp bc\\ ac\amp bc\amp c^2 \end{bmatrix}\text{.} \end{equation*}

    Example 1.20.13. Projection onto specific line.

    Consider the line \(\ell=\Span\{(1,2,1)\}\subseteq \R^3\text{.}\)
    1. Find the matrix \(A\) such that \(\operatorname{proj}_\ell=T_A\text{.}\)
    2. Use your matrix formula from (a) to compute \(\proj_\ell(-2,3,1)\text{,}\) \(\proj_\ell(-2,-4,-2)\text{,}\) and \(\proj_\ell(1,-1,1)\text{.}\)
    3. Compute \(d((-2,3,1), \ell)\) and \(d((-2,-4,-2), \ell)\text{.}\)
    Solution.
    1. Using the general formula described in Example 1.20.12, we have
      \begin{equation*} A=\frac{1}{6}\begin{bmatrix} 1\amp 2\amp 1\\ 2\amp 4\amp 2\\ 1\amp 2\amp 1 \end{bmatrix}\text{.} \end{equation*}
    2. Now compute
      \begin{align*} \proj_\ell(-2,3,1)\amp= \frac{1}{6}\begin{bmatrix} 1\amp 2\amp 1\\ 2\amp 4\amp 2\\ 1\amp 2\amp 1 \end{bmatrix} \begin{amatrix}[r] -2 \\ 3\\ 1 \end{amatrix}= \frac{1}{6}\begin{amatrix}[r] 5\\ 10\\ 5 \end{amatrix}\\ \proj_W(-2,-4,-2)\amp= \frac{1}{6}\begin{bmatrix} 1\amp 2\amp 1\\ 2\amp 4\amp 2\\ 1\amp 2\amp 1 \end{bmatrix} \begin{amatrix}[r] -2 \\ -4\\ -2 \end{amatrix}= \frac{1}{6}\begin{amatrix}[r] -2\\ -4\\ -2 \end{amatrix}\\ \proj_W(1,-1,1)\amp= \frac{1}{6}\begin{bmatrix} 1\amp 2\amp 1\\ 2\amp 4\amp 2\\ 1\amp 2\amp 1 \end{bmatrix} \begin{amatrix}[r] 1\\ -1\\ 1 \end{amatrix}= \frac{1}{6}\begin{amatrix}[r] 0\\ 0\\ 0 \end{amatrix}\text{.} \end{align*}
      The last two computations, \(\proj_{\ell}((-2,-4,-2))=(-2,-4,-2)\) and \(\proj_{\ell}((1,-1,1))=(0,0,0)\text{,}\) should come as no surprise, since \((-2,-4,-2)\in \ell\) and \((1,-1,1)\in \ell^\perp\text{.}\)
    3. We have
      \begin{align*} d((-2,3,1), \ell) \amp=\norm{(-2,3,1)-\proj_\ell(-2,3,1)} \\ \amp = \norm{\frac{1}{6}(-17,8,1)}=\frac{\sqrt{354}}{6} \\ d((-2,-4,-2), \ell) \amp=\norm{(-2,-4,-2)-\proj_\ell(-2,-4,-2)} \\ \amp = \norm{(0,0,0)}=0 \text{.} \end{align*}
      Again, the second computation should come as no surprise. Since \((-2,-4,-2)\) is itself an element of \(\ell\text{,}\) it stands to reason that its distance to \(\ell\) is equal to zero.

    Example 1.20.14. Projection onto planes in \(\R^3\).

    Any plane \(W\subseteq\R^3\) passing through the origin can be described as \(W=\{(x,y,z)\in \R^3\colon ax+by+cz=0\}\text{.}\) Equivalently, \(W\) is the set of all \(\boldx\in \R^3\) satisfying \(\boldx\cdot (a,b,c)=0\text{:}\) i.e., \(W=\ell^\perp\text{,}\) where \(\ell=\Span((a,b,c))\text{.}\)
    Let \(A\) and \(A'\) be the standard matrices for \(\proj_{\ell}\) and \(\proj_W\text{,}\) respectively. Since \(I_{\R^3}=\proj_{\ell}+\proj_W\) ((1.79)), it follows that \(I=A+A'\text{,}\) or equivalently \(A'=I-A\text{.}\) Making use of our standard matrix computation for \(\proj_\ell\) in Example 1.20.12, we now compute
    \begin{align*} A' \amp =I-A\\ \amp = \begin{bmatrix} 1\amp 0\amp 0\\ 0\amp 1\amp 0\\ 0\amp 0\amp 1 \end{bmatrix} - \frac{1}{a^2+b^2+c}\begin{bmatrix} a^2\amp ab\amp ac\\ ab\amp b^2\amp bc\\ ac\amp bc\amp c^2 \end{bmatrix}\\ \amp = \frac{1}{a^2+b^2+c^2}\begin{bmatrix}b^2+c^2\amp -ab\amp -ac\\ -ab\amp a^2+c^2\amp -bc\\ -ac\amp -bc\amp a^2+b^2 \end{bmatrix}\text{.} \end{align*}

    Subsection Trigonometric polynomial approximation

    Consider the inner product space consisting of \(C([0,2\pi])\) along with the integral inner product \(\langle f, g\rangle=\int_0^{2\pi}f(x)g(x) \, dx\text{.}\) In Example 1.19.4 we saw that
    \begin{equation*} B=(1, \cos(x),\sin(x),\cos(2x),\sin(2x), \dots , \cos(nx),\sin(nx)) \end{equation*}
    is orthogonal with respect to this inner product. Thus \(B\) is an orthogonal basis of
    \begin{equation*} \Span B=\{g\in C([0,2\pi])\colon g(x)=a_0+\sum_{k=1}^na_k\cos kx +b_k\sin kx \text{ for some } a_i, b_i\in \R\}\text{.} \end{equation*}
    We call \(W=\Span B\) the space of trigonometric polynomials of degree at most \(n\).
    Since \(B\) is an orthogonal basis of \(W\text{,}\) given an arbitrary function \(f(x)\in C[0,2\pi]\text{,}\) its orthogonal projection \(\hat{f}=\proj_{W}(f)\) is given by
    \begin{equation*} \hat{f}(x)=a_0+a_1\cos(x)+b_1\sin(x)+a_2\cos(2x)+b_2\sin(2x)+\cdots +a_n\cos(nx)+b_n\sin(nx)\text{,} \end{equation*}
    where
    \begin{equation*} a_0=\frac{1}{2\pi}\int_0^{2\pi} f(x) \ dx, \ a_j=\frac{1}{\pi}\int_0^{2\pi}f(x)\cos(jx)\, dx, \ b_k=\frac{1}{\pi}\int_0^{2\pi}f(x)\sin(kx)\, dx\text{.} \end{equation*}
    Here we are using (1.76), as well as the inner product formulas \(\angvec{1,1}=2\pi\) and \(\angvec{\cos n x, \cos n x}=\angvec{\sin n x, \sin n x}=\pi\) from Example 1.19.4.
    What is the relationship between \(f\) and \(\hat{f}\text{?}\) Theorem 1.20.9 tells us that \(\hat{f}\) is the “best” trigonometric polynomial approximation of \(f(x)\) of degree at most \(n\) in the following sense: given any any other trigonometric polynomial \(g\in W\text{,}\) we have
    \begin{equation*} \left\vert\left\vert f-\hat{f}\right\vert\right\vert\leq \norm{f-g}\text{.} \end{equation*}
    Unpacking the definition of norm in this inner product space, we conclude that
    \begin{equation*} \int_0^{2\pi} (f-\hat{f})^2\, dx\leq \int_0^{2\pi} (f-g)^2 \, dx \end{equation*}
    for all \(g\in W\text{.}\)
    Thus, given a continuous function \(f\) on \([0,2\pi]\text{,}\) linear algebra shows us how to find its best trigonometric polynomial approximation of the form
    \begin{equation*} g(x)=a_0+\sum_{k=1}^na_k\cos kx +b_k\sin kx\text{.} \end{equation*}
    However, linear algebra does not tell us just how good this approximation is. This question, among others, is tackled by another mathematical theory: Fourier analysis. There we learn that the trigonometric polynomial approximations get arbitrarily close to \(f\) as we let \(n\) increase. More precisely, letting \(\hat{f}_n\) be the orthogonal projection of \(f\) onto the space of trigonometric polynomials of degree at most \(n\text{,}\) we have
    \begin{equation*} \lim_{n\to\infty}\norm{f-\hat{f}_n}=0\text{.} \end{equation*}

    Subsection Least-squares solution to linear systems

    In statistics we often wish to approximate a scatter plot of points \(P_i=(X_i, Y_i)\text{,}\) \(1\leq i\leq m\text{,}\) with a line \(\ell\colon y=mx+b\) that “best fits” the data. “Finding” this line amounts to finding the appropriate slope \(m\) and \(y\)-intercept \(b\text{:}\) i.e., in this setup, the points \(P_i=(X_i, Y_i)\) are given, and \(m\) and \(b\) are the unknowns we wish to find. For the line to perfectly fit the data, we would want
    \begin{equation*} Y_i=mX_i+b \text{ for all } 1\leq i\leq m\text{.} \end{equation*}
    In other words \((m,b)\) would be a solution to the matrix equation \(A\boldx=\boldy\text{,}\) where
    \begin{equation*} \boldx=\begin{bmatrix}m \\ b\end{bmatrix},A=\begin{bmatrix} X_1\amp 1\\ X_2\amp 1\\ \vdots \amp \vdots \\ X_m\amp 1 \end{bmatrix}, \boldy=\begin{bmatrix} Y_1\\ Y_2\\ \vdots \\ Y_m\end{bmatrix}\text{.} \end{equation*}
    Of course in most situations the provided points do not lie on a line, and thus there is no solution \(\boldx\) to the given matrix equation \(A\boldx=\boldy\text{.}\) When this is the case we can use the theory of orthogonal projection to find what is called a least-squares solution, which we now describe in detail.
    The least-squares method applies to any matrix equation
    \begin{equation} \underset{m\times n}{A}\, \underset{n\times 1}{\boldx}=\underset{m\times 1}{\boldy}\text{,}\tag{1.81} \end{equation}
    where \(A\) and \(\boldy\) are given, and \(\boldx\) is treated as an unknown vector. Recall that
    \begin{align*} A\boldx=\boldy \text{ has a solution } \amp\iff \boldy\in \CS A \text{.} \end{align*}
    When \(\boldy\notin \CS A\text{,}\) and hence (1.81) does not have a solution, the least-squares method proceeds by replacing \(\boldy\) with the element of \(W=\CS A\) closest to it: that is, with its orthogonal projection onto \(W\text{.}\) Let \(\hat{\boldy}=\proj_{W}(\boldy)\text{,}\) where orthogonal projection is taken with respect to the dot product on \(\R^m\text{,}\) and consider the adjusted matrix equation
    \begin{equation} A\boldx=\hat{\boldy}\text{.}\tag{1.82} \end{equation}
    By definition of \(\operatorname{proj}_W\text{,}\) we have \(\hat{\boldy}\in W=\CS A\text{,}\) and thus there is a solution \(\hat{\boldx}\) to (1.82). We call \(\hat{\boldx}\) a least-squares solution to (1.81). Observe that \(\hat{\boldx}\) does not necessarily satisfy \(A\hat{\boldx}=\boldy\text{;}\) rather, it satisfies \(A\hat{\boldx}=\hat{\boldy}\text{.}\) What makes this a “least-squares” solution is that \(A\hat{\boldx}=\hat{\boldy}\) is the element of \(W=\CS A\) closest to \(\boldy\text{.}\) With respect to the dot product, this means that a least-squares solution \(\hat{\boldx}\) minimizes the quantity
    \begin{equation*} \norm{\boldy-A\boldx}=\sqrt{(y_1-y_1')^2+(y_2-y_2')^2+\cdots +(y_n-y_n')^2}\text{,} \end{equation*}
    among all \(\boldx\in \R^n\text{.}\)

    Example 1.20.16. Best fitting line.

    Suppose we wish to find a line \(\ell\colon y=mx+b\) that best fits (in the least-square sense) the following data points: \(P_1=(-3,1), P_2=(1,2), P_3=(2,3)\text{.}\) Following the discussion above, we seek a solution \(\boldx=(m,b)\) to the matrix equation \(A\boldx=\boldy\text{,}\) where
    \begin{equation*} \boldx=\begin{bmatrix}m \\ b \end{bmatrix}, A=\begin{amatrix}[rr]-3\amp 1\\ 1\amp 1\\ 2\amp 1 \end{amatrix} , \boldy=\begin{bmatrix}1\\ 2\\ 3 \end{bmatrix}\text{.} \end{equation*}
    Using Gaussian elimination, we see easily that this equation has no solution: equivalently, \(\boldy\notin W=\CS A\text{.}\) Accordingly, we compute \(\hat{\boldy}=\proj_{W}(\boldy)\) and find a solution to \(A\hat{\boldx}=\hat{\boldy}\text{.}\) Conveniently, the set \(B=\{(-3,2,1), (1,1,1)\}\) is already an orthogonal basis of \(W=\CS A\text{,}\) allowing us to use (1.76):
    \begin{equation*} \hat{\boldy}=\frac{\boldy\cdot (-3,1,2)}{(-3,2,1)\cdot (-3,1,2)}(-3,1,2)+\frac{\boldy\cdot(1,1,1)}{(1,1,1)\cdot (1,1,1)}(1,1,1)=\frac{1}{14}(13, 33, 38)\text{.} \end{equation*}
    Lastly, solving \(A\hat{\boldx}=\hat{\boldy}\) yields \((m,b)=\hat{\boldx}=(5/14, 2)\text{,}\) and we conclude the line \(\ell\colon y=(5/14)x+2\) is the one that best fits the data in the least-squares sense.

    Remark 1.20.17. Visualizing least-squares.

    Figure 1.20.18 helps us give a graphical interpretation of how the line \(\ell\colon y=(5/14)x+2\) best approximates the points \(P_1=(-3,1), P_2=(1,2), P_3=(2,3)\text{.}\)
    Let \(\boldy=(1,2,3)=(y_1,y_2,y_3)\) be the given \(y\)-values of the points, and let \(\hat{\boldy}=(y_1',y_2',y_3')\) be the orthogonal projection of \(\boldy\) onto \(\CS A\text{.}\) In the graph the values \(\epsilon_i\) denote the vertical difference \(\epsilon_i=y_i-y_i'\) between the data points, and our fitting line. The projection \(\hat{\boldy}\) makes the error \(\norm{\boldy-\hat{\boldy}}=\sqrt{ \epsilon_1^2+\epsilon_2^2+\epsilon_3^2}\) as small as possible. This means if I draw any other line and compute the corresponding differences \(\epsilon_i'\) at the \(x\)-values -3, 1 and 2, then
    \begin{equation*} \epsilon_1^2+\epsilon_2^2+\epsilon_3^2\leq (\epsilon_1')^2+(\epsilon_2')^2+(\epsilon_3')^2 \end{equation*}
    To compute a least-squares solution to \(A\boldx=\boldy\) we must first compute the orthogonal projection of \(\boldy\) onto \(W=\CS A\text{;}\) and this in turn requires first producing an orthogonal basis of \(\CS A\text{,}\) which may require using the Gram-Schmidt procedure. The following result bypasses these potentially onerous steps by characterizing a least-squares solution to \(A\boldx=\boldy\) as a solution to the matrix equation
    \begin{equation*} A^TA\boldx=A^T\boldy\text{.} \end{equation*}

    Theorem 1.20.19. Least-squares matrix formula.

    Given an \(m\times n\) matrix \(A\) and \(m\times 1\) column vector \(\boldy\text{,}\) a vector \(\hat{\boldx}\) is a least-squares solution to \(A\boldx=\boldy\) if and only if
    \begin{equation} A^TA\boldx=A^T\boldy\text{.}\tag{1.83} \end{equation}
    In other words, we can find a least-squares solution by solving the matrix equation (1.83) directly.

    Proof.

    Let \(W=\CS A\text{,}\) and let \(\hat{\boldy}=\proj_{W}(\boldy)\text{.}\) The key observation is that a vector \(\hat{\boldx}\) satisfies \(A\hat{\boldx}=\hat{\boldy}\) if and only if
    \begin{equation*} \boldy=A\hat{\boldx}+(\boldy-A\hat{\boldx}) \end{equation*}
    is an orthogonal decomposition of \(\boldy\) with respect to \(W=\CS A\text{;}\) and this is true if and only if \(\boldy-A\hat{\boldx}\in (\CS A)^\perp\text{.}\) Thus we have
    \begin{align*} A\hat{\boldx}=\hat{\boldy} \amp\iff \boldy-A\hat{\boldx}\in (\CS A)^\perp \\ \amp\iff \boldy-A\hat{\boldx}\in \NS A^T \amp ((\CS A)^\perp=\NS A^T) \\ \amp\iff A^T(\boldy-A\hat{\boldx})=\boldzero \\ \amp \iff A^T\boldy-A^TA\hat{\boldx}=\boldzero\\ \amp \iff A^TA\hat{\boldx}=A^T\boldy\text{.} \end{align*}

    Example 1.20.20.

    Consider again the matrix equation \(A\boldx=\boldy\) from Example 1.20.16. According to Theorem 1.20.19 the least-squares solution can be found by solving the equation \(A^TA\boldx=A^T\boldy\) for \(\boldx\text{.}\) We compute
    \begin{align*} A^TA\amp=\begin{amatrix}[rr]14\amp 0\\ 0\amp 3 \end{amatrix} \amp A^T\boldy\amp =\begin{amatrix}[r] 5\\ 6 \end{amatrix} \end{align*}
    and solve
    \begin{equation*} \begin{amatrix}[rr]14\amp 0\\ 0\amp 3 \end{amatrix}\boldx=\begin{amatrix}[r] 5\\ 6 \end{amatrix}\iff \boldx=(5/14, 2), \end{equation*}
    just as before.

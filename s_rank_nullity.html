    Section 1.11 Rank-nullity theorem

    We now prove the rank-nullity theorem, sometimes called the fundamental theorem of linear algebra. This theorem relates the dimensions of the null space and image of a linear transformation \(T\colon V\rightarrow W\text{,}\) assuming \(V\) is finite dimensional. Roughly speaking, it says that the bigger the null space, the smaller the image. More precisely, it tells us that
    \begin{equation*} \dim V=\dim\NS T+\dim\im T\text{.} \end{equation*}
    As we will see, this elegant result can be used to significantly simplify computations with linear transformations. For example, in a situation where we wish to compute explicitly both the null space and image of a given linear transformation, we can often get away with just computing one of the two spaces and using the rank-nullity theorem (and a dimension argument) to easily determine the other. Additionally, the rank-nullity theorem will directly imply some intuitively obvious properties of linear transformations. For example, suppose \(V\) is a finite-dimensional vector space. It seems obvious that if \(\dim W> \dim V\text{,}\) then there is no linear transformation mapping \(V\) surjectively onto \(W\text{:}\) i.e., you should not be able to map a “smaller” vector space onto a “bigger” one. Similarly, if \(\dim W \lt \dim V\text{,}\) then we expect that there is no injective linear transformation mapping \(V\) injectively into \(W\text{.}\) Both these results are easy consequences of the rank-nullity theorem.

    Theorem 1.11.1. Rank-nullity theorem.

    Let \(V\) be a finite-dimensional vector space. If \(T\colon V\rightarrow W\) is a linear transformation, then
    \begin{equation} \dim V=\dim\NS T+\dim \im T\text{.}\tag{1.28} \end{equation}

    Proof.

    Choose a basis \(B'=(\boldv_1, \boldv_2, \dots, \boldv_k)\) of \(\NS T\) and extend \(B'\) to a basis \(B=(\boldv_1, \boldv_2,\dots, \boldv_k,\boldv_{k+1},\dots, \boldv_n)\text{,}\) using Theorem 1.8.8. From the proof of Theorem 1.8.13 it follows that
    \begin{equation*} V=\NS T\oplus W\text{,} \end{equation*}
    where \(W=\Span(v_{k+1},v_{k+2},\dots, v_n)\text{.}\) We claim that \(B''=(T(\boldv_{k+1}),T(\boldv_{k+2}),\dots, T(\boldv_{n}))\) is a basis of \(\im T\text{.}\) Once this is established, (1.28) follows easily, since then we have
    \begin{align*} \dim V \amp = \len B= n\\ \dim \NS T \amp = \len B'=k \\ \dim \im T\amp = \len B''=n-k\text{,} \end{align*}
    and thus
    \begin{align*} \dim V \amp = n\\ \amp = k+(n-k)\\ \amp =\dim \NS T+\dim \im T \text{.} \end{align*}
    We will prove \(B''\) is a basis of \(\im T\) by proving that it is linearly independent and a spanning set.

    Proof that \(B''\) is linearly independent.

    Assume
    \begin{equation*} \sum_{j=k+1}^n c_jT(v_j)=\boldzero \text{.} \end{equation*}
    Since \(T\) is linear, this implies that
    \begin{equation*} T(\sum_{j=k+1}^nv_j)=\boldzero \end{equation*}
    and hence that \(\sum_{j=k+1}^n c_jv_j \in \NS T\text{.}\) On the other hand, we have
    \begin{equation*} \sum_{j=k+1}^n c_jv_j\in W=\Span(v_{k+1},\dots, v_n)\text{.} \end{equation*}
    Since \(V=\NS T\oplus W\text{,}\) we have \(\NS T\cap W=\{\boldzero\}\text{,}\) and hence \(\sum_{j=k+1}^n c_jv_j=\boldzero\text{.}\) Lastly, since \((v_{k+1},\dots, v_n)\) is linearly independent, we conclude that \(c_j=0\) for all \(k+1\leq j\leq n\text{,}\) as desired.

    Proof that \(\Span B''=\im T\).

    We prove \(\Span B''\subseteq \im T\) and \(\im T\subseteq \Span B''\) separately.
    Since \(\im T\) is a subspace, and since \(T(v_j)\in \im T\) for all \(k+1\leq j\leq n\text{,}\) it follows that \(\Span B''\subseteq \im T\) from statement (2) of Theorem 1.6.6.
    For the other inclusion, we have
    \begin{align*} w\in \im T \amp \implies w=T(v) \text{ for some } v\in V\\ \amp \implies w=T(\sum_{j=1}^nc_jv_j) \text{ for some } c_j\in F\\ \amp \implies w=\sum_{j=1}^nc_jT(v_j) \text{ for some } c_j\in F\\ \amp \implies w=\sum_{j=k+1}^nc_jT(v_j) \text{ for some } c_j\in F\text{,} \end{align*}
    since \(T(v_j)=\boldzero\) for all \(1\leq j\leq k\text{.}\) But then
    \begin{equation*} w=c_{k+1}T(v_{k+1})+\cdots c_nT(v_n)\in \Span B''\text{,} \end{equation*}
    proving that \(\im T\subseteq \Span B''\text{.}\)

    Remark 1.11.2. Rank-nullity theorem.

    Some textbooks will define the rank and nullity of a linear transformation \(T\) as \(\dim \im T\) and \(\dim \NS T\text{,}\) respectively. This is where the name of this theorem comes from.
    The rank-nullity theorem relates the size (measured by dimension) of the two spaces \(\NS T\) and \(\im T\text{.}\) This can be useful in computations when one of these two spaces is easy to compute.

    Example 1.11.3. Rank-nullity: image from null space.

    Let \(T\colon \R^4\rightarrow \R^3\) be defined as
    \begin{equation*} T(x_1,x_2,x_3,x_4)=(x_1+x_4, x_2+x_3+x_4, x_3+x_4)\text{.} \end{equation*}
    Use the rank-nullity theorem to prove that \(T\) is surjective.
    Solution.

    Corollary 1.11.4. Injective, surjective, bijective.

    Let \(V\) be a finite-dimensional vector space, and let \(T\colon V\rightarrow W\) be a linear transformation.
    1. If \(\dim V> \dim W\text{,}\) then \(T\) is not injective.
    2. If \(\dim V< \dim W\text{,}\) then \(T\) is not surjective.
    3. Assume further that \(\dim V=\dim W< \infty\text{.}\) The following statements are equivalent.
      1. \(T\) is bijective.
      2. \(T\) is injective.
      3. \(T\) is surjective.

    Proof.

    1. First assume \(\dim V=n< \infty\text{.}\) We have
      \begin{align*} \dim \NS T \amp =\dim V-\dim\im T \amp (\text{rank-nullity})\\ \amp > \dim W-\dim \im T \amp (\dim V> \dim W)\\ \amp \geq 0 \amp (\dim \im T\leq \dim W)\text{.} \end{align*}
      It follows that \(\dim \NS T> 0\text{,}\) and hence that \(\NS T\ne \{\boldzero\}\text{,}\) and thus \(T\) is not injective.
      Now assume that \(\dim V=\infty\) and \(\dim W=m< \infty\text{.}\) Pick an infinite linearly independent tuple \((v_i)_{i\in i}\) in \(V\text{,}\) and let \(B=(v_{i_1}, v_{i_2},\dots, v_{i_n})\) be any finite subtuple of length \(n> m\text{.}\) We know from a homework exercise that \(B\) is linearly independent. Thus \(V'=\Span B\) is an \(n\)-dimensional subspace of \(V\) and satisfies \(\dim V'> \dim W\text{.}\) Applying the previous argument to the restriction \(T\vert_{V'}\colon V'\rightarrow W\text{,}\) we conclude that \(\NS T\vert_{V'}\ne\{\boldzero\}\text{.}\) Since \(\NS T\vert_{V'}\subseteq \NS T\text{,}\) we conclude that \(\NS T\ne \{\boldzero\}\text{,}\) and thus \(T\) is not injective.
    2. We assume \(\dim V=n< \dim W\text{.}\) Since
      \begin{equation*} \dim \im T=\dim V-\dim \NS T\leq \dim V< \dim W \end{equation*}
      we cannot have have \(\im T=\dim W\text{.}\) Thus \(T\) is not surjective.
    3. We assume \(\dim V=\dim W=n< \infty\text{.}\) We prove a cycle of implications (a)\(\implies\)(b)\(\implies\)(c)\(\implies\)(d).
      We have (a) implies (b) by definition.
      Assume \(T\) is injective. By Theorem 1.10.11, this implies \(\NS T=\{\boldzero\}\) and hence \(\dim \NS T=0\text{.}\) It follows that
      \begin{equation*} \dim \im T=\dim V-\dim \NS T=\dim V-0=\dim W\text{.} \end{equation*}
      Since \(\im W\subseteq W\) and \(\dim \im W=\dim W\text{,}\) we have \(\im T=W\) by Corollary 1.8.10. Thus \(T\) is surjective. This proves that (b) implies (c).
    4. If \(T\) is surjective, then \(\im T=W\) in which case \(\dim \im T=\dim W=n\text{.}\) It follows that
      \begin{equation*} \dim \NS T=\dim V-\dim \im T=n-n=0\text{.} \end{equation*}
      Hence \(\dim \NS T=0\text{,}\) or equivalently, \(\NS T=\{\boldzero\}\text{.}\) It follows that \(T\) is injective. Since \(T\) is injective and surjective, it is bijective. This proves that (c) implies (a).

    Example 1.11.5. Polynomial evaluation.

    Let \(I\) be an infinite subset of the field \(F\text{,}\) let \(n\) be a positive integer, and let \(\boldc=(c_1,c_2,\dots, c_{n+1})\in F^{n+1}\) be a choice of \(n+1\) distinct scalars that we think of as inputs. Prove that for any choice \(\boldd=(d_1,d_2,\dots, d_{n+1})\in F^{n+1}\) of \(n+1\) outputs there is a unique polynomial \(f\in P_n(I)\) satisfying
    \begin{align*} f(c_1) \amp = d_1\\ f(c_2) \amp = d_2\\ \amp \vdots \\ f(c_n) \amp = d_n\text{.} \end{align*}
    In other words, given any choice of values \(d_k\text{,}\) we can construct a unique polynomial \(f\in P_n(\C)\) whose value at \(c_k\) is \(d_k\) for all \(1\leq k\leq n\text{.}\)
